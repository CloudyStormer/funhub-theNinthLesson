# 第2集发行版图生视频提示词：12段

统一正向风格：国漫写实漫画质感，校园悬疑恐怖，低饱和冷蓝灰绿色调，红色应急灯点缀，电影感构图，人物五官稳定，干净线条，轻微颗粒质感，压迫感强，不血腥。  
统一负向限制：不要欧美脸，不要日式夸张大眼，不要Q版，不要怪物化，不要血腥，不要多余手指，不要文字乱码，不要水印，不要字幕，不要空教室错位，不要突然新增主要角色。

## AI语音总规则

- 林昼：男声，17岁高中男生，压低声音，紧张但嘴硬，不要女声。
- 许眠：女声，17岁女生，冷静、压低、克制，不要惊叫。
- 值夜老师：中年男声，慢、礼貌、压迫，像背课文，不要怪物音。
- 同桌：普通男高中生声音，疑惑、自然。
- 旁白：低沉男声，可以是林昼内心独白，语速慢，压抑。
- 没有明确台词的片段可以只有环境音、呼吸声、脚步声、纸张声。

## 01 陈雨白天归来

### 图生视频 Prompt

第二天白天高中走廊，冷色自然光，走廊通向教室门口。故事承接第一集结尾：昨晚值夜老师在点名册写下“林昼，待观察”，第二天陈雨却像什么都没发生一样正常回班。林昼站在教室门外靠墙一侧，黑短发，蓝白校服外套敞开，白T恤，单肩背包，疲惫警觉。陈雨穿整齐蓝白校服，齐刘海，甜美微笑但眼神空洞，从走廊尽头单向走来，经过林昼身边，继续推门进入教室，不回头、不折返、不在走廊来回走。周围同学一开始也在走廊里正常往各自教室走，随着镜头推进，他们陆陆续续进教室；尾帧时走廊自然变空或只剩极少数远处学生，门内可以看到普通白天教室和已经坐下的学生。镜头缓慢跟随陈雨从走廊到教室门，最后停在林昼回头看她的震惊反应。保持真实校园空间和自然物理关系。

### AI语音/音画同步

- 旁白，低沉男声，0-4秒：`昨晚，老师在点名册上写下了我的名字。`
- 旁白，低沉男声，4-7秒：`第二天，陈雨先回来了。`
- 旁白，低沉男声，7-10秒：`她笑得很正常，正常得像昨晚什么都没发生。`
- 陈雨不说话，只从林昼身边经过。
- 环境音：白天走廊脚步声、远处学生说话声，声音逐渐压低。

## 02 记忆被全班改写

### 图生视频 Prompt

白天教室后排，林昼坐在课桌边压低声音问同桌，同桌一脸疑惑地看他，桌上有作业本和橡皮；背景里陈雨坐在座位上正常翻书，笑容安静但眼神空。林昼神情发紧，像发现记忆只有自己保留。镜头从林昼肩后推向同桌，动作自然，不要夸张表情。

### AI语音/音画同步

- 林昼，男高中生压低声音，1-4秒：`陈雨……她不是失踪了吗？`
- 同桌，男高中生疑惑，5-8秒：`你睡傻了？她昨天还在。`
- 旁白，低沉男声，8-10秒：`他们都记得她。只有我不一样。`
- 环境音：教室翻书声、笔尖写字声，陈雨翻书不要说话。

## 03 班级群记录消失

### 图生视频 Prompt

手机屏幕和林昼脸部的组合特写，屏幕冷光照在林昼紧张的脸上；班级群聊天记录快速滚动，关于陈雨失踪的消息一条条消失，只留下普通作业通知。画面中手机文字可以模糊，不要出现乱码大字。镜头缓慢拉近林昼眼睛，环境声变低频。

### AI语音/音画同步

- 旁白，低沉男声，0-6秒：`我翻遍班级群，关于陈雨失踪的消息，正在一条条消失。`
- 林昼，极低声，7-9秒：`不可能……`
- 环境音：手机轻微提示音变成低频，屏幕滑动声，教室背景声逐渐消失。

## 04 许眠递纸条警告

### 图生视频 Prompt

白天教室课桌下方手部特写，许眠坐在前排，干净校服，黑色长发低马尾，表情冷静克制；她用两根手指把折好的纸条沿桌角向后推给林昼。林昼的手迟疑接过。纸条只露出局部，不要清晰乱码。冷色课堂光，背景学生虚化。

### AI语音/音画同步

- 旁白，低沉男声，0-4秒：`许眠递给我一张纸条。`
- 许眠，女声压低冷静，5-8秒：`晚上有人问你看见什么，只能说黑。`
- 林昼，男声压低，8-10秒：`你到底知道什么？`
- 环境音：纸张摩擦声、课堂远处老师讲课声模糊。

## 05 黑板出现别看

### 图生视频 Prompt

白天教室，林昼从座位抬头，黑板一角多出一行粉笔字“别看黑板”的感觉，但文字可以半遮半糊，不要生成乱码大字；许眠坐在前排没有回头，只用手指轻敲桌面提醒。画面重点是林昼的视线刚要看向黑板又停住。镜头从林昼侧脸切到黑板边缘，声音突然变空。

### AI语音/音画同步

- 旁白，低沉男声，0-5秒：`可我抬头时，黑板角落又多了一行字。`
- 许眠，女声很低，6-8秒：`别看。`
- 林昼不说话，只屏住呼吸。
- 环境音：粉笔轻响一次、教室环境声突然变空、林昼压低呼吸。

## 06 夜晚宿舍熄灯查寝

### 图生视频 Prompt

夜晚男生宿舍，床铺上下铺排列，灯光全部熄灭，只剩窗外冷蓝微光；林昼躺在上铺，校服外套搭在床边，眼睛睁着不敢睡，手紧抓被角。宿舍门缝下慢慢透进一条冷白手电光，门外有值夜老师瘦高影子停住。镜头从宿舍门口低机位缓慢推向林昼床位，环境音压低。

### AI语音/音画同步

- 旁白，低沉男声，0-4秒：`熄灯后，值夜老师来了。`
- 值夜老师，中年男声，慢而礼貌，6-8秒：`查寝。`
- 林昼不说话，只有短促呼吸。
- 环境音：熄灯铃尾声、门外脚步声、手电开关轻响、门轴轻响。

## 07 老师手电测试瞳孔

### 图生视频 Prompt

宿舍上铺近景，手电光直照林昼眼睛，林昼黑色短发，白T恤和校服外套半披，瞳孔被光照亮但他强忍不眨眼，额头有细汗。值夜老师站在床边下方，白衬衫，金属边眼镜反光看不清眼睛，另一只手拿旧点名册。镜头极慢推近林昼眼睛，呼吸声压低。

### AI语音/音画同步

- 值夜老师，中年男声，慢，1-6秒：`夜盲的人，瞳孔会慢一点。`
- 旁白，低沉男声，6-10秒：`我不能眨，也不能躲。`
- 林昼不说话，只强忍呼吸。
- 环境音：手电轻微电流声、宿舍被子摩擦声、心跳低频。

## 08 点名册正放逼读

### 图生视频 Prompt

值夜老师把旧点名册正放到林昼面前，手电照亮纸页，纸上可以隐约有“林昼 待观察”的结构，但不要生成清晰乱码；林昼明明看清却装茫然，视线故意散开。老师的手稳定，点名册不要变形，不要换成书本或手机。宿舍床边同一空间，强压迫近景。

### AI语音/音画同步

- 值夜老师，中年男声，慢，0-4秒：`念一下你的名字。`
- 旁白，低沉男声，5-8秒：`我的名字，就在第一行。`
- 林昼，男声很低，8-10秒：`我……看不清。`
- 环境音：点名册纸页摩擦声、手电光轻响、心跳继续。

## 09 点名册倒置试探

### 图生视频 Prompt

同一空间同一本点名册，老师慢慢把点名册倒转，纸页上下颠倒，仍贴近林昼脸前；林昼嘴角微动，差点顺着字念出来。手电光固定在纸面，老师眼镜反射一道白光。镜头从点名册边缘推向林昼嘴唇，动作慢。

### AI语音/音画同步

- 值夜老师，中年男声，轻声追问，0-5秒：`倒着呢，也黑吗？`
- 旁白，低沉男声，5-8秒：`我明明看得清清楚楚。`
- 林昼嘴唇微动但不要说出名字。
- 环境音：点名册被翻转的纸张声、心跳突然变大。

## 10 许眠暗中踩脚救场

### 图生视频 Prompt

黑暗宿舍下方细节特写，许眠的鞋尖从床边阴影里伸出，踩住林昼垂下来的脚背；许眠本人不完全出现，只露低马尾轮廓或半张冷静侧脸，暗示她藏在附近提醒。林昼脚背一僵，被子轻动。然后回到床边主视角，林昼强行把视线散开，装作看不清。不要让许眠站到明处，不要破坏宿舍空间逻辑。

### AI语音/音画同步

- 林昼，男声吃痛但压住，0-2秒：短促吸气，不要喊。
- 许眠，女声极低，2-4秒：`说黑。`
- 林昼，男声强装茫然，5-9秒：`黑……都是黑的。`
- 环境音：脚踩布鞋轻声、被子摩擦、呼吸压住。

## 11 老师贴脸镜片反光

### 图生视频 Prompt

值夜老师突然贴近林昼，脸几乎到床铺边缘，白衬衫在黑暗里泛灰，金属边眼镜强烈反光，看不清眼睛，僵硬微笑；林昼身体后缩但不敢转头。老师眼镜反光里出现一块不该出现在宿舍里的黑板，黑板上有“林昼，到”的感觉，文字可以模糊但结构明确。现实背景仍是宿舍，不要把场景整体切成教室。

### AI语音/音画同步

- 值夜老师，中年男声，贴近、慢，0-6秒：`那你为什么一直看着我的眼睛？`
- 旁白，低沉男声，6-10秒：`他的镜片里，反着一块黑板。`
- 林昼不说话，呼吸急促但压低。
- 环境音：低频压迫、镜片反光轻微刺耳声。

## 12 点名册加深待观察

### 图生视频 Prompt

旧点名册特写，老师用黑色钢笔在“林昼，待观察”后面再加重一笔，字迹变重，旁边隐约有“陈雨”名字但不要乱码；手电光照着纸页，背景是宿舍黑暗。镜头慢慢推近字迹，最后切黑。结尾压迫感，不解释。

### AI语音/音画同步

- 旁白，低沉男声，0-4秒：`黑板上写着：林昼，到。`
- 值夜老师，中年男声，低声，5-8秒：`还要再观察一次。`
- 结尾字幕/旁白，低沉男声，8-10秒：`别看。看见了，就会被看见。`
- 环境音：钢笔划纸声、手电电流声、最后低频切黑。
