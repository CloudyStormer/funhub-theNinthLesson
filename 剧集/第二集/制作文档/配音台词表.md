# 第2集配音台词表：《别看黑板》

用途：本表只用于后期固定音色配音，不用于可灵随机生成声音。可灵图生视频阶段保持静音；画面通过后，再用本表生成固定音色音频，并在剪映或 Lip Sync 阶段对齐。

目标片长：约 120 秒  
片段规格：12 段，每段约 10 秒  
配音原则：同一角色全剧使用同一个固定音色；不要每段临时换声音。

## 固定音色

| 角色 | 音色 |
| --- | --- |
| 林昼 | 17岁男高中生，低声、紧张、嘴硬，不能女声 |
| 许眠 | 17岁女生，冷静、压低、克制 |
| 值夜老师 | 中年男声，慢、礼貌、压迫，像背课文 |
| 同桌 | 普通男高中生，疑惑、自然 |
| 旁白 | 低沉男声，像林昼内心独白，语速慢、压抑 |

## 音频命名规则

- 单句音频：`ep02_片段号_句号_角色.wav`
- 每段混音：`ep02_片段号_mix.wav`
- 全集混音：`ep02_full_voice_mix.wav`

例子：

- `ep02_02_01_林昼.wav`
- `ep02_02_02_同桌.wav`
- `ep02_02_03_旁白.wav`
