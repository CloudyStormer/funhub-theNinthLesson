# 第2集 100%锁音色工作流

核心原则：不要让可灵靠提示词临时“猜声音”。提示词只能写台词和声音方向，不能保证每次生成同一个音色。要 100% 锁音色，必须把每个角色的声音先生成成固定音频文件，再把音频用于对口型或剪映配音。

## 固定音色表

先在同一个配音工具里固定每个角色的声音，不要每段重新选。

| 角色 | 固定音色要求 | 禁止 |
| --- | --- | --- |
| 林昼 | 同一个 17 岁男高中生声音，低声、紧张、嘴硬 | 不能女声，不能旁白腔 |
| 许眠 | 同一个 17 岁女生声音，冷静、压低、克制 | 不能尖叫，不能甜妹腔 |
| 值夜老师 | 同一个中年男声，慢、礼貌、压迫，像背课文 | 不能怪物音，不能年轻男声 |
| 同桌 | 同一个普通男高中生声音，疑惑、自然 | 不能和林昼混成同一个人 |
| 旁白 | 同一个低沉男声，像林昼内心独白，语速慢 | 不能换成女声，不能播音腔太亮 |

## 推荐文件夹

所有配音放这里：

`剧集/第二集/配音/`

命名规则：

- 单句原始音频：`ep02_片段号_序号_角色.wav`
- 每段合成音频：`ep02_片段号_mix.wav`
- 全集最终配音：`ep02_full_voice_mix.wav`

例子：

- `ep02_02_01_林昼.wav`
- `ep02_02_02_同桌.wav`
- `ep02_02_03_旁白.wav`
- `ep02_02_mix.wav`

## 正确流程

1. 先生成静音画面。
   - 可灵图生视频阶段不要开音画同步。
   - 提示词里仍然保留台词，用来指导表情和嘴型，但不要花费音画同步积分。
   - 画面里如果有对白，尽量让嘴部清楚但不要做大幅夸张口型。

2. 画面通过后，再生成固定音色音频。
   - 每个角色用同一个音色源生成所有台词。
   - 不满意时只重生成音频，不重跑视频。
   - 每句音频控制在对应时间段内，宁可说短一点，不要超时。

3. 需要精准口型的片段，再上传本地音频做可灵 Lip Sync。
   - 可灵对口型时使用已经生成好的 `wav/mp3`，不要用可灵临时 TTS。
   - 同一个片段如果只有一个正面说话角色，适合做 Lip Sync。
   - 多角色同时露脸说话的片段，不适合全靠一个音频自动对口型，容易串嘴。

4. 多角色对白优先在剪映里锁声音。
   - 可灵负责画面，剪映负责最终声音。
   - 林昼、许眠、老师、旁白分轨放置。
   - 口型要求高的单人特写再单独用可灵 Lip Sync。

## 每段提示词里应该怎么写

视频生成阶段：

```text
本片段包含台词，但本次只生成静音画面，不生成最终声音。角色嘴型只需轻微自然开合，声音将在后期用固定音色音频完成。不要自动配音，不要自动字幕，不要新增说话人。
```

最终 Lip Sync 阶段：

```text
使用上传的本地音频作为唯一声音来源。不要重新生成声音，不要更换音色，不要新增背景旁白。视频只根据上传音频调整口型。
```

## 判断标准

能算“锁音色”的只有这两种：

1. 同一个角色的所有台词都来自同一个固定音色 ID。
2. 或同一个角色的所有台词都来自同一个声音克隆模型。

不能算“锁音色”的做法：

- 每段只写“男声”“低沉”“少女声”。
- 每段让可灵根据提示词自己生成声音。
- 每段随机选相似声音。
- 一个片段里多角色同时露脸，让系统自己判断谁在说话。
