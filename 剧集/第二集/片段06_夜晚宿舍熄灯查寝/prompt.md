# 第2集 片段06_夜晚宿舍熄灯查寝

时长：10秒  
生成模式：先图生视频，上传首帧和尾帧；画面通过后，再用固定外部音频做 Lip Sync / 音画同步。  
用途：发行版第二集，共12段，目标总时长约2分钟。

## 统一画风

国漫写实漫画质感，校园悬疑恐怖，低饱和冷蓝灰绿色调，红色应急灯点缀，电影感构图，人物五官稳定，干净线条，轻微颗粒质感，压迫感强，不血腥。

## 首尾帧要求

首帧：夜晚男生宿舍，灯刚熄灭，床铺上下铺排列，林昼躺在上铺睁着眼不敢睡，校服外套搭在床边，窗外冷蓝微光，其他学生躲在被子里模糊不动。

尾帧：同一男生宿舍，宿舍门被推开，值夜老师已经站到林昼床边下方，白衬衫，深色裤子，金属边眼镜反光，手持旧点名册和手电，手电光抬向上铺。此图同时作为 07 首帧。

连续性：连续到07：06尾帧就是07首帧，老师已站到林昼床边。

## 可灵图生视频提示词

夜晚男生宿舍，床铺上下铺排列，灯光全部熄灭，只剩窗外冷蓝微光；林昼躺在上铺，校服外套搭在床边，眼睛睁着不敢睡，手紧抓被角。宿舍门缝下慢慢透进一条冷白手电光，门外有值夜老师瘦高影子停住。镜头从宿舍门口低机位缓慢推向林昼床位，环境音压低。

## AI语音 / 音画同步

锁音色硬规则：本片段不要让可灵根据文字临时生成最终音色。每个角色的最终声音必须来自固定音色音频文件；视频生成阶段只用台词指导表情、节奏和嘴型，最终发布声音在 Lip Sync 或剪映里用固定音频完成。

视频生成阶段可追加：`本片段包含台词，但本次只生成静音画面，不生成最终声音。角色嘴型只需轻微自然开合，声音将在后期用固定音色音频完成。不要自动配音，不要自动字幕，不要新增说话人。`

最终 Lip Sync 阶段可追加：`使用上传的本地音频作为唯一声音来源。不要重新生成声音，不要更换音色，不要新增背景旁白。视频只根据上传音频调整口型。`

- 林昼：男声，17岁高中男生，压低声音，紧张但嘴硬，不要女声。
- 许眠：女声，17岁女生，冷静、压低、克制，不要惊叫。
- 值夜老师：中年男声，慢、礼貌、压迫，像背课文，不要怪物音。
- 同桌：普通男高中生声音，疑惑、自然。
- 旁白：低沉男声，可以是林昼内心独白，语速慢，压抑。
- 没有明确台词的片段只保留环境音、呼吸声、脚步声、纸张声。

本片段台词与声音：

- 旁白，低沉男声，0-4秒：`熄灯后，值夜老师来了。`
- 值夜老师，中年男声，慢而礼貌，6-8秒：`查寝。`
- 林昼不说话，只有短促呼吸。
- 环境音：熄灯铃尾声、门外脚步声、手电开关轻响、门轴轻响。
