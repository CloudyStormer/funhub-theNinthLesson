# 《第九节课》

# 第1集图生视频提示词

第1集暂名：不准早退

项目代号：夜盲同学

## 使用方式

把 `assets/episode01/` 中对应图片作为首帧或参考图，复制每个镜头下方的提示词到图生视频工具。
每段建议生成 5-8 秒，后期剪辑时按 `episode01_edit_timeline.md` 调整长短。

## 通用要求

```text
保持原图人物身份、脸型、发型、服装和场景一致；不要换脸，不要新增主要角色，不要改变校服款式，不要出现乱码文字；动作幅度小，镜头运动克制，保持国漫写实校园悬疑风格，低饱和冷色调，红色应急灯作为恐怖点缀。
```

## 镜头03：夜晚教学楼红光

素材：

```text
assets/episode01/ep01_02_teaching_building_red_light_1.0.png
```

图生视频提示词：

```text
生成5秒横屏视频，16:9，国漫写实漫画质感，校园悬疑氛围，低饱和冷色调。以已上传首帧为唯一画面参考，保持教学楼外观、窗户位置、红光教室位置、地面反光和夜晚构图稳定，不要改变建筑结构。

画面内容：深夜校园空无一人，教学楼大部分窗户熄灭，只有三楼某一间教室透出暗红色应急灯。红光不是大面积爆亮，而是从窗户里轻微渗出、缓慢闪烁。画面整体安静、潮湿、压迫，地面有微弱冷色反光。

镜头运动：全程一个连续镜头，不切换场景，不新增分镜。前2秒保持远景，镜头非常缓慢向教学楼推进；后3秒继续轻微推近到三楼红光窗户，但不要推进太快，不要飞入窗户。

人物控制：如果首帧里有林昼，只让他作为画面下方或远处的小人物短暂停住、抬头看向三楼红光，身体幅度很小；如果首帧没有清楚人物，不要新生成近景人物，不要突然出现陌生人。

氛围和动作：夜风很轻，树影或窗帘可以有极轻微晃动，红色应急灯可以轻微闪烁。不要闪电，不要爆炸，不要镜头剧烈摇晃，不要出现怪物、血、鬼脸、文字、字幕、水印或额外主要角色。保持首帧画风、透视、建筑比例和红光位置稳定。
```

建议时长：8-11 秒
剪辑重点：红光是第1集进入恐怖主线的信号。

素材：

```text
assets/episode01/ep01_03_door_crack_ninth_class_1.0.png
```

图生视频提示词：

```text
生成5秒横屏视频，16:9，国漫写实漫画质感，校园恐怖悬疑氛围，低饱和冷色调，红色应急灯作为唯一恐怖光源。以已上传首帧为唯一构图参考，保持门缝窥视视角、门框黑边、教室纵深、课桌排列和值夜老师位置稳定。

画面内容：从狭窄门缝向教室内窥视，红色应急灯笼罩教室，一排学生背对黑板或背对门口站着，几乎一动不动。值夜老师站在讲台前，低头看旧点名册，手电光只在桌面或点名册上轻轻移动。

镜头运动：全程一个连续镜头，不切换场景。镜头极慢地向门缝内推进，推进幅度很小，让观众慢慢看清“第九节课”正在发生。门框两侧始终保持黑暗压迫，不要把镜头完全推入教室。

人物和动作：学生只允许有极轻微呼吸感或衣服晃动，不要转身，不要抬头，不要走动。老师可以轻微翻页或低头点名，但不要突然冲向镜头。

禁止项：不要让黑板出现文字，不要出现乱码，不要新增怪物或新角色，不要血腥，不要夸张恐怖脸，不要大幅变换灯光，不要改变门缝构图和教室布局。
```

建议时长：10-14 秒
剪辑重点：让观众看懂“不存在的第九节课”。

## 生成顺序

1. 先生成镜头01“老师宣布晚自习”，把开场逻辑补齐。
2. 再生成镜头02“林昼举病历撒谎”，确认口型和台词。
3. 再生成镜头05“陈雨歪脸回头”，验证恐怖感。
4. 再生成镜头07“林昼装瞎”，验证核心钩子。
5. 最后补齐镜头03、04、06、08。

## 失败重做标准

出现以下情况就重做该镜头，保存为对应 `_2.0`：

1. 人物脸型明显变了。
2. 新增无关人物或怪物。
3. 出现乱码文字。
4. 老师/林昼说话镜头没有明显口型。
5. 陈雨变成血腥怪物。
6. 林昼装瞎变成喜剧摔倒。
7. 老师眼睛露得太清楚。
8. 点名册文字不是“林昼，待观察”。
