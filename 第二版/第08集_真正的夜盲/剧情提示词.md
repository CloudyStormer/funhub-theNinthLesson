# 第8集剧情提示词：真正的夜盲

## 统一正向风格

```text
国漫写实漫画质感，深夜校园器材室悬疑恐怖，冷蓝黑暗、门缝人影、盲杖、黑色布带、旧座位表作为核心视觉符号，低饱和色彩，人物五官稳定，不血腥
```

## 统一负向限制

```text
不要血腥，不要怪物化，不要把真正夜盲学生画成超能力角色，不要乱码文字，不要夸张魔法效果，不要让门外学生变成怪物脸，不要新增无关人物
```

## 关键剧情提示词

### 01 跑进操场器材室

```text
深夜操场边，粉笔线从操场地面延伸到器材室门口；林昼沿粉笔线跑进门内，门半开，门外远处仍有学生方阵和红色应急灯，许眠被人群隔在后方。器材室内部球架、垫子、器材柜靠墙，门在林昼身后
```

### 02 盲杖声出现

```text
同一器材室黑暗角落，折叠盲杖轻轻点在地面，戴深色眼镜的17岁高中男生站在球架旁，校服干净，表情平静；林昼站在门内喘气，门在他身后半开，门缝外有操场红光和模糊人影
```

### 03 门缝外贴脸学生

```text
器材室门缝和门玻璃外，一排学生的脸贴在玻璃外往里看，表情空洞但不怪物化，不血腥；门仍关着，学生不能穿门进入。林昼在门内侧看得清楚，真正夜盲学生仍站在球架旁毫无反应
```

### 04 递黑色布带

```text
同一器材室，真正夜盲学生从球架旁靠近半步，把一条普通黑色布带递给林昼；林昼站在门口附近迟疑接过，门外人影仍在玻璃外，布带不发光，器材柜、门缝、球架位置不变
```

### 05 戴上布带听见错位

```text
林昼把黑色布带蒙在眼前，画面仍是器材室，不变成抽象幻觉；门外学生脚步声和玻璃外影子方向错位，真正夜盲学生站在旁边平静听着。门仍在身后，学生不能进来，不出现魔法特效
```

### 06 第九节课座位表

```text
器材室墙角旧座位表特写，纸张贴在墙上，不是黑板；上面有“第九节课”和“林昼”的核心文字感觉。林昼蒙着布带，伸手摸到座位表边缘，真正夜盲学生站在旁边说明规则
```

### 07 广播要求归位

```text
器材室旧广播喇叭响起，真正夜盲学生脸色第一次变了；林昼蒙眼站在旁边听，门外学生影子慢慢靠近，红色应急灯从门缝下扫进来。空间仍是器材室，不切回操场
```

### 08 交出盲杖

```text
同一器材室，真正夜盲学生把折叠盲杖塞给林昼，动作急促但克制；林昼蒙着黑色布带握住盲杖，门外人影仍在玻璃外逼近，门、球架、器材柜位置不变
```
