# 第8集 Clip提示词：空间连续版

## 本集收紧剧情

林昼从操场沿粉笔线进入器材室，真正夜盲学生在器材室内解释规则。门外学生、门缝、器材室内部要始终保持方向一致。

## 全集空间轴

- 操场边：粉笔线从操场地面延伸到器材室门口。
- 器材室内：球架、垫子、器材柜在四周，门在一侧，门缝外是操场方向。
- 真正夜盲学生：先在器材室角落出现，后面靠近林昼递布带和盲杖。

## 统一负向限制

```text
不要让器材室变成教室，不要让门外学生穿墙进来，不要魔法特效，不要新增怪物，不要乱码文字，不要让许眠突然进入器材室中央
```

## Clip 01 跑进器材室

```text
深夜操场边，林昼沿地上粉笔线跑到器材室门口，许眠被操场方阵人群隔在后方。林昼推门进入器材室，门外还能听见点名。器材室门和操场方向清楚，不要直接切到陌生房间。结尾门在林昼身后关到半开。
```

## Clip 02 盲杖声

```text
器材室内部黑暗，球架和垫子靠墙堆放，林昼站在门内喘气。角落传来盲杖点地声，真正夜盲学生站在球架旁，戴深色眼镜。门在林昼背后，门缝外有操场红光。结尾男生提醒别看门缝。
```

## Clip 03 门缝外学生

```text
器材室门缝视角，门外一排学生脸贴近门玻璃或门缝往里看，表情空洞但不怪物化。林昼看得清楚，真正夜盲学生却没有反应。门仍关着，学生不能穿门进来。结尾真正夜盲学生转向林昼解释夜盲。
```

## Clip 04 解释真正夜盲

```text
同一器材室，真正夜盲学生站在球架旁，林昼站在门口附近，两人保持距离对话。男生解释夜盲不是看不见黑，而是看不见伪装成人的东西。器材柜、门缝、球架位置不变。结尾男生拿出黑色布带。
```

## Clip 05 递黑色布带

```text
真正夜盲学生向林昼递出黑色布带，林昼迟疑接过。门外学生仍在门缝外，不进来。器材室冷蓝暗光，布带是普通布带，不发光。结尾林昼把布带蒙到眼前。
```

## Clip 06 戴布带听见错位

```text
林昼戴上黑色布带后，画面仍在器材室，不变成抽象幻觉。通过脚步影子和声音方向错位表现异常：门外学生脚步声和影子方向不一致。真正夜盲学生在旁边平静站着。结尾男生指向墙角旧座位表。
```

## Clip 07 第九节课座位表

```text
器材室墙角旧座位表特写，纸张贴在墙上，写着“第九节课”和“林昼”的感觉。林昼伸手摸到座位表边缘，真正夜盲学生站在旁边说明他是空着的座位。不要把座位表变成黑板。结尾广播响起。
```

## Clip 08 广播归位与交盲杖

```text
器材室广播喇叭响起“真正夜盲学生不在名单内”，真正夜盲学生表情第一次变化，把折叠盲杖塞给林昼。林昼蒙着布带接住盲杖，门外学生影子靠近但仍在门外。结尾林昼准备离开去第九节课。
```
