# 第8集首尾帧提示词

## Clip 01 首帧

```text
深夜操场边，林昼沿地上粉笔线跑向器材室门口，许眠被操场方阵人群隔在后方，器材室门在前方
```

## Clip 01 尾帧

```text
同一操场边，林昼已经推门进入器材室，门半开，门外还能看到操场红光和学生方阵远影，许眠仍在后方
```

## Clip 02 首帧

```text
器材室内部黑暗，林昼站在门内喘气，球架和垫子靠墙堆放，门在他身后半开，外面有操场红光
```

## Clip 02 尾帧

```text
同一器材室，角落里真正夜盲学生站在球架旁，戴深色眼镜，盲杖点地，林昼回头看向他
```

## Clip 03 首帧

```text
器材室门缝视角，门外学生影子刚靠近门玻璃，林昼在门内侧紧张看着门缝
```

## Clip 03 尾帧

```text
同一门缝，门外一排学生脸贴近玻璃或门缝往里看，表情空洞但不怪物化，真正夜盲学生仍无反应
```

## Clip 04

```text
同一器材室，真正夜盲学生站在球架旁解释规则，林昼站在门口附近听，器材柜、门缝、球架位置不变，门外学生仍在门外
```

## Clip 05 首帧

```text
器材室内，真正夜盲学生刚拿出普通黑色布带，林昼站在他面前迟疑，门外人影仍在门缝外
```

## Clip 05 尾帧

```text
同一器材室，林昼已经接过黑色布带，真正夜盲学生手收回，布带不发光，空间不变
```

## Clip 06 首帧

```text
器材室内，林昼刚把黑色布带蒙到眼前，真正夜盲学生站在旁边，门在后方
```

## Clip 06 尾帧

```text

```

## Clip 07 首帧
