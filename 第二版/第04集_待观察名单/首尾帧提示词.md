# 第4集首尾帧提示词

## Clip 01 首帧

```text
夜晚厕所出口位置，门已经变成黑板，值夜老师站在林昼身后递粉笔，林昼低头装作看不见，镜子在洗手台方向泛红，现实厕所仍是冷白瓷砖
```

## Clip 01 尾帧

```text
同一厕所黑板门前，林昼仍没有接粉笔，手摸墙假装找门，值夜老师保持递粉笔动作，黑板门和镜子位置不变
```

## Clip 02 首帧

```text
现实厕所和镜子同框，现实林昼刚抬手摸墙，镜中林昼在红光教室里准备模仿，值夜老师站在现实厕所后方
```

## Clip 02 尾帧

```text
同一镜面构图，镜中林昼已经提前接过粉笔，现实林昼还没接，镜中教室只存在镜面里，不占满现实厕所
```

## Clip 03 首帧

```text
厕所天花板旧广播喇叭特写，广播刚响起，值夜老师在下方抬头前一瞬，林昼仍在黑板门前
```

## Clip 03 尾帧

```text
同一厕所，值夜老师已经抬头看向广播，手里的点名册合上一点，黑板门边缘开始出现真实门缝
```

## Clip 04 首帧

```text
厕所黑板门边缘出现真实门缝，林昼闭眼准备冲出去，值夜老师停在门内不追，镜子仍泛红
```

## Clip 04 尾帧

```text
夜晚走廊，林昼已经从厕所门冲出，手扶墙跑向走廊深处，厕所门在他身后，红色应急灯在远处
```

## Clip 05 首帧

```text
夜晚走廊尽头，许眠站在阴影里，手握广播室钥匙，林昼从厕所方向跑来，走廊冷蓝，厕所门在后方
```

## Clip 05 尾帧

```text
同一走廊，林昼停在许眠身边喘气，许眠提醒别回头，宿舍方向在前方，厕所方向在后方，空间不反转
```

## Clip 06 首帧

```text
夜晚男生宿舍，林昼刚回到自己床位，其他同学睡在黑暗里不动，床下露出旧点名册一角
```

## Clip 06 尾帧

```text
同一宿舍床边，林昼已经弯腰把旧点名册从床下拖出，门口和窗户方向固定，许眠不进入宿舍明处
```

## Clip 07 首帧

```text
宿舍床边特写，旧点名册刚被翻开，林昼手指压在第一页边缘，窗外冷光照纸面
```

## Clip 07 尾帧

```text
同一点名册特写，第一页写着“许眠，已通过观察”，只出现这组核心文字，不要乱码，林昼手停在名字旁
```

## Clip 08 首帧

```text
男生宿舍，林昼刚从点名册上抬头看向窗玻璃，窗外是冷蓝夜色，床位和点名册仍在旁边
```

## Clip 08 尾帧

```text
同一宿舍窗玻璃反光，玻璃里映出许眠的脸，现实宿舍仍在画面内，窗户没有变成镜中教室，林昼震惊看着反光
```
