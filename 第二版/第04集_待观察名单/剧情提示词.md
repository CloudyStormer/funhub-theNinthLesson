# 第4集剧情提示词：待观察名单

## 统一正向风格

```text
国漫写实漫画质感，校园心理恐怖，厕所冷白灯、走廊冷蓝阴影、镜中红色应急灯对比强烈，人物五官稳定，镜面和黑板作为核心视觉符号，不血腥
```

## 统一负向限制

```text
不要血腥，不要怪物化，不要乱码文字，不要突然新增主要角色，不要让镜中林昼换脸，不要把广播室钥匙画成奇幻道具，不要大幅改变学校空间
```

## 关键剧情提示词

### 01 黑板门前签名

```text
夜晚学校厕所出口位置，原本的厕所门变成一整块黑板，黑板上写着“林昼，到”；值夜老师站在林昼身后把粉笔递到他面前，白衬衫、反光眼镜；林昼低头不接粉笔，伸手摸墙装作看不见，洗手台和镜子仍在背后，镜子里泛出红光
```

### 02 镜中林昼快半拍模仿

```text
现实厕所和镜子同框，现实林昼站在黑板门前僵住，手还没接粉笔；镜中红光教室里的镜中林昼外貌相似但校服更整齐，已经提前接过粉笔。镜中教室只在镜面内部，现实厕所仍是冷白瓷砖，不血腥
```

### 03 广播救场

```text
厕所天花板旧广播喇叭特写，广播只改变局势，不换场；值夜老师站在黑板门前停住，抬头看向广播，镜片反光更亮，手里的点名册微微合上；林昼仍在出口黑板门前，厕所空间不变
```

### 04 黑板门重新裂开门缝

```text
厕所出口的黑板门边缘缓慢出现真实门缝，红光从镜中压来；林昼闭着眼，手扶墙，从出口门缝冲进夜晚走廊，动作慌但不夸张；值夜老师停在厕所门内没有追，门和镜子位置不乱
```

### 05 许眠握着广播室钥匙

```text
夜晚走廊尽头，许眠站在阴影里，黑色低马尾，校服整齐，手里握着一串普通广播室钥匙；林昼从厕所方向跑来，在她身边停住喘气。厕所门在后方，宿舍方向在前方，许眠不进入厕所，只低声提醒别回头
```

### 06 床下旧点名册

```text
男生宿舍夜晚，林昼回到自己床位，跪在床边，从床下拖出一本旧点名册；床、门、窗位置固定，其他学生躺在各自床上不动，许眠不进入宿舍明处。手电冷光照亮点名册封面
```

### 07 许眠已通过观察

```text
旧点名册第一页特写，纸页上写着“许眠，已通过观察”，字迹清楚，只允许出现这组核心文字；林昼手指停在名字旁边，窗外冷蓝光照进宿舍，远处传来低频压迫感
```

### 08 窗玻璃里的许眠

```text
同一男生宿舍，林昼从点名册上抬头看向窗玻璃；窗玻璃反光里只映出许眠的脸，黑色低马尾，表情冷静。现实宿舍仍在画面内，窗外夜色冷蓝，玻璃反光不要变成镜中教室
```
