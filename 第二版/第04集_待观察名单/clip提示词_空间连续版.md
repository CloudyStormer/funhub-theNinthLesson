# 第4集 Clip提示词：空间连续版

## 本集收紧剧情

全集从厕所黑板门开始，许眠通过广播救林昼，林昼沿走廊逃回宿舍，最后在床下发现旧点名册。镜子、厕所门、走廊、宿舍必须顺序连接，不要来回跳空间。

## 全集空间轴

- 厕所：黑板门在出口位置，镜子在洗手台方向，值夜老师在林昼身后。
- 走廊：林昼从厕所门冲出，沿墙摸索，许眠在走廊尽头阴影里。
- 宿舍：林昼回到自己床位，旧点名册在床下。

## 统一负向限制

```text
不要让许眠突然出现在厕所内，不要让林昼瞬移回宿舍，不要让黑板门和镜子位置混乱，不要新增追逐人群，不要乱码文字
```

## Clip 01 黑板门前拒签

```text
夜晚厕所出口位置，门已经变成黑板，值夜老师站在林昼身后递粉笔。林昼低头不接粉笔，伸手摸墙装作找不到门。镜子在洗手台方向泛红，现实厕所仍是冷白瓷砖。结尾镜中林昼开始提前模仿他。
```

## Clip 02 镜中林昼快半拍

```text
镜子画面和现实林昼同框，现实林昼在厕所里慢慢抬手摸墙，镜中林昼却已经接过粉笔。镜中教室仍在镜面内部，不占满现实厕所。值夜老师站在现实厕所后方观察。结尾老师靠近一步，压迫林昼。
```

## Clip 03 广播救场

```text
厕所天花板旧广播喇叭响起，声音提示“夜盲学生请返回宿舍”。值夜老师停住，抬头看天花板，手里的点名册合上一点。林昼仍站在黑板门前。空间不变，广播只改变局势，不新增人物。结尾黑板门边缘出现真实门缝。
```

## Clip 04 逃出厕所

```text
黑板门边缘裂出真实厕所门缝，林昼闭着眼冲出厕所，手扶墙跑入夜晚走廊。走廊冷蓝，红色应急灯在远处，厕所门在他身后。值夜老师停在门内不追。结尾林昼沿走廊看到许眠站在远处阴影里。
```

## Clip 05 许眠走廊接应

```text
夜晚走廊尽头，许眠站在阴影里，手握广播室钥匙。林昼从厕所方向跑来，停在她身边喘气。许眠提醒别回头。走廊方向保持一致：厕所门在后方，宿舍方向在前方。结尾两人分开，林昼继续跑向宿舍。
```

## Clip 06 回宿舍发现床下异物

```text
夜晚男生宿舍，林昼回到自己床位，其他同学睡在黑暗里不动。床位、门口、窗户方向固定。林昼弯腰看床下，发现旧点名册露出一角。不要让许眠进入宿舍明处。结尾林昼把点名册拖出。
```

## Clip 07 点名册写着许眠

```text
宿舍床边特写，林昼翻开旧点名册第一页，纸页上写着“许眠，已通过观察”。只允许出现这组核心文字，不要乱码。手电或窗外冷光照纸面，周围仍是宿舍床下阴影。结尾林昼抬头看向窗户。
```

## Clip 08 窗玻璃里的许眠

```text
同一宿舍，林昼抬头看窗玻璃反光，玻璃里映出许眠的脸。现实宿舍仍在画面内，窗外是冷蓝夜色，不要把窗户变成镜中教室。许眠的反光表情冷静，像早知道点名册。结尾停在林昼震惊反应。
```
