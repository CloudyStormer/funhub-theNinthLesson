# 吉梦提示词七段模板

## 通用结构

```text
风格：

视角：

主体：

细节：

背景：

光彩：

质量词：
```

## 填写标准

```text
风格：校园悬疑，国漫写实漫画质感，低饱和但保留肤色和服装颜色，不要灰死，不要塑料感

视角：写清镜头位置、拍摄方向、景别、人物朝向；例如“镜头从后排看向讲台”“人物身体面向黑板”“侧背影”“不看镜头”

主体：写清主角是谁；参考图必须绑定角色，例如“参考图1=林昼，只用于林昼，不用于老师和路人”

细节：写清一个主要动作、表情、手里物品、站姿坐姿；不要多个动作同时发生

背景：写清真实地点和空间物件；例如黑板、讲台、课桌、门、窗、走廊方向

光彩：写清时间、主光源、颜色和阴影方向；例如白天自然光、夜晚冷色走廊灯、红色应急灯从门缝照出

质量词：高清，画面稳定，干净构图，角色一致，空间连续，真实物理关系，电影感，细节清晰
```

## 合并版模板

```text
风格：校园悬疑，国漫写实漫画质感，低饱和但保留真实肤色和服装颜色。
视角：【镜头位置】看向【空间方向】，【景别】，【人物朝向】，不看镜头。
主体：参考图1=【角色名】，只用于【角色名】；【角色名】位于【具体位置】；其他人物只做背景，不抢主位。
细节：【一个主要动作】，【表情】，【手里物品/姿态】，动作自然，符合真实物理。
背景：【具体地点】，【黑板/讲台/门窗/课桌/走廊等位置】，空间轴清楚，承接上一帧。
光彩：【时间】，【主光源】，【色彩】，【阴影方向】，光源符合真实空间。
质量词：高清，干净构图，电影感，角色一致，空间连续，细节清晰，不要字幕水印，不要乱码文字。
```
