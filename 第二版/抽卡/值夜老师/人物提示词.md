# 值夜老师抽卡提示词

```text
中年中国男教师，瘦高身材，白衬衫，深色裤子，金属边眼镜强反光，看不清眼睛，手持旧点名册和手电，僵硬微笑，红色应急灯照在半边脸上，说话客气但压迫感强，国漫恐怖写实风
```
