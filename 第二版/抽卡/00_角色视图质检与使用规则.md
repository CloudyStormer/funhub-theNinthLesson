# 第二版角色视图质检与使用规则

## 审查结论

现有抽卡可作为身份参考，但不能把每张都当成同等可靠的生产锚点：

- 所有现有旧图右下角带“可灵AI”水印，正式剧情帧必须明确禁止复制水印。
- 林昼、许眠、陈雨的正面、背面、左右侧面总体能辨认同一人，但各角色缺少最常用的三分之四视图。
- 许眠实际图片为低双马尾，旧文字“低马尾”已失效。
- 林昼正面新卡是人物左肩单肩背包，背面旧图却是双肩端正背包；正面/正全为无侧条纹黑裤，左右侧旧图却出现双白裤缝。第一集夜景统一左肩单肩背、无侧条纹黑裤，白天课堂不背。
- 值夜老师的红黑高反差图与学生白底设定图画风、清晰度差异大；生产时只锁定瘦高体型、短发、金属边反光眼镜、黑色外套和浅色衬衫，不复制原图的模糊边缘与水印。
- 左右视图必须按人物自身左右命名，不能把同一张图水平镜像后冒充另一侧。

## 生产规则

1. 参考图必须用“角色名 + 明确的仓库相对文件路径 + 用途”绑定，不再写不透明的“图一/图五”。
2. 多人画面只给每个核心角色一张最匹配镜头方向的锚点，避免总拼图里的身份互相污染。
3. 角色卡中的白背景、站姿、背包和手持道具不是剧情场景的一部分；提示词必须明确哪些只锁身份、哪些需要带入。
4. 正式帧禁止任何水印、Logo、角色卡白底和设定图边框。
