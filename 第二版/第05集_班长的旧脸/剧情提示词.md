# 第5集剧情提示词：班长的旧脸

## 统一正向风格

```text
国漫写实漫画质感，白天校园悬疑逐渐转向恐怖，荣誉墙玻璃反光、旧照片、医务室走廊作为核心视觉符号，低饱和冷色调，人物五官稳定，不血腥
```

## 统一负向限制

```text
不要血腥，不要怪物化，不要照片文字乱码，不要让旧照片变成夸张鬼脸，不要把许眠换成陌生发型，不要出现奇幻服装或夸张特效
```

## 关键剧情提示词

### 01 林昼怀疑许眠

```text
白天教室，林昼坐后排，身体面向黑板和讲台，只用余光看前排许眠的侧背影，表情警觉又困惑；许眠坐前排，低马尾，校服整齐，身体也面向黑板和讲台，没有回头。背景同学都面向前方正常上课，不看镜头，课堂冷光，画面带轻微压迫感
```

### 02 学校荣誉墙

```text
课间学校走廊荣誉墙，玻璃橱窗里贴着历届优秀学生照片，林昼站在墙前寻找许眠名字，走廊冷色自然光，学生从后方经过但虚化，荣誉墙玻璃有轻微反光，不要让文字变成乱码主体
```

### 03 三年前的许眠照片

```text
荣誉墙照片特写，照片下方标注许眠的感觉，照片里的女生也是黑色低马尾和校服，但脸和现在的许眠明显不同，照片年代感轻微发黄，玻璃反光压住画面，恐怖感克制
```

### 04 许眠出现在身后

```text
林昼后退时撞到身后的许眠，许眠黑色低马尾，表情冷静但眼神里有一丝疲惫，背景是荣誉墙旧照片，玻璃里同时映出现许眠和旧照片许眠，构图制造真假对照
```

### 05 旧照片嘴角动

```text
荣誉墙玻璃特写，旧照片里的许眠嘴角像轻轻动了一下，但仍然保持照片质感，不变成真人，不血腥；现实许眠的倒影叠在玻璃上，冷光下气氛诡异
```

### 06 广播要求归还旧脸

```text
白天走廊旧广播喇叭特写，广播响起时走廊声音突然变空，许眠和林昼同时抬头，远处同学像没听见一样继续走动，荣誉墙玻璃轻微震动，校园悬疑恐怖
```

### 07 走向医务室

```text
许眠拉住林昼手腕，沿着冷色走廊快步走向医务室方向，许眠表情克制但明显紧张，林昼回头看荣誉墙，走廊尽头光线变暗，医务室门牌隐约可见
```

### 08 值夜老师守在医务室门口

```text
走廊尽头医务室门口，值夜老师瘦高身影站在阴影里，白衬衫，金属边眼镜反光，手里翻开旧点名册，林昼和许眠停在远处，白天走廊却出现夜晚红色应急灯反光，压迫感强
```
