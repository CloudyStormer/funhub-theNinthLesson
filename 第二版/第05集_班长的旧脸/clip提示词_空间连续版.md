# 第5集 Clip提示词：空间连续版

## 本集收紧剧情

白天从教室怀疑许眠，移动到荣誉墙确认旧脸，再沿同一条走廊转向医务室。荣誉墙玻璃反光和旧照片是核心，不要让照片变成活人实体。

## 全集空间轴

- 教室：林昼坐后排，身体面向黑板和讲台；许眠坐前排，身体也面向黑板和讲台。林昼只能用余光看许眠侧背影，许眠不整个人转身。
- 荣誉墙走廊：照片在玻璃橱窗内，现实人物站在橱窗前。
- 医务室走廊：荣誉墙在后方，医务室在走廊尽头。

## 统一负向限制

```text
不要让旧照片走出玻璃，不要让许眠瞬移到不同走廊，不要让荣誉墙文字乱码成主体，不要新增围观学生，不要把白天走廊突然变成夜晚教室
```

## Clip 01 教室里怀疑许眠

```text
白天教室，林昼坐后排，身体面向黑板和讲台，只用余光看前排许眠的侧背影；许眠坐前排，身体也面向黑板和讲台，没有回头，只低声提醒别用那种眼神看她。其他同学都面向前方正常上课，不看镜头、不围观。结尾林昼视线落到许眠背影，切到课间走廊。
```

## Clip 02 林昼来到荣誉墙

```text
课间学校走廊，荣誉墙玻璃橱窗在墙边，林昼站在橱窗前查看历届优秀学生照片。路人学生从走廊远处经过，不停下围观。玻璃反光轻微，不遮挡照片。结尾镜头推向“许眠”旧照片。
```

## Clip 03 旧照片里的许眠

```text
荣誉墙照片特写，照片在玻璃后方，照片里的许眠是三年前模样，低马尾但脸和现在不同。文字只保留“许眠”的感觉，不要乱码。林昼手指停在玻璃前，不触碰照片实体。结尾林昼后退一步。
```

## Clip 04 现实许眠出现在身后

```text
同一荣誉墙走廊，林昼后退时撞到身后的许眠。许眠站在他身后半步，现实许眠和玻璃里的旧照片同框形成对比。路人继续远处经过，不围观。空间不变，荣誉墙仍在一侧。结尾两人面对荣誉墙对话。
```

## Clip 05 旧照片嘴角动

```text
荣誉墙玻璃特写，旧照片里的许眠嘴角像轻轻动了一下，但仍然是照片，不变成真人，不走出玻璃。现实许眠的倒影叠在玻璃上，林昼站在旁边看见异常。结尾走廊广播响起。
```

## Clip 06 广播归还旧脸

```text
白天走廊广播喇叭特写，声音要求许眠归还旧脸。林昼和许眠同时抬头，远处路人像没听见一样继续走。荣誉墙仍在他们身后，不突然换地点。结尾许眠拉住林昼往医务室方向走。
```

## Clip 07 走向医务室

```text
同一条走廊，许眠拉着林昼从荣誉墙位置离开，沿走廊朝医务室方向快步走。荣誉墙在后方远离，医务室门牌在前方逐渐出现。两人动作真实，不奔跑夸张。结尾走廊尽头出现值夜老师影子。
```

## Clip 08 医务室门口的值夜老师

```text
走廊尽头医务室门口，值夜老师站在阴影里，手里翻开点名册。林昼和许眠停在走廊中段，不要瞬间靠到老师身边。白天走廊出现不自然红色反光但空间仍是走廊。结尾老师说旧脸不能私藏。
```
