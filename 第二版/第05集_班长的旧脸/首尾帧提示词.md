# 第5集首尾帧提示词

## Clip 01

```text
白天教室，林昼坐后排，身体面向黑板和讲台，只用余光看前排许眠侧背影；许眠坐前排，身体面向黑板和讲台，没有回头，只低声提醒，其他同学都面向前方正常上课，不围观
```

## Clip 02 首帧

```text
课间学校走廊，林昼刚来到荣誉墙前，玻璃橱窗在墙边，路人学生从远处经过，冷色自然光
```

## Clip 02 尾帧

```text
同一荣誉墙走廊，林昼停在橱窗前查看历届照片，手靠近玻璃，路人不围观，荣誉墙空间稳定
```

## Clip 03

```text
荣誉墙照片特写，照片在玻璃后方，照片里的许眠是三年前模样，低马尾但脸和现在不同，文字只保留“许眠”的感觉，不要乱码
```

## Clip 04 首帧

```text
荣誉墙走廊，林昼看完照片后后退一步，许眠刚出现在他身后半步，玻璃橱窗仍在一侧
```

## Clip 04 尾帧

```text
同一走廊，林昼已经撞到身后的许眠，现实许眠和玻璃里的旧照片同框，路人仍在远处经过，不围观
```

## Clip 05 首帧

```text
荣誉墙玻璃特写，旧照片里的许眠仍是静态照片，现实许眠的倒影叠在玻璃上
```

## Clip 05 尾帧

```text
同一玻璃特写，旧照片里的许眠嘴角像轻轻动了一下，但仍然是照片，不走出玻璃，不变真人
```

## Clip 06 首帧

```text
白天走廊广播喇叭特写，荣誉墙在下方或后方可见，广播刚响起，走廊学生还在正常经过
```

## Clip 06 尾帧

```text
同一走廊，林昼和许眠同时抬头听广播，远处路人像没听见一样继续走，荣誉墙仍在他们身后
```

## Clip 07 首帧

```text
荣誉墙走廊，许眠刚拉住林昼手腕，准备离开荣誉墙位置，医务室方向在走廊前方
```

## Clip 07 尾帧

```text
同一走廊，许眠拉着林昼已经朝医务室方向走出几步，荣誉墙在后方远离，医务室门牌在前方隐约出现
```

## Clip 08

```text
走廊尽头医务室门口，值夜老师站在阴影里翻开点名册，林昼和许眠停在走廊中段，白天走廊出现不自然红色反光，空间仍是真实走廊
```
