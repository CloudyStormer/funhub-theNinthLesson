# 第9集 Clip提示词：空间连续版

## 本集收紧剧情

林昼主动进入第九节课。空间以三楼走廊和高二三班教室为核心：门口、讲台、黑板、学生座位、点名册必须方向一致。

## 全集空间轴

- 三楼走廊：许眠和林昼在教室门外，门在前方。
- 第九节课教室：讲台和黑板在前，学生/备用座位在后排，门在后侧；学生默认身体朝向讲台和黑板，只在剧情要求时转头看林昼，身体和座位不移动。
- 林昼路线：走廊门口进入教室，低头避开脸，走到讲台前点名册。

## 统一负向限制

```text
不要让教室布局反转，不要让林昼瞬移到讲台，不要让备用林昼提前出现到门口，不要乱码文字，不要让许眠进入教室深处
```

## Clip 01 门外准备

```text
夜晚十一点前，教学楼三楼走廊，林昼和许眠站在高二三班门外，门在画面前方，红光从门缝微弱漏出。林昼手里拿真正夜盲学生的盲杖和黑色布带，许眠低声交代规则。走廊空无一人。结尾十一点红光增强。
```

## Clip 02 第九节课开门

```text
十一点整，高二三班教室门自动打开，红色应急灯从门内照出。镜头从走廊看进教室，讲台和黑板在教室远处，值夜老师站在讲台前。学生座位和课桌排列固定。林昼仍在门外，没有瞬间进屋。结尾老师点名林昼。
```

## Clip 03 林昼蒙眼进教室

```text
林昼蒙上黑色布带，扶着盲杖从后门进入教室，路线从门口走向讲台。学生坐在座位上，身体仍朝向讲台和黑板，只把头转向林昼，身体和座位不移动；林昼低头只看地面，不看脸。许眠留在门外或走廊边，不进入教室深处。结尾林昼走到教室中段。
```

## Clip 04 只看桌脚鞋尖

```text
教室低角度视角，林昼用盲杖探路，只看课桌腿、学生鞋尖和地面粉笔线。讲台方向在前，门在后，空间不反转。学生身体仍朝向讲台和黑板，只有头部轻微转向林昼，座位不移动，脸不抢画面。结尾林昼到达讲台边。
```

## Clip 05 点名册自动翻页

```text
讲台桌面特写，旧点名册自动翻到林昼那一页，红色应急灯照纸面。值夜老师站在讲台后，林昼站在讲台前侧，位置清楚。字迹像往纸里钻，但不要生成乱码。结尾林昼伸手拿钢笔试图划掉名字。
```

## Clip 06 改名失败

```text
同一讲台桌面，林昼用钢笔划掉名字却写不出横线，纸上反复出现“林昼”的感觉。值夜老师的手压在点名册旁边。林昼仍蒙眼，不看老师脸。讲台、黑板、点名册位置不变。结尾老师说名字不能删，只能换。
```

## Clip 07 黑板备用座位

```text
镜头从讲台转向黑板，黑板在讲台后方，出现一排座位编号和“备用林昼”的感觉。林昼站在讲台旁，值夜老师在讲台后。文字简洁，不要乱码。结尾镜头转向教室后排。
```

## Clip 08 备用林昼坐直

```text
教室后排，几个备用林昼坐在不同座位上慢慢坐直，外貌相似但气质不同。后排位置固定，不要让他们瞬移到讲台。普通学生保持静止。红光来自教室应急灯。结尾值夜老师把粉笔放到林昼手里。
```

## Clip 09 选择替身

```text
讲台旁，值夜老师把粉笔放进林昼手里，黑板上的备用座位在他身后，后排备用林昼安静等待。林昼站在原地，手握粉笔但还没有写。门、讲台、黑板、后排方向保持一致。结尾停在“选一个”的压迫感。
```
