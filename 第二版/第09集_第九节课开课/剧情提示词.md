# 第9集剧情提示词：第九节课开课

## 统一正向风格

```text
国漫写实漫画质感，红光第九节课教室，低饱和冷蓝灰绿色调，红色应急灯为主要恐怖光源，点名册、黑色布带、盲杖、备用座位作为核心视觉符号，人物五官稳定，不血腥
```

## 统一负向限制

```text
不要血腥，不要怪物化，不要乱码文字，不要让备用林昼变成陌生人，不要改变主角校服设定，不要让教室变成奇幻空间，不要大规模动作打斗
```

## 关键剧情提示词

### 01 课前走廊准备

```text
夜晚十一点前的教学楼三楼走廊，林昼和许眠站在高二三班门外，门在前方，红光从门缝微弱漏出；林昼手握折叠盲杖和黑色布带，许眠站在旁边低声交代规则，走廊空无一人
```

### 02 第九节课开门

```text
十一点整，高二三班教室门自动打开，红色应急灯从门内涌出；镜头从走廊看进教室，讲台和黑板在远处，课桌排列整齐，学生坐在座位上身体朝讲台和黑板，值夜老师在讲台前低头点名，门外林昼停住不进屋
```

### 03 林昼蒙眼进教室

```text
林昼用黑色布带蒙住眼睛，手持折叠盲杖，从后门慢慢走向讲台；所有学生坐在座位上，身体仍朝讲台和黑板，只把头轻微转向他，座位不移动。林昼低头只看地面和盲杖，不看脸
```

### 04 只看鞋尖和桌脚

```text
第九节课教室低角度视角，林昼视线只落在学生鞋尖、课桌腿和地面粉笔线，周围学生身体仍朝讲台和黑板，只有头部轻微转向林昼，脸不入画，红色应急灯闪烁，盲杖轻轻点地，保持紧张克制
```

### 05 点名册自动翻页

```text
讲台桌面特写，旧点名册自动翻到林昼那一页，纸页在红光下轻轻抖动；林昼站在讲台前侧，值夜老师站在讲台后方，位置清楚。文字只保留“林昼”的核心，不要乱码
```

### 06 改名失败

```text
同一讲台桌面，林昼试图用钢笔划掉点名册上的名字，但钢笔写不出横线，纸上反复出现“林昼”的感觉；值夜老师的手压在点名册旁边，林昼仍蒙眼不看老师脸，讲台和黑板位置不变
```

### 07 黑板备用座位

```text
红光教室黑板上出现一排座位编号，每个编号下面有“备用林昼”的感觉；黑板在讲台后方，林昼站在讲台旁，黑色布带蒙眼，粉笔被塞到手里。文字简洁清楚，不要生成大量乱码
```

### 08 备用林昼坐直

```text
教室后排几个和林昼相似的备用林昼坐在不同座位上，慢慢坐直，身体仍朝讲台和黑板，不离开座位；一个校服整齐像好学生，一个眼神空洞，一个标准微笑像镜中林昼，外貌相似但气质不同，不血腥
```

### 09 选择替身

```text
讲台旁，值夜老师把粉笔放到林昼手里，反光眼镜看不清眼睛；林昼站在讲台边，黑色布带、盲杖、点名册和黑板备用座位同时入画，后排备用林昼坐在座位上安静等待，门、讲台、黑板方向一致
```
