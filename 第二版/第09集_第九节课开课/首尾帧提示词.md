# 第9集首尾帧提示词

## Clip 01 首帧

```text
夜晚十一点前，教学楼三楼走廊，林昼和许眠站在高二三班门外，门在前方，红光从门缝微弱漏出，林昼手拿盲杖和黑色布带
```

## Clip 01 尾帧

```text
同一走廊，许眠已经交代完规则，林昼握紧盲杖和黑色布带，教室门缝红光增强，走廊空无一人
```

## Clip 02 首帧

```text
高二三班门外，十一点整，教室门刚开始自动打开，红光从门缝涌出，林昼仍在门外
```

## Clip 02 尾帧

```text
同一门口，教室门已经打开，镜头从走廊看进教室，讲台和黑板在远处，值夜老师站在讲台前点名
```

## Clip 03 首帧

```text
教室后门口，林昼刚蒙上黑色布带，手扶盲杖，准备从门口走进红光教室，许眠留在门外
```

## Clip 03 尾帧

```text
同一教室，林昼已经走进教室中段，学生身体仍朝讲台和黑板，只把头转向林昼，座位不移动，林昼低头不看脸，门仍在他身后
```

## Clip 04

```text
第九节课教室低角度，林昼只看课桌腿、学生鞋尖和地面粉笔线，盲杖轻点地面，讲台方向在前，门在后，学生身体朝讲台和黑板，座位不移动
```

## Clip 05 首帧

```text
讲台桌面，旧点名册刚开始自动翻页，林昼站在讲台前侧，值夜老师站在讲台后，红色应急灯照纸面
```

## Clip 05 尾帧

```text
同一讲台桌面，点名册已经翻到林昼那一页，字迹像往纸里钻，不要乱码，林昼伸手准备拿钢笔
```

## Clip 06 首帧

```text
讲台桌面，林昼刚用钢笔划向自己的名字，值夜老师的手压在点名册旁边，林昼仍蒙眼不看老师
```

## Clip 06 尾帧

```text
同一桌面，钢笔没有划出横线，纸上反复出现“林昼”的感觉，讲台和黑板位置不变
```

## Clip 07 首帧

```text
讲台前，镜头刚从点名册转向黑板，黑板还保持原状，林昼和值夜老师在讲台旁
```

## Clip 07 尾帧

```text
同一黑板，黑板上出现一排座位编号和“备用林昼”的感觉，文字简洁，不要乱码，林昼站在讲台旁
```

## Clip 08 首帧

```text
教室后排，几个备用林昼坐在不同座位上，刚开始慢慢坐直，红光从教室应急灯照来
```

## Clip 08 尾帧

```text
同一后排座位，多个备用林昼已经坐直，外貌相似但气质不同，仍在后排，不瞬移到讲台
```

## Clip 09

```text
讲台旁，值夜老师把粉笔放进林昼手里，黑板上的备用座位在他身后，后排备用林昼安静等待，林昼还没有选择
```
