# 第6集剧情提示词：医务室没有医生

## 统一正向风格

```text
国漫写实漫画质感，校园医务室悬疑恐怖，冷白检查灯、旧病历柜、冷柜档案盒、病历纸作为核心视觉符号，低饱和冷蓝灰绿色调，人物五官稳定，不血腥
```

## 统一负向限制

```text
不要血腥器官，不要手术恐怖，不要怪物化，不要乱码文字，不要把医务室画成医院手术室，不要让档案盒和病历出现大量不可控文字，不要新增医生角色
```

## 关键剧情提示词

### 01 傍晚潜入医务室

```text
傍晚学校医务室门口，走廊拐角在侧后方，医务室门在前方；林昼和许眠躲在拐角观察，值夜老师瘦高影子刚从门前离开。两人等老师离开后再靠近门，不直接切到室内，门牌清晰但不要乱码
```

### 02 空无医生的医务室

```text
医务室内部，镜头从门口向内看；病历柜靠墙，检查床在一侧，桌面在中间，冷柜在最里面，没有医生。林昼站在门口警觉，许眠先走向病历柜，空间真实，不出现手术设备
```

### 03 病历柜标签

```text
旧病历柜近景，病历柜靠墙，检查床和桌面在背景固定位置；抽屉标签分别有头痛、失声、夜盲、失忆的感觉，下面夹着学生姓名卡片但不要生成复杂乱码。林昼的手停在“夜盲”抽屉前
```

### 04 谎言变成病

```text
医务室桌面旧病历特写，这本不是林昼的病历，用来说明规则；第一页是普通请假理由“嗓子疼”的感觉，下一页变成正式诊断“失声观察”的感觉，不要大段乱码。林昼的指尖压住纸边，许眠站在桌边，背景病历柜位置不变
```

### 05 脸部档案盒

```text
医务室最里面的冷柜打开，冷白雾气轻微散出，里面整齐摆放白色档案盒，每个盒子贴着姓名标签；许眠站在冷柜前，林昼在她身后半步，冷柜位置固定在房间深处，不血腥，不出现器官
```

### 06 许眠原件

```text
白色档案盒特写，标签上写着“许眠，原件”的感觉；许眠的手停在盒子前，指尖发白，林昼在旁边看着她。档案盒不要打开，不出现脸或器官，冷柜和医务室空间保持真实
```

### 07 林昼新病历

```text
同一医务室桌面在检查灯下亮起，一本新病历滑到桌面中央；林昼和许眠从冷柜方向回头看。病历上写着“林昼，重度夜盲，建议永久留校观察”，文字尽量清楚且不要添加其他乱码
```

### 08 视力表消失

```text
医务室墙上的视力表一行行变黑消失，林昼站在检查灯前眯眼，像真的开始看不清；许眠站在他旁边，立刻伸手捂住他的眼睛，提醒他不要承认。视力表、桌面、病历柜位置不变
```

### 09 门外补登记

```text
医务室门内视角，林昼和许眠站在室内后方，没有开门；门外只有值夜老师的黑色影子和手电光。一张新的病历纸从门缝下缓慢滑进来，纸上只出现“补登记”和“值夜老师”的核心感觉，老师不要进屋
```
