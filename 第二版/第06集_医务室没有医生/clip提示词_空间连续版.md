# 第6集 Clip提示词：空间连续版

## 本集收紧剧情

林昼和许眠从医务室门口进入室内，依次看到病历柜、冷柜档案盒、林昼新病历。所有异常都发生在同一个医务室，不要跳到医院或手术室。

## 全集空间轴

- 医务室门口：走廊拐角在外，医务室门在前方。
- 医务室内部：病历柜靠墙，检查床在一侧，桌面在中间，冷柜在最里面。
- 门外值夜老师：只通过门缝、敲门声和影子出现，不直接瞬移进室内。

## 统一负向限制

```text
不要把医务室画成医院手术室，不要出现血腥器官，不要新增医生，不要让病历柜和冷柜位置乱换，不要乱码文字，不要让值夜老师瞬移到室内
```

## Clip 01 医务室门口潜入

```text
傍晚学校医务室门口，林昼和许眠躲在走廊拐角，看着值夜老师影子从医务室门前离开。走廊冷色，医务室门牌在前方，门和拐角方向清楚。两人等老师离开后靠近门，不直接切到室内。结尾许眠拿钥匙开门。
```

## Clip 02 打开医务室

```text
医务室门被许眠打开，镜头从门口向内看。室内没有医生，病历柜靠墙，检查床在一侧，桌面在中间，冷白灯照亮空间。林昼站在门口警觉，许眠先进入。空间真实，不出现手术设备。结尾两人走向病历柜。
```

## Clip 03 病历柜标签

```text
医务室靠墙病历柜近景，抽屉标签有头痛、失声、夜盲、失忆的感觉，标签不要乱码成主体。林昼站在柜前抽出一本病历，许眠在旁边观察。检查床和桌面在背景同一位置。结尾林昼翻开病历。
```

## Clip 04 谎言变病历

```text
同一医务室桌面，林昼把一本旧病历放在桌上翻页，这本不是林昼的新病历，只用来说明规则。第一页是普通请假理由“嗓子疼”的感觉，下一页变成正式诊断“失声观察”的感觉。纸面只保留核心词，不生成大段乱码。许眠站在桌边，表情严肃。灯光冷白，病历柜仍在背景。结尾许眠走向最里面冷柜。
```

## Clip 05 冷柜里的脸部档案

```text
医务室最里面冷柜打开，里面不是药，而是整齐白色档案盒。许眠站在冷柜前，林昼在她身后半步。冷柜位置固定在房间深处，冷白雾气轻微，不血腥。结尾许眠看到“许眠，原件”的档案盒。
```

## Clip 06 许眠原件档案盒

```text
冷柜内部档案盒特写，标签上有“许眠，原件”的感觉。许眠伸手去拿，手微微发抖，林昼在旁边看见。档案盒不要打开，不出现脸或器官。结尾医务室灯突然熄灭，只剩检查灯。
```

## Clip 07 林昼新病历出现

```text
医务室桌面在检查灯下亮起，一本新病历自动滑到桌面中央。林昼和许眠从冷柜方向回头看，空间仍是同一房间。病历上写“林昼，重度夜盲，建议永久留校观察”的感觉，文字尽量清楚，不加乱码。结尾林昼眼前灯光变暗。
```

## Clip 08 视力表变黑

```text
医务室墙上的视力表一行行变黑消失，林昼站在检查灯前眯眼，像真的看不清。许眠立刻伸手捂住他的眼睛，让他别承认。视力表、桌面、病历柜空间关系不变。结尾门外响起敲门声。
```

## Clip 09 门缝补登记

```text
医务室门内视角，门外响起值夜老师敲门声，门缝下滑进一张新病历纸。林昼和许眠站在室内后方，没有开门。门外只见老师影子和手电光，不让老师瞬移进来。纸上只保留“补登记”和“值夜老师”的感觉。结尾压在门缝纸张上。
```
