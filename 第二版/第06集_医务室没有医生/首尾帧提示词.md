# 第6集首尾帧提示词

## Clip 01 首帧

```text
傍晚学校医务室门口，林昼和许眠躲在走廊拐角，值夜老师影子还停在医务室门前，门牌在前方，冷色走廊
```

## Clip 01 尾帧

```text
同一医务室门口，值夜老师影子已经离开，林昼和许眠从拐角靠近门，许眠拿出钥匙准备开门
```

## Clip 02 首帧

```text
医务室门刚被打开，镜头从门口向内看，病历柜靠墙，检查床在一侧，桌面在中间，室内没有医生
```

## Clip 02 尾帧

```text
同一医务室，林昼站在门口警觉，许眠已经先走进室内，冷白灯照亮病历柜和检查床，空间真实
```

## Clip 03 首帧

```text
医务室病历柜前，林昼站在柜前准备拉开抽屉，许眠在旁边观察，检查床和桌面在背景固定位置
```

## Clip 03 尾帧

```text
同一病历柜前，林昼已经抽出一本病历，柜子标签有头痛、失声、夜盲、失忆的感觉，不要乱码主体
```

## Clip 04 首帧

```text
医务室桌面，林昼刚把一本旧病历放到桌上，这本不是林昼的新病历，许眠站在桌边，病历柜仍在背景
```

## Clip 04 尾帧

```text
同一桌面，旧病历翻到诊断页，显示“嗓子疼”变成“失声观察”的感觉，纸面只保留核心词，不要大段乱码
```

## Clip 05 首帧

```text
医务室最里面冷柜前，许眠刚打开冷柜门，林昼站在她身后半步，冷白光从柜内透出
```

## Clip 05 尾帧

```text
同一冷柜，里面整齐摆放白色档案盒，许眠站在冷柜前，林昼在身后，不血腥，不出现器官
```

## Clip 06 首帧

```text
冷柜档案盒特写，许眠的手刚伸向写有“许眠，原件”的盒子，手微微发抖
```

## Clip 06 尾帧

```text
同一冷柜，许眠手停在“许眠，原件”档案盒前，档案盒没有打开，医务室灯开始变暗
```

## Clip 07 首帧

```text
医务室桌面，检查灯突然亮起，桌面中央还没有病历，林昼和许眠从冷柜方向回头看
```

## Clip 07 尾帧

```text
同一桌面，一本新病历已经滑到检查灯下，写着“林昼，重度夜盲，建议永久留校观察”的感觉，文字尽量清楚
```

## Clip 08 首帧

```text
医务室墙上视力表，林昼站在检查灯前眯眼，许眠站在旁边，视力表还完整
```

## Clip 08 尾帧

```text
同一医务室，视力表一行行变黑，许眠已经伸手捂住林昼眼睛，提醒他别承认
```

## Clip 09 首帧

```text
医务室门内视角，门外响起敲门声，门缝下还没有纸，林昼和许眠站在室内后方
```

## Clip 09 尾帧

```text
同一门内视角，一张新病历纸已经从门缝下滑进来，门外只有值夜老师影子和手电光，老师没有进屋
```
