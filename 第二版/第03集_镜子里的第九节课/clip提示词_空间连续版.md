# 第3集 Clip提示词：空间连续版

## 本集收紧剧情

白天在楼梯角解释第九节课和镜子规则；夜晚固定在学校厕所。现实厕所和镜中教室是两个空间：现实是冷白灯、瓷砖、洗手台；镜中是红光教室、讲台、学生队伍。不要让现实厕所直接变成教室，必须通过镜面过渡。

## 全集空间轴

- 白天楼梯角：许眠和林昼靠近墙边说话，路人从楼梯经过。
- 夜晚厕所：洗手台和镜子在同一面墙，林昼洗脸时身体正对洗手台和镜子；厕所隔间在侧后方，门在林昼转身逃跑方向。
- 镜中教室：只存在镜面里，红光、讲台、学生队伍和镜中林昼都在镜中。

## 统一负向限制

```text
不要让镜中教室直接占满现实空间，不要让厕所门位置乱变，不要新增学生进入厕所，不要让镜中林昼换脸，不要血腥，不要乱码文字
```

## Clip 01 楼梯角解释规则

```text
白天课间楼梯角，许眠和林昼靠墙站在楼梯平台一侧，楼梯向上和向下方向清楚。许眠压低声音解释第九节课，林昼站在她旁边听。路人学生只从楼梯远处经过，不停下围观。结尾许眠拿出旧照片，镜头推向两人手部。
```

## Clip 02 旧照片与镜子警告

```text
同一楼梯角，许眠把旧照片递给林昼，照片里陈雨的旧脸和现在不同，但文字不要乱码。林昼低头看照片，许眠警告不要照镜子。两人位置不变，楼梯背景不突然切换教室。结尾林昼收起照片，夜晚切到厕所洗手台。
```

## Clip 03 夜晚厕所洗脸

```text
夜晚学校厕所，现实空间清楚：洗手台在画面前方，镜子在洗手台上方，隔间门在侧后方，厕所出口在林昼后侧。林昼站在洗手台前洗脸，身体正对洗手台和镜子，冷白灯闪烁，厕所没有其他学生。结尾林昼抬头看镜子。
```

## Clip 04 镜中出现第九节课

```text
同一厕所镜子视角，现实中林昼身体正对镜子站在洗手台前，但镜中映出的不是厕所，而是红光第九节课教室。镜面边框必须保留，说明教室只在镜子里。镜中讲台、黑板、学生队伍位置稳定，现实厕所仍在镜子外。结尾镜中队伍最后出现另一个林昼。
```

## Clip 05 镜中林昼抬头

```text
镜中教室空间不变，镜中林昼站在队伍最后，校服比现实林昼更整齐，缓慢抬头露出标准微笑。现实林昼在镜外后退半步，洗手台仍在身前。镜中学生保持静止，值夜老师不突然出现在前景。结尾厕所灯开始熄灭。
```

## Clip 06 值夜老师从隔间走出

```text
现实厕所，隔间门从侧后方缓慢打开，值夜老师从隔间里走出，手拿旧点名册和手电，镜片反光。林昼从正对镜子的姿态转身看向隔间，镜子里仍有红光教室，现实厕所冷白灯一盏盏熄灭。老师从隔间到洗手台之间的走位真实，不瞬移。结尾林昼再转身冲向厕所门。
```

## Clip 07 厕所门变成黑板

```text
现实厕所出口方向，林昼冲向门口，却发现原本厕所门变成黑板，黑板在门的位置，不要把整面墙都变成教室。黑板上只有“林昼，到”的感觉，不要其他乱码。林昼手停在黑板前，背后镜子仍泛红。结尾老师把粉笔递到他面前。
```

## Clip 08 影子替他举手

```text
同一厕所门口和镜面空间，值夜老师把粉笔递给林昼，林昼低头看见自己的影子在镜子里举手。镜中林昼和现实林昼保持外貌关联，镜中动作像提前发生。现实厕所、镜中教室、黑板门三者位置都要清楚，不混成一个空间。结尾镜中林昼说替他签。
```
