# 第3集首尾帧提示词

## Clip 01

```text
白天课间楼梯角，许眠和林昼靠墙站在楼梯平台一侧，楼梯上下方向清楚，许眠压低声音解释第九节课，路人学生只从远处经过，不围观
```

## Clip 02 首帧

```text
同一楼梯角，许眠刚把旧照片拿出来，林昼低头看她手里的照片，楼梯背景不变，路人虚化经过
```

## Clip 02 尾帧

```text
同一楼梯角，林昼已经接过旧照片，照片里陈雨旧脸和现在不同但文字不乱码，许眠警告不要照镜子
```

## Clip 03 首帧

```text
夜晚学校厕所，林昼站在洗手台前，身体正对洗手台和镜子，镜子在洗手台上方，隔间在侧后方，厕所出口在林昼后侧，冷白灯闪烁
```

## Clip 03 尾帧

```text
同一厕所，林昼洗完脸抬头看镜子，身体仍正对洗手台和镜子，现实厕所空无一人，洗手台和镜子位置不变
```

## Clip 04 首帧

```text
厕所镜子视角，镜面边框清楚，现实林昼正对镜子站在洗手台前，镜中刚开始泛出红光，现实厕所仍在镜外
```

## Clip 04 尾帧

```text
同一镜子，镜中已经出现红光第九节课教室，讲台、黑板、学生队伍都只在镜面内部，现实厕所没有变成教室
```

## Clip 05 首帧

```text
镜中红光教室，队伍最后站着镜中林昼，他还低着头，现实林昼在镜外后退半步，洗手台仍在身前
```

## Clip 05 尾帧

```text
同一镜中教室，镜中林昼已经抬头露出标准微笑，现实林昼惊恐后退，厕所灯开始熄灭，镜面边框仍保留
```

## Clip 06 首帧

```text
现实厕所隔间门刚开始打开，值夜老师还在隔间阴影内，镜子里红光教室仍然存在，林昼站在洗手台前，身体刚从镜子方向转向隔间
```

## Clip 06 尾帧

```text
同一厕所，值夜老师已经从隔间走出，手拿点名册和手电，镜片反光，厕所灯更暗，林昼转身准备逃向门口
```

## Clip 07 首帧

```text
现实厕所出口方向，林昼冲向门口，门刚开始变成黑板，背后镜子泛红，值夜老师在后方
```

## Clip 07 尾帧

```text
同一厕所出口，厕所门已经完全变成黑板，黑板上只有“林昼，到”的感觉，林昼手停在黑板前
```

## Clip 08 首帧

```text
厕所门口和镜面同框，值夜老师刚把粉笔递向林昼，林昼低头不接，镜中红光仍在
```

## Clip 08 尾帧

```text
同一空间，林昼的影子在镜子里举手，镜中林昼标准微笑，现实林昼僵住，厕所和镜中教室位置都清楚
```
