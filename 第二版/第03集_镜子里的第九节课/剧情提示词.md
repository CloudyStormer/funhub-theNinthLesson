# 第3集剧情提示词：镜子里的第九节课

## 统一正向风格

```text
国漫写实漫画质感，校园心理恐怖，低饱和冷蓝灰绿色调，厕所冷白灯和镜中红色应急灯形成对比，电影感构图，镜面反射诡异但不血腥，人物五官稳定
```

## 统一负向限制

```text
不要血腥，不要怪物化，不要夸张畸形，不要乱码文字，不要水印，不要字幕，不要让镜中林昼变成陌生人，不要让厕所空间变得不符合校园现实
```

## 关键剧情提示词

### 01 楼梯角解释第九节课

```text
白天课间楼梯角，许眠黑色低马尾，校服整齐，压低声音对林昼说话，手里拿着一张旧照片；林昼靠在墙边，表情从嘴硬变成紧张，背景学生经过但虚化，冷色自然光，校园悬疑
```

### 02 陈雨旧照片

```text
手部特写，许眠把一张旧班级照片递给林昼，照片里陈雨的脸和现在不同但文字模糊不可读，林昼手指停在照片边缘，教室走廊冷光，画面重点是“旧脸”和“现在的人不一样”
```

### 03 夜晚厕所洗脸

```text
夜晚学校厕所，洗手台在前方，镜子在洗手台上方，隔间在侧后方，出口在林昼后侧；林昼站在洗手台前洗脸，身体正对洗手台和镜子，黑短发湿了一点，校服外套敞开，现实厕所空无一人，冷白灯闪烁
```

### 04 镜中出现第九节课

```text
同一厕所镜子里映出的不是厕所，而是红光第九节课教室；镜面边框必须保留，说明教室只在镜子里。镜中有讲台、旧点名册、面向讲台和黑板的学生队伍；现实中的林昼仍站在洗手台前，惊恐后退半步，现实厕所不变成教室
```

### 05 镜中林昼抬头

```text
镜中红光教室队伍最后，镜中林昼站在那里，外貌与林昼相似但校服更整齐，扣子扣好，标准微笑，眼神没有温度，缓慢抬头看向镜外；镜中学生保持面向讲台和黑板静止，现实厕所仍在镜外
```

### 06 值夜老师从隔间走出

```text
现实厕所隔间门从侧后方缓慢打开，值夜老师从里面走出，白衬衫，金属边眼镜强反光，手拿旧点名册和手电；林昼从镜子方向转身看向隔间，厕所灯一盏盏熄灭，镜子里的红光教室仍只存在镜面里
```

### 07 厕所门变成黑板

```text
林昼冲向厕所出口，原本的门在出口位置变成一整块黑板，不要把整个厕所变成教室；黑板上只有“林昼，到”的感觉，字迹尽量清楚但不要出现其他乱码；林昼伸手停在黑板前，背后镜子泛红
```

### 08 影子替他举手

```text
厕所门口和镜面组合画面，现实中的林昼低头不接粉笔，看见自己的影子只在镜子里举手；镜中林昼标准微笑，值夜老师在现实厕所递来粉笔，现实厕所、黑板门、镜中教室三者位置清楚，不混成一个空间
```
