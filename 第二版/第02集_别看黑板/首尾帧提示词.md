# 第2集首尾帧提示词

## Clip 01 首帧

```text
白天高中走廊，林昼站在教室门外靠墙一侧，陈雨从走廊远处走来，路人学生自然经过，教室门在林昼旁边，冷色自然光，空间真实
```

## Clip 01 尾帧

```text
同一走廊，陈雨已经从林昼身边经过并推门进入教室，林昼回头看她背影，路人学生陆续进教室，不围观，走廊方向不变
```

## Clip 02 首帧

```text
白天教室后排，林昼坐自己座位，身体面向黑板和讲台，同桌坐旁边也面向前方，陈雨已经坐在背景座位面向讲台翻书，讲台和黑板在前方，路人同学正常上课状态
```

## Clip 02 尾帧

```text
同一教室后排，同桌只轻微侧头疑惑看林昼，林昼身体仍朝前但神情发紧，陈雨仍在背景座位面向讲台翻书，所有人位置不变，林昼准备低头拿手机
```

## Clip 03 首帧

```text
同一白天教室，林昼坐后排身体朝前，低头拿出手机，手机刚亮起，冷光照到他的脸，背景教室虚化但仍能看出陈雨坐在后方面向讲台
```

## Clip 03 尾帧

```text
同一教室，手机屏幕里的班级群记录正在消失，只保留模糊文字结构，林昼眼神紧张抬起头，背景不换场景
```

## Clip 04 首帧

```text
白天教室课桌下方，许眠坐前排身体朝讲台，只把手从桌侧向后伸，手里折好的纸条刚沿桌角向后推，林昼坐后排身体朝前，只伸手接，周围同学腿和书包只是背景
```

## Clip 04 尾帧

```text
同一课桌下方，林昼已经接过折好的纸条，许眠的手收回前排，纸条不出现乱码，教室空间不变
```

## Clip 05 首帧

```text
白天教室，林昼坐后排身体朝黑板，刚从纸条抬头，视线准备看向黑板，许眠在前排身体朝讲台没有回头，黑板在讲台后方固定位置
```

## Clip 05 尾帧

```text
同一教室，黑板角落出现“别看黑板”的感觉，林昼僵住没有继续看，许眠前排身体朝讲台轻敲桌面，其他同学仍面向前方正常上课
```

## Clip 06 首帧

```text
夜晚男生宿舍，灯刚熄灭，林昼躺在自己的床位，手抓被角，其他床位同学躲在被子里，宿舍门在固定方向
```

## Clip 06 尾帧

```text
同一宿舍，门缝下透进手电光，值夜老师影子停在门外，林昼睁眼不敢动，床铺和门口位置不变
```

## Clip 07 首帧

```text
同一宿舍床边，值夜老师刚站到林昼床下方，手电还未直照眼睛，另一只手拿旧点名册，林昼躺在床上紧张
```

## Clip 07 尾帧

```text
同一床边，手电光直照林昼眼睛，林昼强忍不眨眼，值夜老师镜片反光看不清眼睛，点名册在老师另一只手里
```

## Clip 08 首帧

```text
同一宿舍床边，值夜老师把旧点名册刚抬到林昼脸前，纸页正放，手电照亮纸面，林昼眼神故意散开
```

## Clip 08 尾帧

```text
同一床边，点名册正放贴近林昼脸前，林昼明明看清却装茫然，嘴唇微动，老师手稳定拿着点名册
```

## Clip 09 首帧

```text
同一本点名册同一床边位置，老师刚开始把点名册倒转，手电光固定在纸面，林昼紧张看着纸页
```

## Clip 09 尾帧

```text
点名册已经上下倒置贴近林昼脸前，林昼嘴角微动差点念出名字，宿舍背景仍黑暗，老师站位不变
```

## Clip 10 首帧

```text
黑暗宿舍床下，许眠只藏在床边阴影里，鞋尖刚伸向林昼垂下来的脚背，许眠不站到明处
```

## Clip 10 尾帧

```text
同一床下细节，许眠鞋尖已经踩住林昼脚背，林昼脚背一僵，被子轻动，空间仍是林昼床位下方
```

## Clip 11 首帧

```text
宿舍床边，值夜老师刚贴近林昼，镜片强反光，林昼身体后缩但不敢转头，现实背景仍是宿舍
```

## Clip 11 尾帧

```text
旧点名册特写，老师在“林昼，待观察”后面加重一笔，背景是宿舍黑暗，纸页和钢笔真实，只保留核心文字
```
