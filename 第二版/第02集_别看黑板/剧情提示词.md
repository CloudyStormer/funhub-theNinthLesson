# 第2集剧情提示词：别看黑板

## 统一正向风格

```text
国漫写实漫画质感，校园悬疑恐怖，低饱和冷蓝灰绿色调，夜晚红色应急灯点缀，电影感构图，人物五官稳定，干净线条，压迫感强，不血腥
```

## 统一负向限制

```text
不要欧美脸，不要日式夸张大眼，不要Q版，不要怪物化，不要血腥，不要多余手指，不要文字乱码，不要水印，不要字幕，不要突然新增主要角色，不要把宿舍切成教室
```

## 关键剧情提示词

### 01 陈雨白天归来

```text
第二天白天高中走廊，冷色自然光，林昼站在教室门外靠墙，黑短发，校服外套敞开，白T恤，单肩背包，疲惫警觉；陈雨齐刘海，蓝白校服，甜美微笑但眼神空洞，从他身边经过进入教室，不回头，周围同学自然走动
```

### 02 记忆被全班改写

```text
白天教室后排，镜头从林昼肩后看向同桌和讲台方向；林昼坐后排，身体面向黑板和讲台，只轻微侧头低声问同桌；同桌坐旁边也面向前方，只侧头回应；背景里陈雨坐在座位上面向讲台翻书，不看镜头，桌上有作业本和橡皮，座位方向不变
```

### 03 班级群记录消失

```text
同一白天教室后排，手机屏幕和林昼脸部组合特写，林昼身体仍面向讲台，屏幕冷光照在他紧张的脸上；背景教室虚化但不换场景，陈雨仍坐在原座位面向前方翻书；班级群消息只表现消失的结构，不要乱码大字
```

### 04 许眠递纸条

```text
白天教室课桌下方手部特写，许眠坐在前排，身体面向黑板和讲台，不转身；她只把手从桌侧向后伸，把折好的纸条沿桌角推给后排林昼。林昼坐后排，身体仍朝前，只伸手接纸条。背景学生面向前方虚化，课堂冷光
```

### 05 黑板出现别看

```text
白天教室，林昼坐后排，身体面向黑板和讲台，视线从纸条抬向黑板又停住；黑板固定在讲台后方，角落出现“别看黑板”的感觉，粉笔字半遮半糊，不要乱码大字；许眠在前排身体朝前，没有回头，只用手指轻敲桌面提醒；其他同学面向前方上课
```

### 06 夜晚宿舍熄灯查寝

```text
夜晚男生宿舍，灯光全部熄灭，只剩窗外冷蓝微光，林昼躺在上铺睁眼不敢睡，手抓被角，宿舍门缝下慢慢透进一条冷白手电光，门外有值夜老师瘦高影子停住
```

### 07 老师手电测试瞳孔

```text
宿舍上铺近景，手电光直照林昼眼睛，林昼瞳孔被光照亮但强忍不眨眼，额头有细汗；值夜老师站在床边下方，白衬衫，金属边眼镜反光看不清眼睛，另一只手拿旧点名册
```

### 08 点名册逼读和倒置试探

```text
值夜老师把旧点名册正放到林昼面前，手电照亮纸页，纸上隐约有“林昼，待观察”的结构；林昼明明看清却装茫然。随后老师慢慢把点名册倒转，纸页上下颠倒，仍贴近林昼脸前，镜头推向林昼差点开口的嘴唇
```

### 09 许眠踩脚救场

```text
黑暗宿舍床下细节特写，许眠的鞋尖从床边阴影里伸出，踩住林昼垂下来的脚背；许眠本人不完全出现，只露低马尾轮廓或半张冷静侧脸，林昼脚背一僵，被子轻动
```

### 10 老师贴脸镜片反光

```text
值夜老师突然贴近林昼，白衬衫在黑暗里泛灰，金属边眼镜强烈反光看不清眼睛，僵硬微笑；老师眼镜反光里出现一块不该出现在宿舍里的黑板，黑板上有“林昼，到”的感觉，现实背景仍是宿舍
```

### 11 点名册加深待观察

```text
旧点名册特写，老师用黑色钢笔在“林昼，待观察”后面再加重一笔，字迹变重，手电光照着纸页，背景是宿舍黑暗，镜头慢慢推近字迹，最后切黑
```
