# 第2集 Clip提示词：空间连续版

## 本集收紧剧情

白天证明陈雨“正常回来”和全班记忆被改写；夜晚进入男生宿舍，值夜老师围绕林昼床位做连续测试。空间必须从白天走廊到白天教室，再到夜晚宿舍，不能中途跳回第九节课教室；镜片反光里的黑板只是反光，不改变现实宿舍。

## 全集空间轴

- 白天走廊：林昼站在教室门外，陈雨从走廊远处走来，经过他进入教室。
- 白天教室：林昼坐后排，同桌在旁边，陈雨在背景座位，许眠在前排；所有学生默认身体面向黑板和讲台，林昼和同桌低声说话时只轻微侧头，不正对镜头。
- 夜晚宿舍：林昼在上铺或靠墙床位，值夜老师站在床边下方，许眠只藏在床边阴影里，不站到明处。
- 点名册始终由值夜老师拿在手里，正放、倒放、加重字迹都发生在同一床边空间。

## 统一负向限制

```text
不要突然从宿舍切进教室，不要让许眠公开站在宿舍中央，不要让点名册变成手机，不要让陈雨夜晚出现在宿舍，不要新增老师或学生围观，不要让床铺方向和门口位置反转，不要乱码文字
```

## Clip 01 陈雨白天归来

```text
白天高中走廊，镜头沿走廊朝教室门口方向拍摄。林昼站在教室门外靠墙一侧，陈雨从走廊远处走来，经过林昼身边后进入教室，不回头、不停顿。其他路人学生只做正常上课前走动，陆续进各自教室，不围观。结尾林昼回头看陈雨背影，空间仍停在走廊门口。
```

## Clip 02 同桌否认记忆

```text
白天教室后排，承接陈雨已经进教室坐下。林昼坐自己座位，身体面向黑板和讲台；同桌坐旁边也面向前方，两人低声说话时只轻微侧头。陈雨在背景座位安静翻书，身体面向讲台，不看镜头。教室讲台和黑板仍在前方，不要改变座位方向。路人同学正常写字翻书，不突然集体看向林昼。结尾林昼低头拿出手机。
```

## Clip 03 班级群记录消失

```text
同一白天教室，林昼仍坐后排，身体面向讲台，手机在桌面或手中，屏幕冷光照到他的脸。镜头从手机屏幕推到林昼眼睛，背景教室逐渐虚化但不换场景。聊天记录只表现消息消失的感觉，文字不要乱码大字。陈雨仍在背景座位面向讲台翻书，不突然靠近。结尾林昼抬头，准备接许眠纸条。
```

## Clip 04 许眠递纸条

```text
白天教室同一空间，许眠坐前排，身体仍面向黑板和讲台，只把手从桌侧向后伸，把折好的纸条沿桌角推给后排林昼；林昼坐后排，身体仍朝前，只伸手接纸条。镜头低位拍课桌下方，其他同学腿和书包只是背景，不抢戏。纸条不出现乱码。结尾林昼打开纸条，视线准备抬向黑板。
```

## Clip 05 黑板出现别看

```text
白天教室，林昼坐后排，身体面向黑板和讲台，视线从纸条抬向黑板；黑板仍在讲台后方固定位置，角落出现“别看黑板”的感觉，但文字半遮半糊。许眠在前排身体面向讲台，不回头，只用手指轻敲桌面提醒。其他同学都面向前方继续上课，不看镜头。结尾林昼停住视线，画面切到夜晚宿舍。
```

## Clip 06 宿舍熄灯查寝

```text
夜晚男生宿舍，灯熄灭后只剩窗外冷蓝微光。林昼躺在自己的床位，校服外套搭在床边，手抓被角。宿舍门在固定方向，门缝下慢慢透进手电光，值夜老师影子停在门外。其他床位同学躲在被子里不动，不露脸抢戏。结尾宿舍门被推开，老师进到林昼床边。
```

## Clip 07 手电测试瞳孔

```text
同一宿舍床边，值夜老师站在林昼床下方，手电从下往上照林昼眼睛，另一只手拿旧点名册。林昼躺在床上强忍不眨眼，额头有细汗。老师镜片反光看不清眼睛。床架、被子、门口方向保持一致。结尾老师把点名册抬到林昼面前。
```

## Clip 08 点名册正放逼读

```text
同一床边空间，值夜老师把点名册正放到林昼脸前，手电照亮纸页。林昼明明看得清，却故意让眼神散开，装作看不见。点名册纸张不要变成书本或手机，只隐约保留“林昼，待观察”的结构。结尾林昼嘴唇微动，老师准备把点名册倒过来。
```

## Clip 09 点名册倒置试探

```text
同一本点名册、同一床边位置，老师慢慢把纸页倒转，仍贴近林昼脸前。镜头从倒置点名册边缘推向林昼嘴唇，林昼差点顺着字念出口。老师手电光固定在纸面，宿舍背景黑暗不变。结尾许眠从床边阴影里伸脚提醒。
```

## Clip 10 许眠暗中踩脚

```text
黑暗宿舍床下细节，许眠只藏在床边阴影里，不完整站到明处，只露鞋尖或低马尾侧影。她轻轻踩住林昼垂下来的脚背，林昼被提醒后立刻改口说黑。空间仍是林昼床位下方，不能让许眠突然出现在走廊或床上。结尾回到老师贴近林昼的主视角。
```

## Clip 11 老师贴脸与加重待观察

```text
同一宿舍床边，值夜老师突然贴近林昼，镜片强反光，现实背景仍是宿舍。镜片反光里只出现一块模糊黑板和“林昼，到”的感觉，不把整个场景切成教室。随后切到点名册特写，老师在“林昼，待观察”后加重一笔。点名册仍在老师手里，背景是宿舍黑暗，最后切黑。
```
