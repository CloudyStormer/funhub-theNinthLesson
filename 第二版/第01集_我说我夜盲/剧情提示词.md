# 第1集剧情提示词：我说我夜盲

## 统一正向风格

```text
国漫写实漫画质感，校园悬疑恐怖，低饱和冷蓝灰绿色调，夜晚红色应急灯点缀，电影感构图，人物五官稳定，干净线条，轻微颗粒质感，压迫感强，不血腥
```

## 统一负向限制

```text
不要欧美脸，不要日式夸张大眼，不要Q版，不要怪物化，不要血腥，不要多余手指，不要文字乱码，不要水印，不要字幕，不要突然新增主要角色，不要改变校服款式
```

## 角色连续性

- 林昼：黑短发，校服外套敞开，白T恤，无侧条纹黑裤；白天课堂不背包，夜晚始终人物左肩单肩背黑色双肩包；懒散机灵，夜晚紧张但强装镇定。
- 许眠：黑色低双马尾，校服整齐，冷静克制，像知道规则。
- 值夜老师：瘦高中年男教师，黑色拉链外套、浅色衬衫，金属边眼镜强反光；Clip06-10旧点名册始终在人物左手、手电始终在人物右手；Clip11书写时人物右手改持钢笔、左手压页，手电放在讲台左侧。
- 陈雨：齐刘海，圆眼睛，校服整齐，白天乖巧；夜晚仍是同一张正常稳定的脸，诡异感来自过分平静的凝视、台词和暗红侧光，不丑化、不扭曲、不血腥。

## 关键剧情提示词

### 01—02 白天教室：讲台宣布；硬切到病历质问

```text
白天高中教室固定5排x5列，镜头从后墙看向讲台和黑板，左侧整排窗、右侧前后门。Clip01只上传一张首帧：普通白天班主任始终留在讲台、面向全班宣布晚自习，只做自然口型、一次小幅纸张手势和轻微呼吸；老师不下台、不进入过道、不走向林昼，林昼始终趴在最后一排右数第二桌。随后正常硬切到Clip02自己的独立首帧，省略老师离台过程；老师此时已经持唯一病历位于右侧过道中段，再沿空过道连续走到最后一排右数第二桌。老师只把病历放在林昼本人桌上并轻拍一次、提出质问；只有同桌林昼抬头、抬左手并解释，最后一排右数第三桌男生保持坐直沉默。许眠固定第2排第2列、陈雨固定第3排第2列，两人身体仍朝前；其他学生一桌一人坐满，只做停笔、压低偷笑或余光察觉，不起立、不围观、不换座。冷色自然光，病历纸无可读乱码。白天班主任不是值夜老师，不戴强反光眼镜
```

### 02 许眠低声提醒

```text
同一白天教室前排，许眠黑色低双马尾，两束都从耳后偏下位置扎起并垂到肩胛以下，校服扣好；腰、髋部、膝盖与脚尖仍面向黑板和讲台，头部自然转向右后方直至与后排林昼形成警告视线，肩部只做必要的小幅跟随，最终姿态以正式尾帧为准。林昼坐后排偏右，身体朝前，只侧头回应；背景同学面向前方虚化，课堂冷光，空间方向不变。禁止单马尾，禁止整个人转身。
```

### 03 夜晚教学楼红光

```text
同一天23:00后校园空无一人，四层教学楼大部分窗户熄灭，只有从地面数第三排的一间教室透出暗红色应急灯，顶层全部熄灭；林昼左肩单肩背黑色双肩包，停在楼下抬头看，地面有冷色反光，压迫感强，国漫写实恐怖
```

### 04 门缝里的第九节课

```text
从右侧后门外的5厘米窄门缝窥视同一夜晚教室，门框两侧黑暗；远端正前方是讲台和黑板，画面左侧是窗，空间不镜像。值夜老师站在讲台前，人物左手翻旧点名册、人物右手持手电。25名学生保持白天5排x5列的课桌坐标，站在各自课桌之间，全部背对后门、面向讲台和黑板，只露后背或侧背；陈雨位于左中区域且此刻尚未转头
```

### 05 陈雨歪脸回头

```text
红光教室里，陈雨站在学生队伍中，身体面向讲台和黑板、背对门口；被值夜老师点名后察觉门缝窥视，只把头非常缓慢地转向门缝约55—65度，身体、髋部和脚尖不转，双肩只允许极小自然牵动。齐刘海、圆眼睛、左侧银色发夹和校服保持一致，回头脸正常稳定并与“三分之四_正常脸.png”是同一人；近侧眼清楚，远侧眼只自然少量露出。背景学生静止，恐怖感来自凝视与台词，不丑化、不扭曲、不血腥
```

### 06 值夜老师走向门口

```text
值夜老师在讲台听见门扇轻响后抬头，沿画面右侧过道缓慢走向后门；黑板和讲台始终在远端，窗始终在画面左侧。瘦高身影压迫，黑色拉链外套和浅色衬衫被红光压暗，金属边眼镜强反光看不清眼睛；人物左手旧点名册、人物右手手电，全程不换手。除陈雨头部继续保持身份修正版尾帧的55—65度回看门缝外，所有学生躯干、髋部、膝盖和脚尖始终面向讲台，不得让陈雨头部复位
```

### 07 林昼撞门框装瞎

```text
夜晚三楼教室后门，固定走廊侧面180度轴：林昼在门外画面左侧，值夜老师在门内画面右侧。林昼低头，用人物左手贴住门框旁灰墙，试探前进半步并让左手掌缘故意轻碰门框，假装看不见；左肩背包仍在身后且不再碰门。老师人物左手点名册、人物右手手电，静看后僵硬微笑并开始向教室内转身。这是Clip08书包误撞后的第二次、故意接触
```

### 08 点名册写下待观察

```text
旧点名册特写，值夜老师的手拿黑色钢笔，在红色应急灯下缓慢写下“林昼，待观察”，纸张陈旧，手电光照在纸页边缘，镜头极慢推近，最后停在字迹上，只允许出现这几个字
```
