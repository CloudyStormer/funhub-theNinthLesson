# 第1集剧情提示词：我说我夜盲

## 统一正向风格

```text
国漫写实漫画质感，校园悬疑恐怖，低饱和冷蓝灰绿色调，夜晚红色应急灯点缀，电影感构图，人物五官稳定，干净线条，轻微颗粒质感，压迫感强，不血腥
```

## 统一负向限制

```text
不要欧美脸，不要日式夸张大眼，不要Q版，不要怪物化，不要血腥，不要多余手指，不要文字乱码，不要水印，不要字幕，不要突然新增主要角色，不要改变校服款式
```

## 角色连续性

- 林昼：黑短发，校服外套敞开，白T恤，单肩背包，懒散机灵，夜晚紧张但强装镇定。
- 许眠：黑色低马尾，校服整齐，冷静克制，像知道规则。
- 值夜老师：瘦高中年男教师，白衬衫，金属边眼镜强反光，旧点名册和手电。
- 陈雨：齐刘海，圆眼睛，校服整齐，白天乖巧，夜晚脸部轻微错位但不血腥。

## 关键剧情提示词

### 01 白天教室，林昼举病历撒谎

```text
白天高中教室，镜头从后排看向讲台和黑板；林昼坐后排偏右，身体面向黑板和讲台，只侧头举起假病历纸，表情心虚又得意；许眠坐前排偏左，身体朝前，只轻微回头看林昼；背景同学都坐着面向黑板，不看镜头，冷色自然光，不要让病历纸出现乱码文字
```

### 02 许眠低声提醒

```text
同一白天教室前排，许眠黑色低马尾，校服扣好，身体仍面向黑板和讲台，只把头和肩膀轻微转向后排林昼，眼神像是警告；林昼坐后排偏右，身体朝前，只侧头回应；背景同学面向前方虚化，课堂冷光，空间方向不变
```

### 03 夜晚教学楼红光

```text
深夜校园空无一人，教学楼大部分窗户熄灭，只有三楼某间教室透出暗红色应急灯，林昼背着包停在楼下抬头看，地面有冷色反光，红光轻微闪烁，压迫感强，国漫写实恐怖
```

### 04 门缝里的第九节课

```text
从狭窄门缝窥视夜晚教室，门框两侧黑暗；教室远处是讲台和黑板，值夜老师站在讲台前低头翻旧点名册；一排穿校服的学生站在课桌间，背对门口，身体面向讲台和黑板，只露后背或侧背，不要让学生转身
```

### 05 陈雨歪脸回头

```text
红光教室里，陈雨站在学生队伍中，身体面向讲台和黑板、背对门口；听到门口动静后只把头非常缓慢地转向门缝，身体不转，齐刘海，圆眼睛，校服整齐，脸部五官只有轻微错位感，背景学生静止，不血腥，不怪物化
```

### 06 值夜老师走向门口

```text
值夜老师从讲台方向缓慢走向后门，黑板和讲台在他身后，瘦高身影压迫，白衬衫被红光照得发灰，金属边眼镜强反光看不清眼睛，手里拿旧点名册和手电，背景学生仍面向讲台和黑板不动
```

### 07 林昼撞门框装瞎

```text
夜晚教学楼门口，林昼立刻低头伸手摸墙，故意轻轻撞上门框，假装自己看不见，肩上的书包被撞歪，呼吸变急但强行控制表情，眼神避开红光教室和老师方向，门缝红光压迫
```

### 08 点名册写下待观察

```text
旧点名册特写，值夜老师的手拿黑色钢笔，在红色应急灯下缓慢写下“林昼，待观察”，纸张陈旧，手电光照在纸页边缘，镜头极慢推近，最后停在字迹上，只允许出现这几个字
```
