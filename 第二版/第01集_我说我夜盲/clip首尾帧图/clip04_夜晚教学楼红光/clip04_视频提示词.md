# Clip 04 视频提示词（5秒正式版）

## 输入

- 首帧：`clip04_首帧.png`。
- 尾帧：`clip04_尾帧.png`。
- 模式：首尾帧图生视频。
- 总时长：`5.0秒`。
- 运动强度：低；固定楼下广角，关闭自动切镜、推拉摇移、缩放、转场和自动字幕。

## 可直接复制的大段正式提示词

```text
制作《第九节课》第二版第1集 Clip04“夜晚教学楼红光”的5秒正式画面。Clip03尾帧之后直接硬切到本Clip首帧，表示同一天23:00后；禁止溶解、闪白、渐暗或把白天教室变形成校园。首尾帧是硬性视觉边界，严格继承同一栋四层教学楼、同一窗格和入口、同一树木与湿路、同一名林昼、同一左肩黑色双肩包、同一冷蓝夜色和同一第三层暗红窗。摄影机全程固定。红窗在首帧已经亮着，视频中不得重新点亮或闪烁。

【固定身份与绝对空间】
教学楼共四层，第三层固定一扇暗红应急灯窗口，顶层和其他窗全黑；楼体占画面右侧与上方，林昼位于画面左下道路区域。林昼17岁，黑短碎发，蓝白校服外套敞开、白T、无侧条纹黑裤、白鞋；黑色双肩包始终人物左肩单肩背，包体在左后背。校园无其他人、车或动物。

【逐段时间轴】
0.0—0.2秒：用直接硬切从白天课堂进入23:00后外景，严格落在正式首帧。不要叠化出许眠、教室或白天残影。环境声从教室底噪直接切为极轻夜风和远处电流底噪。

0.2—0.7秒：林昼承接首帧自然步态继续从画面左侧向右行走，头和视线保持平视道路前方，尚未注意教学楼。人物左手轻握左肩背带，包体随步伐有小幅、受重力控制的摆动。楼体和红窗完全静止。

0.7—1.5秒：林昼完成第一步：前脚踩实湿地，重心平滑越过支撑脚，后脚抬起再向前；鞋底与地面接触真实，不滑行、不悬浮。使用总剧本旁白“我以为这事就这么过去了。”，语气是年轻人松了一口气后的自我安慰；旁白不驱动林昼口型。

1.5—2.1秒：林昼完成第二个较短步伐。走到约1.8秒时，眼角/余光先捕捉楼上异常，但头仍基本平视；眉尾有极轻收紧。旁白在此段结束。红窗从0秒起一直稳定，不能因“被发现”而突然变亮。

2.1—2.7秒：林昼因余光判断到异常而减速：步幅缩短，前脚落稳，后脚跟进，躯干惯性轻微向前后回正。左肩背包先按惯性前摆一点，再自然回落，不换肩。不得突然冻结、倒退或滑步。

2.7—3.1秒：林昼完全停住，双脚踩实湿地，身体主方向仍略朝画面右侧；先让眼睛向右上移动，再让头部开始抬起，形成“余光发现→停步→确认”的因果顺序。双肩不耸起，不用手指窗。

3.1—3.8秒：林昼把头和下巴自然抬高约20—30度，视线准确锁定从地面数第三排的同一扇红窗。颈部和身体比例正常，躯干只做极小跟随，双脚与包位置稳定。微表情从松弛转为疑惑警觉：眉间轻收、眼睑略抬、嘴唇准备开口。

3.8—4.6秒：林昼保持看着红窗，以压低、自言自语、困惑而克制的语气说总剧本原话“不是停电了吗？”宽景口型只需轻微自然，不夸张张嘴。第三层红光不回应、不闪、不增强；树、云和湿地只允许几乎不可察的自然环境运动。

4.6—5.0秒：林昼说完，嘴唇收住，头眼继续锁定红窗，身体与双脚不动，严格稳定落到正式尾帧并保持至少0.3秒。此后剪辑将硬切到Clip05三楼楼梯平台，省略进楼和上楼过程。

【前中后景运动限制】
前景湿路反光保持位置，只允许与脚步接触产生极轻真实变化，不能像水面波浪大幅扭曲。中景林昼只有走两步、减速、停步、抬头；绿化和入口不动。后景楼体、立柱和窗格完全锁死；树叶可有极轻夜风，厚云移动不可察，不能制造时间流逝。

【镜头与声音】
摄影机绝对固定，不推、不拉、不摇、不移、不变焦、不重构楼体。0.2—2.1秒使用旁白“我以为这事就这么过去了。”，不驱动口型；0.7—2.7秒配两步湿地脚步和极轻背带摩擦；3.8—4.6秒使用林昼原话“不是停电了吗？”，语气低、疑惑、不是惊叫。环境声只有极轻夜风、远处电流底噪和空校园混响；禁止雷声、尖叫、警报或新增对白。

【真实画面差异桥】
正式首帧中的红窗已经清楚可见，因此动作逻辑不是“红灯突然亮起把林昼吓停”，而是“红灯一直存在，林昼先从旁边走过，余光后知后觉发现，减速停步，再抬头确认”。必须按这个可实现桥接生成，不得假装首帧没有红光。

【终态与下一镜】
5.0秒严格等于正式尾帧：林昼在楼下原路线停住、头向右上看第三层红窗、左肩包不变；第三层单窗稳定暗红、顶层全黑。下一镜硬切到三楼楼梯平台，表示他在画外完成走进入口和上楼；禁止在本片末尾飞入窗户、冲到门口或瞬移到楼内。

【禁止项】
禁止白天到夜晚溶解、许眠残影、时间蒙太奇；禁止建筑呼吸、楼层增减、窗格漂移、入口换位、空间镜像；禁止红窗突然点亮、闪烁、换层、复制、爆白或扩散到整楼；禁止林昼走超过两步、滑步、瞬移、跌倒、指窗、拿手机、尖叫，禁止包换肩、服装变化；禁止新增人车、窗内人影、怪物；禁止自动切镜、推拉摇移、闪电和曝光跳变；禁止新增对白、字幕、气泡、可读校名、数字、水印、Logo、乱码；禁止照片写实、3D CGI、低龄萌系、血腥怪物化。
```

## 逐项验收

- [ ] 5秒单镜头，Clip03后使用硬切；没有白天残影或溶解。
- [ ] 红窗从首帧起就在第三层同一位置稳定发暗红光，顶层全黑，无新增亮窗。
- [ ] 林昼按“继续两步→余光发现→减速→停稳→眼神先移→抬头→低声疑问”完成动作。
- [ ] 左肩背包、服装、脚步接触和建筑结构全程连续，没有换肩、滑步或漂移。
- [ ] 只用旁白“我以为这事就这么过去了。”和对白“不是停电了吗？”，无新增台词。
- [ ] 尾帧稳定，下一镜通过硬切进入三楼楼梯平台；本镜不表现进楼上楼。
