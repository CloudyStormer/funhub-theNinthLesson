# Clip 04 尾帧提示词

先完整读取 `../../00_连续性锁定表.md`。本帧只推进“行走→减速→停步→抬头看同一红窗”，不改变建筑、红窗、背包或道路坐标。下一镜会省略林昼进入教学楼和上楼的过程，硬切至三楼楼梯平台；本帧不能把他瞬移到楼内。

## 可直接复制的正式尾帧提示词

```text
生成《第九节课》第二版第1集 Clip04“夜晚教学楼红光”的正式尾帧，一张完整单幅横版16:9画面，1920×1080 PNG。第一参考必须是已验收的 `clip04_首帧.png`，锁死同一四层教学楼、同一窗格、入口、道路、树木、天空、湿地反射、人物身份、服装和23:00后冷蓝夜光；不得重新设计建筑或移动异常窗口。

【摄影机与建筑完全不变】
摄影机仍在楼下道路同一地面位置、同一轻微仰角、同一广角焦段和同一16:9构图。四层楼完整可数；第三层固定单窗保持原位置和稳定暗红光；顶层全部黑暗，其他窗户不亮。楼体立柱、入口、绿化、左右树框和湿路反光与首帧连续，不得推镜、变焦、裁切、镜像或让建筑呼吸变形。

【林昼终点坐标与姿态】
林昼仍在画面左下道路区域，只沿首帧从左向右的路线前进了一至两步，未走出画面、未移到楼门口。他已经减速并完全停稳：双脚以自然前后或略开站姿踩实湿地，膝盖不过度弯曲，身体主方向仍略朝右；躯干没有跟着旋转到正对建筑。人物左肩继续单肩背同一个黑色双肩包，背带与包体位置连续，左手仍可轻握背带，包没有换肩、落地或漂浮。

【头部、视线与微表情】
林昼只把头和视线上移到人物右上方，准确看向从地面数第三排的同一扇红窗；下巴自然抬起约20—30度，颈部长度正常，肩膀不夸张后仰。眼睛聚焦红窗，眉间从松弛转为轻微收紧，眼睑略抬，嘴唇因疑惑轻启或刚说完“不是停电了吗？”后将合；表情是警觉与困惑，不是尖叫式恐惧，不张大嘴、不指窗、不掏手机。

【红窗与环境终态】
红窗相对首帧只允许视觉上略更明确，原因是林昼注意力转移而非灯光突然增强；物理亮度应基本稳定。红光仍只在第三层该窗内部，顶层和其他窗全黑；无窗内人影、手印或文字。道路、树木、厚云与绿化不因林昼抬头而变化，校园仍空无一人。

【光线与反射】
冷蓝灰环境光稳定；湿地只保留柔弱冷反射，林昼脚边不得突然出现巨大红色光池。人物面部因抬头略进入天空冷光，校服与肤色仍自然。不得闪电、红光脉冲、曝光跳变或增加新光源。

【下一Clip接口】
尾帧让林昼在楼下停住、抬头看第三层红窗并稳定保持。Clip05将用明确硬切省略他走向入口、进入楼内和上楼的数分钟/数十秒，从“三楼楼梯平台”重新开始；不得在本尾帧让林昼已经站在楼门、走入楼内、飞向三楼或通过红窗传送。

【绝对禁止】
禁止楼体、窗格、入口、树木、道路或反射漂移；禁止红窗换层、移列、复制、多窗亮起、顶层亮起、爆白闪烁；禁止林昼继续走出画面、正对镜头、指向窗户、拿手机、跌倒或惊叫；禁止背包换到右肩、变款、消失，禁止裤侧白条；禁止新增人车、窗内人影、怪物；禁止可读文字、数字、字幕、气泡、水印、Logo；禁止转场叠化、照片写实、3D CGI、低龄萌系、血腥怪物化。
```

## 逐项验收

- [ ] 与首帧同一固定楼下广角机位、同一四层楼与同一第三层红窗。
- [ ] 林昼只前进一至两步后停稳，头眼抬向右上方红窗，身体仍略朝右。
- [ ] 左肩单肩背包、服装、脚步和湿地接触连续；没有换肩或滑步。
- [ ] 红窗稳定不闪，顶层全黑，校园无人；无额外光源和可读文字。
- [ ] 终态可稳定停留，下一镜硬切三楼楼梯平台，绝不在本帧内瞬移入楼。
