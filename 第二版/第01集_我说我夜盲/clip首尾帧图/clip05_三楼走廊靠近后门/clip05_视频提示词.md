# Clip 05 视频提示词｜三楼走廊靠近后门

## 上传帧与参数

- 使用前先读：同集 `00_连续性锁定表.md`
- 首帧：`clip05_首帧.png`，必须作为 0.0 秒硬边界
- 尾帧：`clip05_尾帧.png`，必须作为 10.0 秒硬边界
- 模式：首尾帧图生视频
- 总时长：10 秒
- 画幅：16:9，正式输出 1920×1080
- 运动强度：中低
- 镜头：林昼左后侧同轴稳定慢速前跟；不切镜、不越轴、不绕人
- 对白：无对白、无旁白；Clip04 的楼下疑问句已经说完，本镜禁止重复

## 可直接复制的正式长提示词

```text
【硬性视觉边界与真实画面桥接】
输入首帧就是0.0秒，输入尾帧就是10.0秒。严格服从两张正式PNG的真实状态：0.0秒林昼刚从画面左后方三楼楼梯平台进入走廊，人物距离右前方高二三班后门仍有一段完整步行距离；10.0秒林昼已到门框左侧，身体侧对门，左眼靠近约5厘米门缝，左肩单肩背包位于左后背并贴近但尚未碰到向走廊微开的门扇。必须用连续步行、减速、短暂停步观察并屏息、继续靠近和侧身窥视完成两帧之间的可实现动作桥，不能假装首帧已经在门边，也不能用瞬移直接落尾帧。

【空间轴、人物和道具总锁定】
同一天23:00后，同一栋旧教学楼三楼。楼梯平台与栏杆全程在画面左后，走廊纵深向画面右前延伸，高二三班木质后门全程在右前方墙面；木门始终朝走廊微开约5厘米，走廊侧金属门把手位置不变，暗红门缝光在地砖上形成方向固定的窄梯形。林昼是同一名17岁中国男高中生：黑色短碎发、蓝白校服外套敞开、白T、纯黑无白侧条长裤、白色运动鞋。黑色双肩包从头到尾只挂人物左肩，包体落在左后背；右肩没有背带。

镜头在林昼左后侧约成人胸口高度，沿同一走廊轴线做非常缓慢、稳定、等速的前向跟随，跟随量只用于保持人物与后门关系清楚。镜头不能超过林昼、不能绕到他右侧、不能越过门扇、不能进入教室、不能反打或自动切镜。开头是较宽空间建立，随人物前行自然变成尾帧的更近全身构图；这种变化由真实前跟产生，不是变焦抽动。

【逐段时间轴】
0.0—0.4秒：完全保持正式首帧。林昼在左中区域处于自然迈步起点，头和视线已经被右前方门缝暗红光吸引；眉心轻收，嘴唇闭合。左肩承受背包重量，包体因上一动作只有极轻惯性，楼梯、后门和地面红色窄梯形全部稳定。

0.4—1.6秒：林昼完成第一个谨慎自然脚步，前脚鞋底从脚跟到前掌真实着地，骨盆和胸廓只有走路需要的微小反向摆动；手臂摆幅很小。视线一直落在后门和红光上，不看镜头、不看楼梯。镜头沿原轴小幅同速前跟。

1.6—2.9秒：完成第二个自然脚步。左肩背带随步伐略收紧，左后背包体比肩膀迟滞数厘米再回落，不弹跳、不换肩、不穿过手臂。林昼上眼睑略抬，疑惑增加，但还没有恐惧。走廊没有任何其他人或物件反应。

2.9—4.2秒：完成第三个较短脚步，林昼开始主动减速；脚步声变轻，躯干前倾角减小，头比胸口先朝门缝转约5—10度。镜头也同步减速，仍沿同一轴，不越过人物。后门开度、门把手和红光落点绝不变化。

4.2—5.0秒：林昼在距离后门仍有一小段位置短暂停住，双脚先后落稳，重心居中。视线从地面红色窄梯形上移到竖直门缝，眉峰轻抬、下颌微绷，表现他对停电后仍有异常红光的疑惑。此段没有对白，先保留0.2—0.3秒观察停顿。

5.0—6.4秒：林昼保持看向后门，完全不说话。先用视线沿竖直门缝从下向上确认一次，再停在门缝内部远端；眉间比上一段更紧，下颌轻绷，喉结做一次压住呼吸后的细微吞咽。约5.6秒时门内传来极低、模糊、绝对无法辨词的房间声，他因此屏息半拍并决定继续靠近。嘴唇始终闭合，不重复Clip04已经说过的楼下疑问句，周围无人回应。

6.4—7.5秒：观察停顿结束，林昼继续保持闭嘴，先用视线再次确认门缝，再迈出第四个更小、更慢的谨慎步伐继续接近后门。左手开始轻握左肩背带以压住包体，右手仍自然下垂；双手都不碰门。镜头继续小幅同轴前跟。

7.5—8.4秒：林昼接近门框左侧，速度降到几乎停止，脚步完整落地。身体开始侧向木门调整，但脚、膝、髋、胸依次自然转，不瞬间扭成尾态。门位于人物左侧/左后侧而非人物正前方；他始终留在走廊，不跨门槛。

8.4—9.2秒：林昼在门框左侧稳定停住，躯干侧对门，左肩靠近门叶，颈部向门缝轻伸。左眼逐渐对准约5厘米门缝，鼻梁和眼球不穿过门面；左手轻握背带，右手不碰门把。左后背包外缘接近门扇但必须保持约1—2厘米可见空隙。

9.2—10.0秒：所有动作缓慢收住，准确落到输入尾帧：左眼靠门缝窥视、身体和双脚都在走廊、左肩背包位于左后背并邻近门扇但尚未接触。林昼的表情仍是谨慎确认，不是已经看见陈雨后的受惊；门扇完全静止，为Clip06门缝POV和Clip08首次误撞保留正确起点。

【视线、微表情和身体物理】
视线链必须是：远处红光→后门竖缝→沿门缝确认→门缝内部远端，不能四处乱看。表情链必须是：轻疑惑→警觉确认→听见不可辨房间声后屏息→安静窥视，不能提前惊恐。走路必须有真实重心转移、鞋底接地、衣料与背包轻微延迟；禁止脚底滑行、脚步循环、膝盖反折、手指增生。靠门阶段左肩与包体的距离关系必须满足下一镜：包外缘近门但0次接触。

【前中后景反应】
前景地砖和顶灯倒影随摄影机前跟产生连续视差；中景只有林昼，校服与背包随步伐有克制物理摆动；背景楼梯逐渐后退但始终保持在左后，右前木门始终是视觉目标。走廊从头到尾无人，无鬼影、无风吹杂物、无门自行摆动。门内人物不可见，只允许极低、模糊、无法辨词的房间声从缝中泄出。

【照明与声音】
冷蓝灰顶灯稳定，不频闪；暗红应急光只勾亮门边、地面窄梯形和林昼靠门侧的极窄轮廓，不扩散成整屏红色。0.4—4.2秒是三组清晰但克制的鞋底脚步、衣料摩擦和背带声；4.2—6.4秒脚步停止，只留顶灯低电流、一次轻吞咽和从门缝泄出的极低不可辨房间声；6.4—8.4秒是一组更轻的小步与衣料声；8.4—10.0秒只留顶灯低电流、林昼极轻呼吸和门内不可辨低声。全镜无任何对白。绝对不要在本Clip加入背包碰门声、门轴震动声或教室骤停声。

【终态与下一镜接口】
10.0秒必须与输入尾帧一致：楼梯左后、后门右前，门向走廊微开约5厘米；林昼侧身站在门框左侧，左眼靠门缝，左手握左肩背带，右手不碰门；左后背包外缘与门扇近而不接触。下一镜可直接切到林昼左眼的门缝主观视角，不能在本镜提前展示教室。

【禁止项】
禁止首尾帧被模型重绘成不同走廊或不同人物；禁止楼梯/后门左右镜像、门扇开向反转、门开度变化、红光梯形漂移；禁止相机绕人、越轴、冲进门缝、手持乱晃、自动切镜、突变变焦；禁止林昼跑、跳、瞬移、滑步、撞墙、开门、推门、抓门把、穿门或提前把眼贴到门上；禁止包换右肩、双肩背、漂浮、提前碰门；禁止新增对白、旁白、学生声清晰成句；禁止血腥、怪物、鬼影、儿童萌系、换脸、换衣、黑裤白侧条、字幕、拟声字、可读门牌、乱码、水印、Logo和闪烁。

【验收】
1. 0.0秒真实保留首帧距离，10.0秒真实落到尾帧距离，中间完整看见3—4个自然脚步与同轴慢跟。
2. 楼梯全程左后、后门全程右前，镜头不越轴、不入教室。
3. 背包全程只挂左肩并落在左后背，10.0秒距门1—2厘米但没有碰撞。
4. 全镜无对白、无旁白；Clip04的楼下疑问句只出现一次，本镜绝不重复。
5. 本镜没有陈雨、老师、教室内画面和撞门声，可无缝接Clip06与Clip08的既定因果。
```

## 生成失败时优先降级

若平台容易让林昼面部漂移，5.0—6.4秒保持侧背/三分之四侧脸，只做闭嘴、屏息、轻吞咽和眼神确认；优先保住同轴步行、无对白和左肩背包连续性。
