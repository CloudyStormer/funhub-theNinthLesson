# Clip 03 尾帧提示词

先完整读取 `../../00_连续性锁定表.md`。本帧只推进许眠一次克制回头；正式PNG中许眠的回望幅度就是终值。不得让她在尾帧转回前方，也不得用全身转动代替头肩动作。

## 可直接复制的正式尾帧提示词

```text
生成《第九节课》第二版第1集 Clip03“许眠低声提醒”的正式尾帧，一张完整单幅横版16:9画面，1920×1080 PNG。第一参考必须是已验收的 `clip03_首帧.png`；锁死同一机位、焦段、教室、桌椅、人物坐标、服装、左窗日光和景深，只把许眠从完全朝前推进到正式回望姿态。

【摄影机与构图不变】
摄影机仍固定在第5排第4列林昼的右后肩附近，朝左前方第2排第2列许眠看。许眠上半身居画面中央，林昼只以虚化右肩、后脑侧缘和少量侧脸占右前景；远端黑板、讲台、左窗和朝前学生保持首帧位置。不得切到正反打、不得推近成大头特写、不得越轴或镜像。

【许眠终点姿态】
许眠仍坐在第2排第2列，臀部不离椅面。她的腰、髋、腹部、膝盖和脚尖百分之百继续朝黑板；胸口大方向也保持朝前。只让头部自然向人物右后方转到正式尾帧的三分之四回望角度，约60—75度，以能清楚看见面向林昼的一侧眼睛、鼻梁和嘴唇为准，但绝不超过自然颈部活动范围。肩部只做维持颈部自然所需的轻微跟随：靠林昼一侧肩膀向后不超过约10—15度，另一侧肩仍明显朝黑板；不得整块躯干跟着转。

【身份、发型与发束物理】
许眠是17岁中国女生，成熟自然比例，第二版明确低双马尾。两束黑发仍从耳后偏下/后颈两侧扎起并垂至肩胛以下；转头后靠镜头一侧的低马尾随头部产生很小的滞后摆动，另一束仍能看见，发绳位置和发量不变。不得变成单马尾、披发、高双马尾或一边发束消失。蓝白校服、白衬衫和深蓝领结保持整齐，衣领不变形。

【视线与微表情】
许眠的眼睛准确看向画面右前景的林昼，不看镜头、不看后墙、不看老师。上眼睑略压、眉尾轻收，目光冷静而确定，像她知道某条规则；嘴唇轻微张合至原台词“你最好别后悔。”刚说完/句尾将收的自然状态，下颌放松，不露齿，不微笑，不恐吓式瞪眼。脸部五官、皮肤与年龄保持稳定，不美颜成低龄偶像脸。

【林昼与背景反应】
林昼仍在右前景坐着，只把头朝许眠方向极轻侧转几度以听见警告，肩膀和身体不离开5排4列，不说话、不做夸张惊讶。唯一病历仍平放他桌面，即使被裁切也不能转移。陈雨固定3排2列，头、长直发、肩、躯干和脚尖仍朝黑板，绝不跟着许眠回头。其他学生把刚才的压低偷笑逐渐收住，只做呼吸和非常轻的肩部余动；没有第二个人转头。白天班主任处于本机位画外右后方的林昼桌边，不得复制进后景讲台。

【光线】
同一左窗冷白日光落在许眠转出的半边脸和鼻梁侧缘，形成柔和自然侧光；另一半脸保持蓝灰柔影。发丝亮边与首帧连续，不新增轮廓光、红光或夜景色。背景曝光和窗外白天保持不变。

【下一Clip接口】
尾帧至少表现许眠警告句结束后的稳定停留：她保持回望，不自动转回。剪辑时在这张尾态保留约0.5秒，然后直接硬切到Clip04同一天23:00后的教学楼外景；不得在本帧内渐暗、溶解、变夜或让教室变成室外。

【绝对禁止】
禁止全身转身、反坐、腰髋膝脚转向林昼、180度鬼转头、颈部拉长或扭曲；禁止低双马尾变单马尾/披发/高马尾、发束复制或穿脸；禁止许眠笑、流泪、尖叫、露齿或夸张瞪眼；禁止林昼正脸抢镜、林昼新增对白；禁止陈雨和背景学生回头围观；禁止老师回到讲台复制；禁止教室漂移镜像、切镜变焦；禁止字幕、对白气泡、水印、Logo、可读文字、数字、乱码；禁止日式萌系、照片写实、3D CGI、血腥怪物化。
```

## 逐项验收

- [ ] 许眠只完成一次头部60—75度回望，肩部仅10—15度必要跟随；腰髋膝脚仍朝黑板。
- [ ] 低双马尾两束仍存在且扎位低，脸部身份与首帧一致。
- [ ] 视线准确落在右前景林昼，微表情冷静克制，句尾口型自然。
- [ ] 林昼仅轻侧头听；陈雨及其他学生不回头；老师不在讲台复制。
- [ ] 尾态稳定保留，下一镜用硬切进入23:00后外景，不在本镜内变夜。
