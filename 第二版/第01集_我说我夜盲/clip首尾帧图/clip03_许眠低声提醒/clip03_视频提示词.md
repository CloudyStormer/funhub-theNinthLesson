# Clip 03 视频提示词（5秒正式版）

## 输入

- 首帧：`clip03_首帧.png`。
- 尾帧：`clip03_尾帧.png`。
- 模式：首尾帧图生视频。
- 总时长：`5.0秒`。
- 运动强度：低；固定林昼右后肩机位，关闭自动运镜、切镜、变焦、转场和自动字幕。

## 可直接复制的大段正式提示词

```text
制作《第九节课》第二版第1集 Clip03“许眠低声提醒”的5秒正式画面。首帧与尾帧是硬性视觉边界，锁死同一白天教室、同一林昼右后肩机位、同一焦段、同一许眠、同一低双马尾、同一学生与桌椅、同一左窗冷白日光。全片只有一个主动作：许眠从头身完全朝前，先用眼神察觉，再自然把头转向右后方林昼，肩部仅做必要跟随，低声说出总剧本原话，最后稳定停在尾帧。腰、髋、膝盖和脚尖从头到尾朝黑板。

【固定人物与坐标】
许眠固定2排2列，17岁中国女生，低双马尾、整齐蓝白校服、白衬衫、深蓝领结；林昼固定5排4列，仅以虚化右肩、后脑侧缘和少量侧脸位于画面右前景；陈雨固定3排2列、长直黑发、完全朝前。唯一病历仍在林昼桌面。白天班主任仍在林昼桌边但位于当前机位画外，不得复制到讲台。

【逐段时间轴】
0.0—0.4秒：严格从首帧稳定起步。许眠的头、鼻尖、肩、胸、腰髋、膝盖和脚尖全部朝黑板；两束低双马尾自然垂落。林昼右前景保持坐姿。背景只有极轻课堂底噪和前一镜遗留的压低笑声尾音。

0.4—0.9秒：许眠先停止手指/笔尖的细微动作，颈后和肩胛产生很轻的警觉张力；身体仍不动。靠后排少数同学的压笑逐渐收住，没有人回头。这个停顿表明她听清了林昼的谎言。

0.9—1.3秒：许眠只先移动眼睛：视线从黑板方向缓慢移向人物右后方，但头部仅开始转动几度；嘴唇闭合，眉尾轻收。必须先有眼神，再有明显转头，形成“听见→判断→警告”的因果。

1.3—2.5秒：许眠头部平滑、自然地向右后方转到正式尾帧角度，约60—75度。转动速度均匀，颈部长度不变，不出现停顿跳帧。肩部到约1.8秒后才开始小幅跟随，最终只跟随约10—15度；胸口大方向、腰、髋、膝盖和脚尖始终朝黑板，臀部不离开座椅。两束低马尾受惯性轻轻滞后摆动后回落，发绳位置不变，不能合成单马尾。

2.5—2.9秒：许眠完成回头并准确锁定画面右前景林昼的眼睛/侧脸。她不看镜头。上眼睑略压、眉尾轻收、下颌放松，形成冷静、确定、像知道规则的警告表情。林昼只把头朝她方向轻侧几度，不改变身体坐姿。

2.9—4.1秒：许眠以很低、近距离、克制的语气说总剧本唯一原话“你最好别后悔。”口型幅度小，逐字清楚但不露齿，不喊、不笑、不带哭腔。说话时头部和肩部保持尾帧方向，只允许呼吸带来的极小起伏。林昼不回答；背景学生的笑声在她开口后完全收住。

4.1—4.5秒：许眠说完，嘴唇自然合拢或停在句尾将闭的轻微状态，眼神仍锁定林昼。发束摆动彻底收住。林昼保持听见后的轻微侧头，不做夸张反应。

4.5—5.0秒：全画面稳定落到正式尾帧并保持至少0.5秒。许眠不转回前方；陈雨和其他学生仍朝黑板；桌椅、黑板、讲台和左窗光不漂移。5.0秒结束后剪辑直接硬切到Clip04同一天23:00后的教学楼夜景。

【视线、微表情与对白连锁反应】
许眠必须先听见林昼圆谎和后排压笑，手指/笔尖停住；再由眼睛先向人物右后方移动，头和肩随后跟进。她锁定林昼时，上眼睑略压、眉尾轻收、下颌放松、嘴角保持水平，既不是生气也不是幸灾乐祸，而是“知道规则且确认他会触犯规则”的克制警告。她说话后眼神不躲闪，句尾嘴唇自然收住。林昼因她的语气只产生极轻侧头和呼吸停顿，不回答、不笑；背景压笑在她开口的因果刺激下逐层停止，形成“许眠警告→林昼收住→周围安静”的连锁反应。陈雨及其他学生不跟随回头。

【前中后景反应】
前景：林昼仅右肩与侧脸，听而不答，不抢镜。主体层：许眠只完成一次眼神+头部回望，肩部必要跟随。中景：陈雨3排2列和相邻学生保持朝前。后景：黑板、讲台和其他学生稳定；老师不得复制出现。压低笑声由0秒逐渐消失，不能演成全班哄笑。

【镜头与声音】
摄影机绝对固定，不推、不拉、不摇、不移、不变焦、不切正反打，景深不抽动。声音只用极轻日间教室底噪、衣领与发束的细微摩擦，以及许眠原话“你最好别后悔。”林昼无对白，教师无对白，禁止添加任何旁白或解释。句尾留约0.5秒静默，再硬切夜景；禁止溶解、闪白、渐变或钟表蒙太奇。

【终态】
5.0秒严格等于正式尾帧：许眠头转60—75度看林昼，肩仅小幅跟随，腰髋膝脚朝黑板，低双马尾完整；林昼轻侧头；其他人不回头。该终态停住，不自动复位。

【禁止项】
禁止许眠全身转身、反坐、180度扭头、颈部拉伸、脸部变形；禁止双马尾变单马尾/高马尾/披发、发束融合穿模；禁止林昼正脸抢镜、起立、回话；禁止陈雨或背景学生跟着回头；禁止老师复制到讲台；禁止病历漂移；禁止教室镜像、桌椅人数变化、自动切镜推拉、光线闪烁；禁止新增对白、字幕、气泡、水印、Logo、伪文字；禁止照片写实、3D CGI、低龄萌系、血腥怪物化。
```

## 逐项验收

- [ ] 总时长5秒，单镜头，首尾帧顺序正确。
- [ ] 动作因果为“听见→停手→眼神先动→头转→肩少量跟随→警告→停住”。
- [ ] 许眠低双马尾稳定，腰髋膝脚从头到尾朝黑板，没有全身转身。
- [ ] 只说“你最好别后悔。”，林昼无对白，背景笑声克制收住。
- [ ] 陈雨和其他学生不回头，老师不复制，病历与教室不漂移。
- [ ] 尾帧至少停0.5秒，随后硬切Clip04夜景，不使用渐变转场。
