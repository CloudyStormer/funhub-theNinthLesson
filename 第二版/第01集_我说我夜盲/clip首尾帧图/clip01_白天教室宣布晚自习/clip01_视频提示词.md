# Clip 01 视频提示词（10秒正式版）

## 输入

- 首帧：`clip01_首帧.png`
- 尾帧：`clip01_尾帧.png`
- 尾帧必须与 `../clip02_病历拍桌解释夜盲/clip02_首帧.png` 为同一状态；通过后优先复用同一PNG。
- 模式：首尾帧图生视频。
- 总时长：`10.0秒`。
- 运动强度：中低；摄影机绝对固定，关闭自动切镜、自动运镜、自动补剧情和自动字幕。

## 可直接复制的大段正式提示词

```text
制作《第九节课》第二版第1集 Clip01“白天教室宣布晚自习”的10秒正式画面。首帧与尾帧是硬性视觉边界，必须严格继承两帧中的同一间白天教室、同一后墙中央略高机位、同一焦段、同一名白天班主任、同一批25名学生、同一5排×5列座位、同一服装和同一左窗冷白自然光。全片单镜头，不切镜、不变焦、不摇移、不跨轴、不改变构图。老师必须用完整可见的物理步伐从讲台走到右侧过道的尾帧位置，不能瞬移或溶解。

【固定坐标与身份】
黑板和讲台在画面正前方；整排窗户在左侧；前门在右前、后门在右后。林昼固定5排4列，黑短碎发、蓝白校服外套敞开、白T、无侧条纹黑裤，白天不背包；许眠固定2排2列，低双马尾、校服整齐，头身朝前；陈雨固定3排2列，长直黑发、头身朝前。白天班主任是30—45岁普通中国男教师，短黑发、深色西装/夹克、浅色衬衫，无强反光眼镜；他不是值夜老师。全程只有一张病历纸，起点在老师手中，终点仍在老师右手，绝不能提前落桌。

【逐段时间轴与动作因果链】
0.0—0.4秒：从正式首帧稳定起步。老师在讲台中央、双手于胸腹前持唯一病历纸，嘴唇正在做宣布晚自习的自然口型；林昼仍趴在5排4列，其余学生全朝黑板。摄影机、桌椅、窗门、光线保持绝对稳定。

0.4—1.8秒：老师继续完成宣布的最后一小段，仅做克制口型和一次很小的右手/纸张提示动作，纸张始终受手控制、不弯折、不复制。眉间轻收，视线平稳扫过中前排。学生只做呼吸、眨眼、个别人停笔；林昼不抬头。原剧本没有提供班主任宣布晚自习的逐字台词，因此这一段只有远景口型和很轻的教室底噪，不生成任何新增对白，也不让旁白驱动老师口型。

1.8—2.6秒：老师把宣布动作收住：嘴唇自然闭合，抬起的手和病历回到胸腹前，肩膀放松半拍。视线先停在全班中央，再开始向画面右后方移动。全班短暂安静，背景同学不能集体转头。

2.6—3.5秒：老师的眼睛和头部先转向第5排第4列的林昼，随后上半身才做很小幅度跟随，形成“先看见趴桌的林昼，再决定走过去”的明确因果。表情从日常宣布转为克制审视：眉心轻压、下唇收住，不发怒、不冷笑。林昼仍趴着，但听见讲台停声后肩背极轻绷紧，头不抬。

3.5—4.5秒：老师把唯一病历纸稳定换成右手单手持于身前偏下，左手自然垂下；先侧身朝讲台右侧出口转约30—45度，再迈出第一步。纸不能换到左手，不能变成文件夹、书本或多张纸。讲台与黑板保持后景稳定。

4.5—5.7秒：老师从讲台右侧完整走下讲台区域。先落一脚，再把另一脚带下，重心和膝关节自然；不得跳下、漂下或身体穿过讲台。老师始终看向林昼所在的右后方，嘴唇闭合，无新增对白。左窗光造成的影子随人体位移自然向右后方移动。

5.7—7.1秒：老师进入画面右侧过道，沿过道由前向后走两至三小步。路线必须贴合桌列之间的真实空隙，不碰桌角、不穿学生、不跨到左侧过道。病历在右手轻微随步伐摆动，但角度稳定、始终只有一张。右侧近中景学生听见脚步后可停笔、肩部轻僵；头、躯干、髋、膝盖和脚尖仍朝黑板。

7.1—8.4秒：老师继续沿右侧过道走近第5排第4列。脚步由正常放慢为审视性的克制步伐；头与目光锁定林昼。林昼听见脚步逼近，肩背再绷紧一点，趴伏姿态仍不改变，不抬头、不举手。许眠在2排2列只能有非常轻的眼神察觉，低双马尾与身体都不动；陈雨3排2列完全朝前。

8.4—9.4秒：老师走到尾帧前最后一个步位，距林昼桌边只剩一至两个正常小步；一只脚落稳，另一只脚收至自然准备继续的状态。右手病历垂在身前，左臂收住，宣布口型已经完全停止。后景讲台此时空出。邻近学生可有极轻的压低呼吸或停笔反应，但不得回头围观。

9.4—10.0秒：所有动作缓慢收束并严格落到正式尾帧：老师位于画面右侧过道、面向林昼、距桌边一至两步，唯一病历仍在右手且尚未落桌；林昼仍趴桌、仅肩背略绷；许眠与陈雨仍朝前。最后至少稳定保持0.3秒，为Clip02从同一共享帧继续“再走一至两步”建立无跳变接口。

【人物视线、微表情与连锁反应】
老师的表演顺序必须是：说话口型结束→手势落下→视线先找到林昼→头肩跟随→整理唯一病历到右手→从讲台右侧下台→沿右过道走近。老师由日常宣布转为克制审视时，只表现眉心轻压、眼神停住、下唇收紧，不发怒、不冷笑。林昼先听见讲台停声，肩背轻绷；再听见脚步靠近，肩背第二次略绷，始终不提前抬头解释。右侧邻近学生因老师经过先停笔、再收肩，但仍面向黑板；许眠只允许眼神轻察，陈雨完全不动。不得边宣布边瞬移，不得先走后看，许眠和陈雨都不能在Clip01回头。

【背景学生分层反应】
前景第5排：靠右过道的学生可停笔或稍稍收肩，绝不转头；林昼保持趴桌。中景第2—4排：多数学生继续面向黑板，最多有眨眼、握笔停顿、肩部呼吸；许眠只用眼神极轻察觉，陈雨不动。后景第1排：保持课堂坐姿，不追随老师移动。25人数量、发型分布、桌椅位置从头到尾不变。

【镜头与声音】
镜头绝对固定，禁止推拉摇移、焦点抽动或自动切近景。环境声为极轻白天教室底噪、衣料摩擦、纸张轻响和老师离台后的自然脚步。使用总剧本唯一已有旁白“我只是想逃一次晚自习。”，建议1.0—3.2秒以林昼内心旁白出现，语气年轻、懒散、自我安慰；旁白不驱动林昼或老师口型。原剧本没有班主任宣布晚自习的逐字台词，禁止自行编造。

【终态】
10.0秒必须等于Clip01正式尾帧，也必须等于Clip02正式首帧：老师在右侧过道距林昼桌边一至两步，右手唯一病历未落桌，嘴唇闭合、视线看林昼；林昼仍坐着趴桌；全班仍在固定座位；讲台空出。

【禁止项】
禁止换脸、换衣、老师变成值夜老师、强反光眼镜、手电、点名册；禁止老师瞬移、滑步、穿桌、跨学生、从左侧过道走、重新回讲台；禁止病历复制、漂浮、弹跳、变形、换手或提前落桌；禁止林昼站起、抬头、举手、提前对白；禁止许眠/陈雨/其他学生回头、围观、起立、换座；禁止教室镜像、窗门换边、桌椅漂移、人数增减、人体融合；禁止可读黑板文字、病历文字、字幕、气泡、伪文字、字母数字、水印、Logo；禁止自动转场、溶解、闪白、镜头呼吸、日式萌系、照片写实、3D CGI、血腥怪物化。
```

## 逐项验收

- [ ] 总时长10秒，首尾帧顺序正确，摄影机全程固定。
- [ ] 老师完整经历“收住宣布→先看林昼→右手持唯一病历→从讲台右侧下台→沿右过道走近”。
- [ ] 尾态只到距林昼桌边一至两步；病历尚未落桌，林昼尚未抬头解释。
- [ ] 林昼5排4列、许眠2排2列低双马尾、陈雨3排2列，25名学生和桌椅均不漂移。
- [ ] 只使用旁白“我只是想逃一次晚自习。”；没有编造班主任宣布内容。
- [ ] 尾帧与Clip02首帧完全一致，可无跳变接续。
