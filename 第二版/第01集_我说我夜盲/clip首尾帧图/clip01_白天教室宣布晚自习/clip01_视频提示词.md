# Clip 01 视频提示词（10秒物理路径修正版）

## 输入

- 首帧：`clip01_首帧.png`
- 尾帧：`clip01_尾帧.png`
- 图1就是0.0秒首帧，图2就是10.0秒尾帧；老师只走到图2中的位置，不能继续走到林昼桌前。
- 尾帧必须与 `../clip02_病历拍桌解释夜盲/clip02_首帧.png` 为同一状态；通过后继续复用同一PNG。
- 模式：首尾帧图生视频。
- 总时长：`10.0秒`。
- 运动强度：中低；摄影机绝对固定，关闭自动切镜、自动运镜、自动补剧情和自动字幕。
- 若平台支持“主体运动区域/运动笔刷”，只选择老师；讲台、课桌、学生、门窗和黑板全部设为静止保护区。
- 若平台支持“运动轨迹”，给老师画一条四点折线：`讲台后中央A → 讲台画面右侧外缘B → 右侧纵向过道入口C → 尾帧中老师所在的过道中途位置D`。D点绝不是林昼桌前；绝对不能画A到D的对角直线。

## 可直接复制的大段正式提示词

```text
制作《第九节课》第二版第1集 Clip01“白天教室宣布晚自习”的10秒正式画面。首帧与尾帧是不可改动的硬性视觉边界。全片单镜头，摄影机固定在教室后墙中央略高机位，不切镜、不推拉摇移、不变焦、不跨轴、不改变构图。

【首尾帧含义必须严格执行】
输入的第一张图是0.0秒首帧，输入的第二张图是10.0秒尾帧。视频只负责把老师从第一张图的位置移动到第二张图里已经明确显示的位置；第二张图不是方向参考，而是本Clip不可越过的精确终点。老师到达第二张图中的站位后必须立即停下，不能继续走到林昼面前，不能碰林昼课桌，不能放病历，不能和林昼对话。老师从尾帧位置继续走完剩余过道、来到林昼桌边的动作属于Clip02，本Clip绝不提前完成。

【最高优先级：只允许折线路径，禁止起终点直线插值】
老师从首帧位置到尾帧位置只能走四点折线：A讲台后中央→B讲台画面右侧外缘→C画面右侧纵向过道入口→D尾帧中老师所在的过道中途位置。前半段必须先沿黑板方向横向移动到画面右侧，完整绕出讲台；后半段才能转向画面下方、沿空过道前进到D点。D点由第二张输入图精确决定，仍明显没有到达右下方林昼的课桌。禁止从A点斜穿讲台、斜穿学生或课桌直接走向D点，也禁止越过D点继续靠近林昼。若运动无法一次完成，宁可缩短步幅并准确落尾帧，也绝不能穿过实体或擅自延长路线。

【实体碰撞与遮挡规则】
讲台是坚硬、完整、不透明的木质实体，正面木板、两侧立板和桌面都具有真实厚度。老师在讲台后横移时，下半身应该被讲台正面自然遮挡；腿和鞋不能透过木板显现，身体不能进入讲台体积。必须等老师的头、双肩、躯干、双腿和右手病历全部越过讲台画面右侧竖直外缘后，才允许向画面下方转弯。

所有课桌、椅子和学生也都是坚硬实体。老师进入右侧过道后，双脚只能踩在课桌之间清楚可见的灰色地砖上；鞋底不能踩桌面、穿桌腿或滑进学生身体。老师身体中心始终处于过道中央，左右肩与两侧桌角保持可见空隙，不碰、不挤、不跨越任何学生。讲台、课桌、椅子和学生全程原地不动，不能为了给老师让路而漂移、缩小、透明或消失。

【唯一目标人物与终点方向】
林昼不是任意后排学生。林昼是画面右下前景、离镜头较近、全教室唯一趴在课桌上的黑短发男生，蓝白校服，头伏在双臂上。老师可以在远处沿过道方向看向林昼，但本Clip不能走到他面前。老师绝不能把林昼前方那个坐直的学生认成林昼，也不能与任何学生发生接触。

终点D只能复制第二张输入图中的教师站位：老师停在画面右侧纵向过道的中途位置，仍位于林昼前方较远处；老师与画面右下趴桌的林昼之间必须保留尾帧里原有的完整课桌、坐直学生和过道距离，不能把这些间隔压缩掉。老师在D点不再迈步，只让身体与头部保持尾帧角度，目光沿过道看向远处的林昼。视线落点是林昼本人，不是林昼身后的空地、右墙、门或镜头；但“看向林昼”绝不等于“走到林昼面前”。

【固定人物、场景与道具】
同一间白天教室、同一后墙机位、同一焦段、同一25名学生、同一5排×5列座位、同一左窗冷白自然光。黑板和讲台在画面正前方，窗户在画面左侧，门在画面右侧。许眠固定2排2列，低双马尾、头身朝前；陈雨固定3排2列，长直黑发、头身朝前。白天班主任为30—45岁普通中国男教师，短黑发、深色西装或夹克、浅色衬衫，无强反光眼镜；绝不是值夜老师。全程只有一张病历纸，起点在老师双手，离开讲台前换到老师右手，终点仍在右手，绝不能提前落桌、复制或变形。

【逐段时间轴】
0.0—0.3秒：严格保持正式首帧。老师站在讲台后中央A点，双手在胸腹前持唯一病历纸；林昼仍在画面右下前景趴桌，其他学生面向黑板。所有实体坐标稳定。

0.3—1.2秒：老师完成宣布的最后一小段，只做克制远景口型和一次很小的纸张提示动作。原剧本没有宣布内容的逐字台词，因此不生成新增对白。林昼不抬头，学生只允许呼吸和眨眼。

1.2—1.8秒：老师停止口型、嘴唇闭合，视线先准确找到画面右下唯一趴桌的林昼。眼睛先转，头随后转，肩膀暂时仍正对全班；眉心轻压，确认目标。不得看向林昼身后的区域。

1.8—2.2秒：老师把唯一病历纸稳定换到右手，左手自然垂下；视线短暂扫一下讲台画面右侧外缘和外缘后的空地砖，确认绕行路线。这次短暂看路是正常避障，不是丢失林昼。

2.2—3.5秒：老师仍处于讲台后方，沿着与黑板平行的方向向画面右侧横移两小步，从A点走向B点。此时绝不能朝镜头或林昼走直线。老师下半身被不透明讲台正面板自然遮住；每一步由髋、膝、脚真实完成，腿和鞋绝不从讲台木板中穿出。病历只在右手，左手用于自然平衡。

3.5—4.1秒：老师从讲台画面右侧外缘完整露出。先确认双肩、躯干和双脚全部清出讲台碰撞范围，再在B点停半拍；随后脚尖、膝、髋、胸口依次转约90度，朝画面下方和右侧纵向过道C点。禁止身体尚未离开讲台就提前斜切。

4.1—5.2秒：老师迈出进入右侧纵向过道的第一步，双脚只踩灰色地砖。身体中心对准过道中线，左右肩与桌角保持空隙。老师先看一眼脚下过道，再重新把目光锁定画面右下趴桌的林昼。

5.2—6.3秒：老师沿同一条过道朝画面下方完成第二步。鞋底从脚跟到前掌真实着地，身体随透视自然变大；摄影机和背景不动。老师不靠近左右桌面，不穿过坐直学生的椅背。

6.3—7.4秒：老师完成第三步，继续处在同一过道中央。右手病历随步伐只产生很小自然摆动，左臂小幅反向摆动。右侧相邻学生听见脚步后只能停笔或收肩，不能转身，也不能被老师身体挤开。

7.4—8.4秒：老师不再增加第四个大步，只把上一小步自然收稳并减速到D点。D点就是尾帧里老师位于右侧过道中途的站位；画面中老师与右下趴桌的林昼之间仍保留原尾帧的完整距离。老师绝不能趁减速继续滑到林昼桌前。

8.4—9.1秒：老师双脚在D点完全停稳，身体仍在过道中央，不再向画面下方移动。先让眼睛沿过道看向画面右下远处的林昼，再让头部按尾帧角度轻微低下，最后让胸口做极小幅度跟随；顺序必须是“到达尾帧站位→完全停步→远距离看向林昼”。

9.1—10.0秒：严格逐像素趋近第二张输入图并稳定保持。老师只在尾帧标明的右侧过道中途位置，尚未走到林昼桌前；右手持唯一病历，尚未落桌。老师与林昼之间的课桌、坐直学生和空间距离必须与尾帧一致，不能消失或缩短。老师目光沿过道朝林昼方向，林昼仍趴着、不抬头。最后至少稳定0.4秒；Clip02将从这张同一共享帧开始，让老师继续沿过道走剩余距离并最终到达林昼桌边。

【人物视线与因果链】
老师的完整因果顺序必须是：宣布结束→准确发现右下趴桌的林昼→短暂看讲台右边缘确认避障→向画面右侧横移并绕出讲台→完整清出讲台后转90度→进入右侧纵向过道→重新锁定林昼→只走到尾帧所示的过道中途位置→停步并远距离看向林昼。老师不能为了持续盯人而无视障碍物直线穿越，也不能把Clip02的“继续走到桌边”提前到本镜。林昼听见讲台停声和远处脚步后肩背只做极轻绷紧，始终不抬头。

【背景学生反应】
老师是唯一进行大范围位移的人。其余25名学生和全部桌椅保持原坐标。靠右过道的学生可因脚步停笔、轻收肩，但头、胸、髋、膝和脚尖仍朝黑板；他们不能给老师让路、转头、起立或移动桌椅。许眠只允许眼神轻察，低双马尾不动；陈雨完全不动。

【镜头与声音】
摄影机绝对固定，不追随老师，不推拉摇移，不变焦，不自动切近景。环境声只有极轻白天教室底噪、纸张轻响、衣料摩擦和逐步对应的真实鞋底声。使用林昼内心旁白“我只是想逃一次晚自习。”，安排在0.8—2.8秒，旁白不驱动任何人口型。老师无可听见对白。

【禁止项】
禁止A到D直线或斜线移动；禁止越过尾帧D点继续走向林昼；禁止本Clip到达林昼桌边、把病历放桌或开始师生对话；禁止穿讲台、穿木板、穿桌面、穿桌腿、穿椅背、穿学生；禁止讲台、课桌或学生透明、让路、漂移、缩小或消失；禁止老师瞬移、滑步、漂浮、跳过转弯、身体分层错位；禁止从讲台左侧离开；禁止把坐直学生误认成林昼、看林昼后方、看右墙或看镜头；禁止病历复制、换左手或提前落桌；禁止林昼抬头、起立或提前对白；禁止许眠、陈雨或其他学生回头；禁止换脸、换衣、人数变化、教室镜像、自动转场、字幕、可读文字、水印、Logo、照片写实、3D CGI和血腥怪物化。
```

## 逐项验收

- [ ] 10秒单镜头，首尾帧顺序正确，摄影机全程固定。
- [ ] 老师路径清楚呈现四点折线A→B→C→D；没有从讲台直线斜穿到林昼。
- [ ] 老师在讲台后横移时，下半身被讲台自然遮挡；完整清出讲台右边缘后才转向画面下方。
- [ ] 老师进入过道后，双脚始终踩灰色地砖，肩膀与桌角有空隙，没有碰撞或穿模。
- [ ] 图1严格作为0秒首帧，图2严格作为10秒尾帧；老师没有越过图2位置继续走。
- [ ] 林昼始终是画面右下唯一趴桌男生；老师只停在尾帧所示的过道中途位置，没有到达林昼桌前。
- [ ] 尾态老师保持尾帧身体角度，目光沿过道远距离看向林昼，不看林昼后方、右墙或镜头。
- [ ] 病历始终只有一张，离台后固定在右手，尾帧尚未落桌。
- [ ] 尾帧与Clip02首帧完全一致，可无跳变接续。
