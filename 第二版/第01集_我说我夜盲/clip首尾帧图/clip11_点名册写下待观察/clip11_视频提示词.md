# Clip 11 视频提示词｜点名册写下待观察

## 上传帧与参数

- 总成片时长：5秒
- 画幅：16:9
- 视频模型输入模式：必须使用“单首帧”模式，必须且只能上传一张图；不是“首尾帧”模式
- 第一张图／唯一视频输入（0.0秒首帧）：`clip11_首帧.png`，只生成0.0—2.0秒
- `clip11_尾帧.png` 不是第二张视频输入，绝对不要上传给视频模型，不得与首帧交换或用于倒放
- 第二步后期剪辑：2.0秒整在剪辑软件中硬切到 `clip11_尾帧.png`，原图定格保持到5.0秒
- 不要让视频模型做首尾插值；否则准确中文会被模型变形
- 对白：无对白、无旁白

## 可直接复制的0—2秒图生视频正式提示词

```text
【任务边界】
必须使用“单首帧”模式，必须且只能上传一张图；不是“首尾帧”模式。第一张图也是唯一视频输入，必须为 `clip11_首帧.png`，作为0.0秒首帧，只生成一段严格2.0秒、16:9的图生视频。`clip11_尾帧.png` 不是第二张视频输入，绝对不要上传给视频模型，不得与首帧交换、不得用于倒放或首尾插值；它只在2.0秒以后由剪辑软件硬切进入并定格。不要尝试生成完整中文“林昼，待观察”，不要把任何可辨汉字写进视频。

【时间省略说明】
上一镜值夜老师只是在后门内开始向教室回身。本镜用匹配剪辑明确省略他从后门沿右侧过道走回讲台的数秒，直接切到同一讲台桌面特写。这是有意的时间省略，不是瞬移；视频中绝不补画老师走路，也不出现门口、走廊、老师脸、林昼或学生。

【因果链】
值夜老师在上一镜确认“对，你夜盲”并开始向教室内回身→匹配剪辑省略走回讲台的数秒→同一老师已经回到同一讲台→左手压住旧点名册→右手钢笔落笔形成少量不可辨识笔画→2.0秒后剪辑硬切到准确后期文字尾帧。顺序不能倒置，不能把时间省略误做成老师在门口和讲台同时出现。

【固定画面】
摄影机全程锁死在讲台上方45度俯拍特写，不推、不拉、不摇、不移、不变焦。旧点名册横向摊在讲台中央，人物左手压住左页/书脊，人物右手持黑色钢笔悬在右页空白登记行上方，黑色手电固定平放讲台左侧。点名册角度、纸页、表格线、木纹、黑板下沿、手电和光斑全程不漂移。暗红应急灯来自右上，手电冷白小光区来自左侧，光线不闪烁。

【分段时间轴：0.0—2.0秒逐段动作】
0.0—0.3秒：完全保持输入首帧。人物左手稳定压页，右手钢笔笔尖悬在空白行上方约3—5毫米，纸面没有任何可读文字。只允许非常轻微的手部呼吸性微动。

0.3—0.7秒：人物右手手腕极小幅度下沉，黑色钢笔笔尖垂直靠近纸面并轻轻接触；左手五指继续压住旧纸，点名册和手电不动。动作真实缓慢，不突然跳到书写终点。

0.7—1.3秒：钢笔在同一空白行内做两至三个短小、自然、干涩的书写起笔动作，路径总体从左向右，但笔画必须保持抽象、零散、不可辨识，不能组成任何完整汉字、英文、数字或符号。右手手指和手腕只移动必要距离，左手与纸页不动。

1.3—1.7秒：右手继续补一至两个很短的不可辨墨迹，笔尖位置逐渐靠近正式尾帧中钢笔的终笔区域；所有墨迹仍无法读成文字，不能提前出现“林”“昼”“待”“观”“察”中的任何完整字。

1.7—2.0秒：钢笔停止移动，笔尖离纸轻抬约1—2毫米，右手停在接近输入尾帧的终笔位置，以便2.0秒硬切时手部跳变最小。左手、点名册、手电、光斑和镜头完全静止。2.0秒处直接结束生成，不延长、不自动补帧。

【手部与物件锁定】
始终是人物左手压页、人物右手持黑色钢笔；两只手均为正常成年男性手，五指数量和关节正确。手电从头到尾只在讲台左侧，绝不回到右手，绝不复制。点名册不翻页、不旋转、不鼓起，钢笔不变成铅笔、毛笔或粉笔。

【声音】
0.0—0.3秒只有极低教室电流与室内空气声；0.3—1.7秒加入非常轻、干涩、近距离的钢笔划过旧纸声，声音与笔尖运动同步；1.7—2.0秒笔声停止，只留低电流。无对白、无旁白、无脚步、无学生声、无音乐重击、无字幕。

【全班反应与人物可见性】
本镜是桌面特写，老师的脸和身体、林昼以及全班学生全部不入画；不要为了表现“反应”新增人脸、手、倒影或学生背景。全班在画外继续保持上一镜后的死静，不发出议论、惊叫、脚步、椅子或衣料声。画面里只保留值夜老师执行书写动作所必需的两只手和黑色袖口。

【视线与微表情的替代表达】
本镜不出现任何人物面部，因此禁止凭空补画眼睛、嘴或面部微表情。值夜老师的情绪只由手部行为表达：左手压页稳定无颤，右手落笔果断但速度缓慢，短笔画之间没有犹豫，1.7秒后干净停笔；这组“稳定、机械、无情绪”的手部节奏等同于本镜的微表情。全班反应只通过画外绝对死静表达，不新增可见人物。

【禁止项】
禁止生成任何可辨中文、英文、数字、名字、乱码或字幕；禁止自动补出“林昼，待观察”；禁止老师脸、林昼、学生、门口或走廊入画；禁止镜头推拉旋转、纸页漂移、手电移动、光线闪烁、右手变左手、左手拿笔、额外手指、笔穿纸、墨水大面积晕染、自动切镜、水印、Logo和品牌。

【0—2秒验收】
1. 画面始终是同一45度讲台特写，所有静物位置稳定。
2. 只有人物右手钢笔做小幅真实书写，人物左手持续压页。
3. 纸面只出现少量不可辨识短笔画，绝无可读文字。
4. 2.0秒右手和钢笔停在接近正式尾帧终笔位置，方便无缝硬切。
```

## 2—5秒后期剪辑指令

```text
在时间线2.0秒整，直接硬切到 `clip11_尾帧.png`；禁止溶解、淡入淡出、形变插值、AI补帧或文字渐变。确认尾帧中的后期文字逐字准确为“林昼，待观察”，只有这五个汉字和一个全角中文逗号。将尾帧作为静态原图从2.0秒保持到5.0秒，整整定格3秒；不做推镜、呼吸缩放、手部运动、墨迹动画或光线闪烁。声音在2.0秒切入时停止笔划声，只保留极低教室电流并逐渐抽空。5.0秒结束后再接独立结尾字幕层；不要把“第九节课，开始点名。”烧录进本镜画面。

2.0—3.5秒：保持准确尾帧前1.5秒，画面所有像素静止，观众清楚读到“林昼，待观察”；只有极低教室电流。

3.5—5.0秒：继续保持同一准确尾帧后1.5秒，文字、双手、钢笔、手电和光线全部不动；环境声逐渐抽空，在5.0秒干净结束。

终态必须保持：点名册仍在讲台中央，人物左手压页、人物右手钢笔停在“察”字右下方，黑色手电仍在讲台左侧，准确文字“林昼，待观察”完整可读；从2.0秒到5.0秒所有像素不动，作为本集悬疑结尾定格。
```

## 最终验收

- 0—2秒只看见不可辨识书写动作，没有任何可读文字。
- 2.0秒是硬切，不是文字变形或渐显。
- 2—5秒准确尾帧完整定格三秒，文字严格为“林昼，待观察”。
- 左手压页、右手钢笔、左侧手电、讲台与光线在切点前后位置接近。
- 全片无对白、无旁白、无字幕、水印、Logo、乱码和新增人物。
