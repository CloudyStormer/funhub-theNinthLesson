# Clip 10 视频提示词｜林昼撞门框装瞎

## 上传帧与参数

- 首帧：`clip10_首帧.png`
- 尾帧：`clip10_尾帧.png`
- 总时长：10秒
- 画幅：16:9
- 运镜：固定走廊侧面中景，180度轴不变
- 对白：只使用总剧本原话；林昼“老师？我看不见，我夜盲。”，值夜老师“对，你夜盲。”

## 可直接复制的正式长提示词

```text
【硬性视觉边界与真实首帧桥接】
输入首帧是0.0秒，输入尾帧是10.0秒。严格服从两张正式PNG的真实状态：0.0秒首帧里，林昼位于画面左侧门外，仍抬眼看向画面右侧门内的值夜老师，双手尚未抬起；老师正面凝视林昼，人物左手旧点名册、人物右手手电。视频不能假装林昼在首帧已经低头，必须在0.4秒以后真实完成“移开视线、低头、抬左手”的动作桥。10.0秒必须落到尾帧：林昼低头、人物左掌贴灰墙且掌缘触门框，背包不碰门；老师已凝视三秒、确认微笑并开始向教室内回身。

【固定空间与角色锁定】
摄影机固定在三楼走廊侧面中景，林昼永远在画面左侧门外，值夜老师永远在画面右侧门内，木门框和门槛始终分隔两人。左后方是空走廊和楼梯纵深，右后方是暗红教室。林昼为17岁中国男生，黑色短碎发，蓝白校服外套敞开、白T、无侧条纹黑裤、白鞋，黑色双肩包只挂人物左肩并落在左后背。老师为瘦高中年中国男教师，黑色拉链外套、浅色衬衫、深色裤、金属边眼镜强反光；人物左手始终抱旧点名册，人物右手始终握黑色手电。

【逐段时间轴】
0.0—0.4秒：完全继承真实首帧。林昼还抬眼看着老师的反光镜片，肩膀微紧，双手自然垂下；老师在门内不动，刚说完上一镜的“林昼？”。二人短暂停顿，背包与门已有明显空隙。

0.4—1.2秒：林昼先把眼神从老师镜片移到门槛下方，再压低下巴和头，动作迅速但克制，像突然想起自己必须表现“看不见”。身体和双脚暂时不动，背包不晃向门。

1.2—1.9秒：林昼以压低、试探、带一点慌乱但努力装镇定的少年声说原话前半句：“老师？”同时只抬人物左手向门框左侧的灰墙靠近；人物右手继续下垂。老师仍无表情凝视，不回应。

1.9—3.0秒：林昼人物左掌轻轻落在灰墙上，掌心贴墙、五指自然分开；他保持低头，沿墙摸索极短距离，并说：“我看不见，”语速略慢，像在证明自己的谎言。左肩背包只随身体轻微摆动，但与门扇、门框始终有空隙。

3.0—4.2秒：林昼谨慎向前试探半步，左掌继续贴墙滑向竖直木门框；只有靠小指一侧掌缘故意轻碰门框外沿，产生一次很轻、很短的触碰。他同步说完：“我夜盲。”说完立刻收住脚步和左手，不摔倒、不夸张后仰。人物右手仍下垂，书包、背带和左肩绝不碰门。

4.2—4.5秒：林昼保持尾帧式低头触墙姿态，嘴唇收住，像等待裁决；老师身体、头和道具不动，镜片仍强反光。由这一刻开始计算老师的三秒凝视。

4.5—6.0秒：老师开始无声凝视的前1.5秒。身体、头、嘴角和道具完全不动，不说话、不微笑、不点头；强反光镜片遮住眼睛。林昼低头冻结在左掌贴墙、掌缘触框的位置，不偷看老师。

6.0—7.5秒：老师继续无声凝视后1.5秒，与上一段无缝连接，使凝视从4.5秒到7.5秒累计完整、连续、整整三秒。只允许一次极轻呼吸，仍不说话、不微笑、不点头、不走动。林昼和画外全班继续死静。

7.5—8.1秒：三秒凝视结束后，老师嘴角才极小幅度上提，形成僵硬、像确认规则成立一样的微笑，并做一次几乎不可察觉的轻点头。左手点名册、右手手电不动。

8.1—9.1秒：老师用低、平、没有安慰意味的男声说总剧本原话：“对，你夜盲。”口型克制，镜片仍反光。林昼不抬头、不回应、不移动左手。

9.1—10.0秒：老师说完后先把头转向教室内部/远端讲台方向，再让肩线和上身跟随约15—25度，开始回身但不迈出返回讲台的步子、不离开门内位置。林昼保持低头，左掌贴墙、掌缘触框，背包继续与门分离。准确停在输入尾帧。

【因果链与表演重点】
因果顺序必须是：老师叫出名字→林昼从真实首帧的抬眼状态迅速低头→抬人物左手贴门框旁灰墙→半步试探→左手掌缘故意轻碰门框→说完夜盲谎言→老师完整凝视三秒→三秒后才僵硬微笑→说“对，你夜盲”→开始转回教室。Clip 08 是左后背书包意外碰门；本镜是人物左手故意触门框，两次接触的部位、原因和时间必须严格分开。

【全班反应】
门内深处如能看见学生，只保留模糊静止背影。所有学生仍站在原课桌之间、身体朝讲台；没有人靠近后门、回头围观、交头接耳或做口型。走廊从头到尾无第三人。

【镜头语言】
固定侧面中景，不推、不拉、不摇、不移、不变焦、不切近景。180度轴锁死，林昼始终左外、老师始终右内。触墙动作必须在同一镜头里完整可见，左掌、灰墙、竖直门框、左肩背包与门之间的空隙都应保持清楚，不能被景深糊掉或被身体遮住。

【微表情与视线】
林昼：0.0秒紧张抬眼→0.4—1.2秒移开视线并低头→之后始终看地面，不闭眼、不看老师；声音慌而克制，动作像临时表演。老师：0.0—7.5秒无笑、镜片强反光、脸部几乎不动→7.5秒后才出现极小僵笑→说完后转向室内。绝不能把老师的笑提前到三秒凝视内。

【声音设计】
全程保留极低走廊顶灯电流、远处建筑沉闷底噪和门内压低的红灯教室房间声。1.2—4.2秒只有林昼原话；3.7—4.1秒加入一次极轻的皮肤/校服袖口擦墙声和掌缘触木框的短促闷响，绝不是书包撞门的响声。4.5—7.5秒必须保留三秒近乎真空的沉默。8.1—9.1秒只有老师原话。不要旁白、不要学生声音、不要惊悚音效重击、不要字幕。

【终态与下一镜接口】
10.0秒准确匹配尾帧：林昼画面左侧门外低头，人物左掌贴灰墙、掌缘轻触门框，左肩背包与门完全分离；老师画面右侧门内，左手点名册、右手手电，已说完“对，你夜盲”，头和肩刚向教室内部转动。下一镜使用匹配剪辑明确省略老师走回讲台的数秒，因此本镜绝不能直接让老师瞬移到讲台。

【禁止项】
禁止首帧就假装林昼已低头或已贴墙；禁止林昼与老师左右互换、跨轴、穿门；禁止把人物左手变成右手，禁止手拍门板、抓门把手或整掌撞框；禁止书包、背带、肩膀再次碰门；禁止老师凝视不足三秒、凝视期间提前笑、提前说话或提前转身；禁止点名册/手电换手、消失、复制或出现钢笔；禁止学生围观；禁止闭眼装瞎、夸张喜剧动作、摔倒、换脸、换衣、怪物化、血腥、自动切镜、闪烁、字幕、可读文字、水印、Logo和乱码。

【验收】
1. 0.0秒保留林昼真实抬眼首态，0.4秒后才完成低头动作桥。
2. 触框者必须是人物左手，掌心贴灰墙、只有掌缘轻触木门框；书包全程不碰门。
3. 老师必须完整凝视3.0秒，三秒结束后才笑并说“对，你夜盲。”
4. 对白逐字来自总剧本，无新增台词；林昼与老师口型、语气和时间对应。
5. 10.0秒与尾帧一致，老师只开始回身，为 Clip 11 的时间省略建立方向。
```
