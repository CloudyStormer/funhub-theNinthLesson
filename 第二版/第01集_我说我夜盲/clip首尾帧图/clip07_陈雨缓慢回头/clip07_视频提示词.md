# Clip 07 视频提示词｜稳定重制版

> 旧“三图同时输入”方案已停用。陈雨正脸卡只用于事先修好尾帧，不能再作为第三张图拖入视频模型。

## 生成前必须完成

1. 先用本目录 `clip07_尾帧提示词.md` 重做尾帧，输出为 `clip07_尾帧_身份修正版.png`。
2. 重做尾帧时使用 `clip07_首帧.png` 锁定教室空间，使用 `第二版/抽卡/陈雨/三分之四_正常脸.png` 锁定陈雨的回头脸。
3. 先单独检查修正版尾帧：陈雨必须是正常、稳定、与抽卡一致的脸；检查通过后才生成视频。
4. 原 `clip07_尾帧.png` 只保留作历史对照，未通过身份一致性验收，不再直接用于视频生成。

## 视频模型输入与参数

- 输入模式：**首尾帧模式，只上传两张图**。
- 第一张图：`clip07_首帧.png`，作为 0.0 秒首帧。
- 第二张图：`clip07_尾帧_身份修正版.png`，作为 5.0 秒尾帧。
- 第一张永远是首帧，第二张永远是尾帧；不可交换、倒放或漏传。
- 视频阶段不要上传 `陈雨/正.png`、`陈雨/三分之四_正常脸.png` 或任何第三张人物卡。
- 总时长：5 秒；16:9；低运动强度；门缝摄影机绝对固定。
- 本镜不生成可见口型。台词在后期配音，避免远景小脸因口型再次变形。

## 可直接复制的正式提示词

```text
【输入方式｜最高优先级】
本Clip使用首尾帧图生视频，只上传两张已经修正完成的图片。第一张 `clip07_首帧.png` 是0.0秒首帧与唯一动作起点；第二张 `clip07_尾帧_身份修正版.png` 是5.0秒尾帧与唯一动作终点。第一张是首帧，第二张是尾帧，不可交换、不可倒放、不可漏传。

视频生成时不得上传第三张图片。陈雨正脸卡和正常三分之四脸卡只用于事先生成第二张修正版尾帧，不参与视频插值。视频中陈雨最终的脸型、五官、发型、视角和回头角度，只以第二张修正版尾帧为唯一标准。

【镜头与动作目标】
同一天23:00后的第九节课教室，摄影机固定在后门外，通过约5厘米门缝观察。左右黑色门框、远端黑板讲台、左侧窗户、右侧墙面、中间过道、5排×5列课桌、老师和25名学生的坐标全部锁定；不推近、不变焦、不切镜、不改变景深。

陈雨固定站在画面左中原课桌位置，身体面向讲台。她察觉门缝有人后，沿正常人体颈椎结构缓慢回头，准确到达第二张修正版尾帧规定的55—65度三分之四侧后回望。双脚、膝盖、骨盆和躯干不转；双肩只允许出现不超过3度的自然牵动。不得为了展示正脸继续多转，不得正对镜头。

【5秒时间轴】
0.0—0.45秒：严格保持第一张首帧。陈雨背对门缝、面向讲台，嘴唇闭合；老师和其余学生保持原位。

0.45—1.25秒：陈雨先产生很轻的颈部预备张力，下巴与头部作为一个整体开始向门缝方向转动。五官不能先于头骨滑动；长发比头部动作略慢，只有轻微重力延迟。

1.25—2.20秒：陈雨沿同一方向平稳完成回头，逐渐露出侧脸。双脚、膝盖、骨盆和躯干保持朝向讲台，双肩只有极小自然牵动；不突然加速、不抽动、不反向摆头。

2.20—2.50秒：回头动作完全结束，陈雨准确到达第二张修正版尾帧的头部角度和面部状态。她安静看向门缝，嘴唇保持闭合，先把脸稳定下来。

2.50—5.0秒：陈雨保持修正版尾帧姿态不再转头。只允许极轻呼吸和发丝余动，五官、下颌、眼距和脸型完全稳定；嘴唇不做可见口型。老师和其余24名学生始终面向讲台，不回头、不说话、不移动。

【陈雨面部稳定】
陈雨露出的脸必须始终是第二张修正版尾帧中的同一张正常脸：脸型、眼睛、眼距、鼻梁、嘴唇、下颌、肤色、齐刘海、长直黑发和左侧银色发夹不得改变。回头过程中只允许正常透视变化，不重新设计五官，不美化换脸，不把侧脸强行补成完整正脸。

在55—65度回头角度下，只允许靠近摄影机的一只眼清楚可见，另一只眼按真实透视自然少量露出；禁止强迫两只眼同时完整正对镜头。陈雨的视线通过门缝看向林昼所在方向，但鼻尖仍保持三分之四侧向。诡异感只来自安静回头、过分平静的凝视、停顿、台词含义和暗红侧光，绝不通过丑化或扭曲面部表现。

【光线与声音】
暗红应急光从画面右前形成很薄的侧光，另一半脸保持冷蓝灰；不得新增正面补光。视频画面中不做可见口型，只保留低电流底噪、极轻衣料和发丝摩擦。后期在2.65—3.05秒加入陈雨配音“老师，”，在3.20—4.45秒加入“新的脸不太合适。”，语气平静、礼貌、克制。

【禁止项】
禁止第三张人物卡进入视频插值；禁止边回头边说话、回头未完成就出现口型；禁止强行正脸、两只眼同时完整贴向镜头、过度转头、180度扭头、断颈、脖子拉长、头部漂浮；禁止整个人转身、肩髋大幅跟转、挪脚或换位；禁止换脸、脸型漂移、大小眼、眼距变化、鼻嘴歪斜、下颌融化、五官滑动、年龄变化；禁止头发穿脸、发夹消失；禁止镜头推近、脸部特写、变焦、摇移、切镜；禁止其他学生回头、撞门声、字幕、水印、Logo和新增对白。
```

## 后期操作

- 2.65—3.05 秒：配音“老师，”。
- 3.20—4.45 秒：配音“新的脸不太合适。”
- 画面不需要明显口型；这是门缝远景，稳定的脸比强行口型更重要。
- 如果模型在 2.50 秒后仍继续改脸，直接从修正版尾帧导出静帧，覆盖 2.50—5.0 秒，再铺配音和环境声。

## 验收

- [ ] 视频只上传第一张首帧和第二张身份修正版尾帧，没有第三张人物卡。
- [ ] 0.0—2.20 秒只做一次自然回头，2.50—5.0 秒停止转头并稳定保持。
- [ ] 陈雨始终不是正脸贴镜头；近侧眼清楚，远侧眼只按角度少量露出。
- [ ] 全程无换脸、大小眼、鼻嘴变形、下颌融化或五官滑动。
- [ ] 老师、其余学生、课桌、门缝和光线没有漂移。
- [ ] 台词使用后期配音，不用模型强做远景口型。
