# Clip 07 视频提示词｜陈雨缓慢回头

## 上传帧与参数

- 使用前先读：同集 `00_连续性锁定表.md`
- 首帧：`clip07_首帧.png`，必须作为 0.0 秒硬边界；它与 Clip06 首帧像素同源是有意设计
- 尾帧：`clip07_尾帧.png`，必须作为 5.0 秒硬边界
- 模式：首尾帧图生视频
- 总时长：5 秒
- 画幅：16:9，正式输出 1920×1080
- 运动强度：低，仅陈雨头部和必要口型运动
- 镜头：门缝 POV 绝对固定；不推、不摇、不移、不变焦、不切镜
- 对白：陈雨只说总剧本原话“老师，新的脸不太合适。”；老师和其他学生无对白

## 可直接复制的正式长提示词

```text
【硬性视觉边界与真实首尾帧】
输入首帧就是0.0秒，输入尾帧就是5.0秒。两张正式PNG共同证明：摄影机始终在后门外通过约5厘米门缝看教室；首帧中陈雨位于学生队伍左中区域，后脑、肩、躯干和脚全部朝讲台；尾帧中她仍站在完全相同的课桌坐标，身体与肩膀没有转，只有头部朝门缝转到约60—70度并露出三分之四侧脸。视频必须真实生成从0度到60—70度的连续颈部旋转，不能假装首帧已经转头，也不能为了贴尾帧让整个人旋转或瞬移。

【空间、人物、数量和道具总锁定】
承接Clip06同一天23:00后的同一红灯教室。画面左右各约10%—15%深黑门框/门叶竖向遮挡，远端黑板和讲台固定居中，窗户固定在画面左侧，右墙固定在画面右侧，中间过道和5排×5列课桌位置完全不动。25名学生全部站在自己课桌之间；陈雨固定左中区域，其余24名学生从头到尾背对后门、面向讲台。

陈雨是同一名17岁中国女高中生：齐刘海、长直黑发、圆眼、蓝白校服、无背包；尾帧出现的左侧银色发夹、发量和脸型必须保持。值夜老师仍在远端讲台，瘦高中年中国男性、黑色拉链外套、浅色内搭、金属细框眼镜强反光；人物左手旧点名册、人物右手手电，绝不换手。

【身体硬锁】
从0.0到5.0秒，陈雨的双脚鞋底、脚尖方向、踝、膝、髋、骨盆、腰、胸骨、双肩、双肘和双手必须保持首帧位置与朝向；脚尖、膝盖、髋部、躯干始终面向讲台，肩线始终水平。唯一的大动作是颈椎带动头部沿最短路径朝后门门缝转60—70度。不能90度侧身，不能180度鬼转头，不能肩膀跟转，不能扭腰，不能挪脚。

【逐段时间轴】
0.0—0.35秒：完全保持正式首帧。陈雨头部旋转为0度，后脑朝后门、脸朝讲台；她刚听完老师点名，胸口呼吸停半拍。值夜老师、其余24名学生、课桌、门缝黑边和光线全部静止。

0.35—0.8秒：陈雨只出现动作准备：颈后肌肉极轻绷紧，下巴抬高不到1厘米，视线先在眼眶内向后门方向移动一点，但脸仍基本朝讲台。双肩、胸骨、骨盆、膝盖和脚尖保持像被钉住，不得预转；长发主体仍垂直落在背部。

0.8—1.4秒：头部从0度缓慢转到约15—20度。运动由颈椎连续驱动，速度极慢、无抽帧；后脑轮廓先偏转，靠脸侧一小束发丝比头部延迟几帧再滑动。此时仍看不清完整脸，嘴唇闭合，无对白。

1.4—2.0秒：头部继续从约20度转到约40—45度。鼻梁和一只眼的轮廓逐渐露出，瞳孔持续寻找后门门缝；肩线绝对不动，衣领不被整片拉扯，躯干、髋、膝、脚尖仍朝讲台。老师和其他学生没有任何反应。

2.0—2.55秒：头部从约45度缓慢完成到60—70度，准确接近尾帧三分之四侧脸。颈部长度和厚度保持真实，后脑仍有体积；长直发只产生小幅分束与重力延迟，主体发束仍落在背后。两眼最终对准门缝/镜头，陈雨没有进一步转成正面。

2.55—2.75秒：陈雨稳定保持60—70度回望，先用过分平静的圆眼确认门缝里有人；眉毛几乎水平、下颌放松，嘴唇尚未开。面部异常只能是毫米级：眼、鼻、唇中心关系有2—4毫米视觉错位，皮肤完整，无裂缝、血、缝线、多眼或重影。

2.75—3.15秒：陈雨用平静、礼貌、像在报告普通问题的少女声说：“老师，”口型小，头部角度保持不变，视线仍锁定门缝，不回看讲台。肩、胸、腰、髋、膝和脚全部不动。

3.15—4.65秒：陈雨继续逐字说总剧本原话：“新的脸不太合适。”语速慢而自然，不故意拉长，不做怪物腔，不哭、不尖叫。嘴唇开合幅度克制，脸部毫米错位保持稳定而不是动态融化；眼神始终穿过门缝对准林昼。值夜老师与其他24名学生完全不回应、不回头、不侧目。

4.65—5.0秒：台词结束，陈雨嘴唇轻收，头部仍保持60—70度，眼睛仍看门缝；所有发丝余动在5.0秒前停住，准确落到输入尾帧。绝不自动把头转回讲台，也不让身体追随头部。

【视线、微表情与因果链】
因果必须是：Clip06老师点到陈雨并说“换完了”→陈雨察觉门缝里的窥视→只转头寻找门外观察者→与林昼对视→以异常平静语气报告脸不合适。她不是被尚未发生的撞门声吸引；Clip08的撞门发生在本镜之后。陈雨的微表情始终是礼貌、困惑、过分平静，恐怖来自语义和毫米级错位，不来自大幅鬼脸。

【其他人物与前中后景反应】
前景门缝黑边和后排学生不动；中景陈雨是唯一动作主体，其余学生呼吸幅度接近不可见；后景值夜老师仍在讲台，左手点名册、右手手电、镜片反光，不抬头看门。所有学生继续站在原5×5坐标内，绝不集体回头。旧课桌、金属桌腿、窗、黑板与右墙没有光影重绘。

【镜头、光线与声音】
镜头绝对锁死，不允许任何推近陈雨、数字变焦、焦点猎取、手持呼吸或景深抽动；只允许焦点在原景深内自然保持可辨。暗红应急光仍来自画面右前，在陈雨转来的半边脸上逐渐显露一条很薄的红色侧光；另一半脸和教室阴影维持冷蓝灰，不能新增正面补光。
0.0—2.55秒只有极低电流底噪、陈雨非常轻的衣领/发丝摩擦和近乎不可闻的颈部呼吸；2.75—4.65秒只有陈雨原话；4.65—5.0秒回归压抑底噪。无老师回应、无学生惊叫、无撞门声、无恐怖音效重击、无旁白、无字幕。

【终态与下一镜接口】
5.0秒必须与输入尾帧一致：门缝构图、老师、24名其他学生、课桌与光源完全不动；陈雨身体仍朝讲台，只有头部60—70度朝门缝，露三分之四侧脸，嘴唇刚闭合，眼神正对林昼。这个终态直接触发Clip08林昼受惊后撤；本镜不展示林昼，也不提前出现碰门声音。

【禁止项】
禁止陈雨整个人转身、肩髋跟转、膝脚挪位、身体漂移、从队伍左中移动到画面中心；禁止转角超过70度、180度反折、脖子拉长、断颈、脑袋漂浮；禁止长发突然变短、整片甩起、发夹消失；禁止脸裂、流血、缝线、腐烂、换脸过程、五官游动、多眼多嘴、怪物牙、夸张瞪眼；禁止老师或其他学生回头、走动、做口型、换位、增减；禁止道具换手；禁止镜头移动、推近、变焦、切脸部特写、镜像空间、窗墙换边；禁止新增对白、旁白、撞门声、字幕、台词文字、可读黑板字、乱码、水印、Logo和闪烁。

【验收】
1. 0.0秒陈雨头部确为0度朝讲台，5.0秒只把头转到60—70度，过程连续无抽帧。
2. 陈雨肩、胸、髋、膝、脚尖和原课桌坐标5秒全程不动；其他24名学生与老师也不回头。
3. 门缝黑边、窗左墙右、讲台远端、课桌和灯光全程不漂移。
4. 唯一对白逐字为“老师，新的脸不太合适。”，在2.75—4.65秒完成，平静无怪物腔。
5. 尾帧的三分之四侧脸完整可辨，五官只毫米级错位、皮肤完整，并能直接触发Clip08的受惊因果。
```

## 生成失败时优先降级

若口型使五官崩坏，保留陈雨 0→60—70 度头部运动与尾帧眼神，删去画面中的明显唇形，仅用后期原句配音；绝不能用肩髋跟转来换取更顺的头部动画。
