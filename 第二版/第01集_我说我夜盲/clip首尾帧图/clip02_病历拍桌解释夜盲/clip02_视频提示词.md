# Clip 02 视频提示词（10秒正式版）

## 输入

- 首帧：`clip02_首帧.png`，必须与Clip01尾帧同状态。
- 尾帧：`clip02_尾帧.png`。
- 模式：首尾帧图生视频。
- 总时长：`10.0秒`。
- 运动强度：中低；摄影机固定，关闭自动切镜、推拉摇移、补帧转场和自动字幕。

## 可直接复制的大段正式提示词

```text
制作《第九节课》第二版第1集 Clip02“病历拍桌解释夜盲”的10秒正式画面。首尾帧是硬性视觉边界，严格继承同一白天教室、同一后墙中央略高机位、同一焦段、同一名普通白天班主任、同一25名学生、同一5排×5列座位、同一服装与同一左窗冷白自然光。视频只能完成“老师从首帧现有步位再走一至两步→停在林昼桌边→放下并轻拍唯一病历→老师质问→林昼侧头抬一只手解释→少数同学压低偷笑→落到尾帧”这一条因果链。不要重新表演Clip01的下讲台过程。

【固定身份与坐标】
窗户始终在画面左侧，黑板和讲台在前，前后门在右；教室不镜像。林昼固定5排4列，白天不背包；许眠固定2排2列，低双马尾且头身朝前；陈雨固定3排2列，长直黑发且头身朝前。白天班主任30—45岁，短黑发、深色西装/夹克、浅色衬衫，无强反光眼镜，不是值夜老师。全程只有一张病历：首帧在老师右手，尾帧平放林昼桌面。

【逐段时间轴】
0.0—0.3秒：严格从共享首帧稳定开始。老师已经在画面右侧过道、距林昼一至两步，右手持唯一病历，视线看林昼；林昼仍趴桌。不得让老师返回讲台或改变步位方向。

0.3—1.4秒：老师沿同一右侧过道完成第一小步，脚掌由后跟到前掌自然着地，身体重心随步伐前移；病历受右手控制，只轻微摆动。林昼听见脚步后肩背绷紧一点，仍不抬头。相邻学生可停笔，但不回头。

1.4—2.1秒：老师完成最后半步/第二小步并停在林昼课桌前右侧，身体朝林昼，双脚站稳。老师先低头看一眼病历，再把目光移到林昼；嘴唇尚未开始质问。林昼开始把额头从手臂上抬离，但仍坐着，胸口朝黑板。

2.1—2.7秒：老师右肘自然弯曲，将唯一病历从身前缓慢下降；纸面保持大致水平，先有靠老师一侧的纸边接触桌面，再完整平放在林昼桌面靠过道一侧。左手自然垂下，不参与抢纸。纸张不得穿手、穿桌、弹起、翻卷或复制。

2.7—3.0秒：老师右手在病历上做一次且仅一次克制轻拍，动作幅度很小，手掌/指腹接触后立刻抬离几厘米。声音是一声轻纸响加轻桌面触感，不是重击，不造成全班震动。

3.0—4.9秒：老师保持桌边站位，上身微微前倾，眉间轻收，眼睛看林昼，做与原台词“夜盲？以前怎么没听你说过？”匹配的自然口型。语气普通班主任式怀疑、压低且克制，第一句“夜盲？”略停顿，后半句连贯，不怒吼、不讥笑。右手停在病历上方/纸边附近，左手不拿道具。林昼在这段完成抬头并侧向老师，但不抢话。

4.9—5.2秒：对白后的短停顿。老师嘴唇闭合，维持审视；林昼视线快速扫过病历再回到老师脸上，喉结轻动、眉毛微挑，形成“临场编理由”的因果准备。

5.2—6.2秒：林昼仍坐着，胸口和膝盖朝黑板，只把头和上身小幅侧向右侧老师；人物左手从桌面抬到胸前/桌沿上方，掌心自然向上。另一只手留在桌面。动作克制，手指数量正确，不挥舞。

6.2—8.8秒：林昼用与原台词“最近严重了，医生说天一黑就看不见。”匹配的轻微口型解释。语气要像机灵学生临时圆谎：表面平静、句首略迟疑，后半句刻意说得笃定；不能嬉皮笑脸或夸张装瞎。老师不打断，只以怀疑目光看他。病历保持平放不动。

8.8—9.4秒：林昼说完，嘴唇闭合，人物左手解释姿势收在尾帧高度。老师仍站在桌前，质问表情不变。靠近后排的两三名同学只允许压低嘴角、肩膀轻颤或低头遮笑；不得转身看林昼，不得集体大笑。许眠在2排2列只以眼神轻微察觉后方声音，头、肩、腰、髋、膝盖和脚尖仍朝黑板；陈雨完全朝前。

9.4—10.0秒：所有动作缓慢收住并严格落到正式尾帧：唯一病历平放林昼桌面；老师在林昼前右侧，右手停在纸上方/纸边，嘴唇闭合；林昼坐着侧头、人物左手抬起解释；许眠尚未回头。至少稳定0.3秒，以便下一镜硬切至Clip03近景。

【视线、微表情与对白连锁反应】
老师的视线链必须是病历→林昼趴伏的头肩→林昼抬起的眼睛；质问时眉心轻收、上眼睑略压、嘴角不笑，听解释时只保留怀疑，不点头认可。林昼的视线链必须是桌面病历→老师脸→短暂回避半拍→重新看老师；听到质问时喉结轻动、眉毛微挑，回答开头略迟疑，随后故意把表情压平、语气说得笃定，体现临场圆谎。老师的怀疑导致林昼抬人物左手解释；林昼越装镇定，靠近后排两三名学生越需要低头遮住嘴角和肩部轻颤，但无人转头、无人出声打断。许眠只从声音察觉异常，头身仍朝前，为Clip03的回头保留起点；陈雨不参与反应。

【前中后景反应】
前景第5排：林昼是唯一明显动作主体；相邻学生只停笔、收肩、压笑。中景第2—4排：许眠只用眼神察觉，陈雨和其他人头身朝前。后景：讲台始终空着，黑板无可读文字，老师不得复制回讲台。25名学生与桌椅全程锁定。

【镜头与声音】
摄影机绝对固定，不切近景，不追随老师，不改变焦点范围。声音顺序：0.3—2.1秒两步轻脚步；2.1—2.7秒纸张摩擦；2.7—3.0秒一次轻拍；3.0—4.9秒班主任原话；4.9—5.2秒停顿；6.2—8.8秒林昼原话；8.8秒后极轻压笑与教室底噪。只使用总剧本已有两句对白，禁止增加训斥、回应或旁白。

【终态与下一镜】
10.0秒严格等于正式尾帧。下一镜Clip03会硬切到林昼右后肩附近看许眠的中近景，因此本镜结尾必须让许眠的头身仍完全朝前，病历仍在林昼桌面，林昼侧头解释状态可在切镜后保留为右前景侧肩。

【禁止项】
禁止老师重新从讲台出发、走超过两步、走左过道、瞬移、滑步、穿桌或坐到学生位置；禁止老师变值夜老师、强反光眼镜、手电、点名册；禁止病历多张、漂浮、弹起、发光、出现可读文字；禁止重拍桌、全班受惊、林昼站起或双手乱挥；禁止许眠提前回头、陈雨回头、背景学生围观起立；禁止桌椅漂移、人数增减、教室镜像、窗门换边；禁止新增对白、字幕、气泡、水印、Logo、伪文字；禁止照片写实、3D CGI、低龄萌系、血腥怪物化。
```

## 逐项验收

- [ ] 首帧承接Clip01尾帧，老师只再走一至两步，没有重走讲台路线。
- [ ] 唯一病历由右手平放并只轻拍一次；尾帧平贴桌面、不可读。
- [ ] 班主任与林昼只说总剧本两句原话，语气和口型克制。
- [ ] 林昼全程坐着，最终只侧头并抬人物左手解释。
- [ ] 许眠尚未回头，陈雨和背景学生不围观；桌椅、人数和光线不漂移。
- [ ] 10秒终态准确落到尾帧，并给Clip03第一次回头保留起点。
