# 第二版·第1集 Clip 01—03 外部 AI 生成实操清单

> **归档提示：本文件只保留早期操作思路，不再作为正式投喂文本。当前唯一生产入口是 [`clip首尾帧图/00_第一集逐Clip提示词入口.md`](clip首尾帧图/00_第一集逐Clip提示词入口.md)，生成时必须打开对应的 `clipXX_视频提示词.md` 并完整复制其中正式代码块。最新方案已恢复为“Clip01只上传单首帧并留在讲台讲话；Clip02使用自己的首尾帧独立开始”，本文后续关于“Clip01尾帧等于Clip02首帧”的旧步骤全部作废。**
>
> 用途：用任意支持图生视频或首尾帧生视频的 AI，低成本完成前三个 Clip 的画面草稿、验收、配音与正式导出。
> 生产真源：`00_连续性锁定表.md`、`总剧本.md`、已验收的正式首尾帧。
> 本清单不使用平台自动改写的 16 镜头，也不把九宫格当作正式输入。

## 零、先分清两类提示词

- `clip首尾帧图/*/*帧提示词.md` 是**静态图片提示词**。只有正式首尾帧缺失、损坏或验收不合格时，才用它重新生成图片。
- 本清单中的代码块是**视频运动提示词**。它以已验收的正式 PNG 为硬性视觉边界，主要描述几秒内的动作、镜头和终态，不代替静态图片提示词。
- 两类提示词不是二选一：正确顺序是“先按静态提示词得到合格帧，再用合格帧加逐秒运动提示词生成视频”。
- `00_连续性锁定表.md` 是操作人与验收人必须先读的总规则，不需要整份粘贴给视频模型。

## 一、所有 Clip 先统一这样设置

1. 选择“图生视频”。工具支持“首帧＋尾帧”时，两张都上传；不支持时只上传首帧。
2. 画幅固定 `16:9`；草稿优先用最低成本或快速档；每次只生成 `1` 条。
3. 第一轮关闭自动分镜、自动改写、多镜头、自动运镜、字幕、配音和口型同步，先只验画面运动。
4. 不再上传九宫格或无关角色图。正式首尾帧已锁定角色、服装、场景、光线和空间。
5. 有“首尾帧约束/关键帧强度”时设为高；有“运动强度”时按本清单设置。
6. 正片目标为 `1920×1080`。草稿通过后才用同一提示词、同一随机种子出高清。
7. 失败重做时一次只改一个变量；不要让平台自动重写提示词。
8. 第一轮音频全部关闭。只有画面验收通过后，才添加独立配音、环境声和必要口型。

3. 正式首尾帧已经验收通过，不需要重新生成；尾帧与Clip02首帧逐像素复用同一PNG。

### 第1—2步：检查并上传素材

- 建议时长：`10 秒`
- 运动强度：`中低`
- 镜头：绝对固定；关闭推拉摇移

### 第3步：设置参数并复制逐秒视频提示词

正式提示词不在此处重复维护。请完整复制 [`clip01_视频提示词.md`](clip首尾帧图/clip01_白天教室宣布晚自习/clip01_视频提示词.md) 的“可直接复制的大段正式提示词”；其中已逐秒写明“宣布收束→发现林昼→病历换到右手→讲台右侧下台→沿右侧过道完整走近→落到与Clip02共用的尾帧”。

### 第4步：生成1条草稿并逐项验收

- [ ] 班主任仍是普通白天班主任，不能变成戴强反光眼镜的值夜老师。
- [ ] 林昼从头到尾趴着；许眠、陈雨没有回头。
- [ ] 窗左、门右、讲台在前，桌椅与学生位置没有漂移。
- [ ] 老师完整走下讲台并沿右侧过道到达林昼附近，没有瞬移、滑步或穿桌。
- [ ] 没有字幕、乱码、水印和可辨识黑板文字。

### 第5步：画面通过后再做音频

- 林昼旁白：`我只是想逃一次晚自习。`
- 时序：按独立详细提示词在`1.0—3.2秒`使用旁白，其余段落保留教室底噪、纸响与老师真实脚步。
- 原剧本没有规定班主任宣布时的逐字台词，不额外新增一句对白。
- 老师是远景，不必做精确口型；旁白绝不能驱动人物口型。
- 环境声只用很轻的日间教室底噪，不加夜间虫鸣或恐怖音效。

### 第6步：导出并命名

- 草稿：`clip01_v01_draft.mp4`

## 三、Clip 02｜病历拍桌解释夜盲

### 第0步：先读生产规则

1. 先读 [`00_连续性锁定表.md`](00_连续性锁定表.md) 的“白天教室绝对坐标”“Clip状态链”和“物理与因果锁定”。
2. 再核对静态图片提示词：[`clip02_首帧提示词.md`](clip首尾帧图/clip02_病历拍桌解释夜盲/clip02_首帧提示词.md) 与 [`clip02_尾帧提示词.md`](clip首尾帧图/clip02_病历拍桌解释夜盲/clip02_尾帧提示词.md)。
3. 两张正式帧已经验收通过，不重新生成；首尾顺序绝不能放反。

### 第1—2步：检查并上传素材

- 首帧：[`clip02_首帧.png`](clip首尾帧图/clip02_病历拍桌解释夜盲/clip02_首帧.png)
- 尾帧：[`clip02_尾帧.png`](clip首尾帧图/clip02_病历拍桌解释夜盲/clip02_尾帧.png)
- 建议时长：`10 秒`
- 运动强度：`中低`
- 镜头：固定；关闭自动运镜和自动切镜

### 第3步：设置参数并复制逐秒视频提示词

```text
首帧和尾帧是硬性边界，严格继承其中的同一间教室、同一名班主任、同一批学生、座位、服装和光线。固定后墙略高机位，不切镜、不改变焦段、不跨轴。0—1.8秒：班主任已经位于画面右侧过道，只向林昼再走一至两小步，不要让他重新从讲台出发。1.8—2.8秒：班主任用持有病历的手把唯一一张病历缓慢平放在林昼桌面，轻拍一次，纸张不弹起。2.8—4.8秒：班主任保持站位，做克制的询问表情与轻微口型。4.8—8.8秒：林昼仍坐在原位，只侧头并抬起一只手解释，动作自然克制。8.8—10秒：人物收住动作，稳定到达给定尾帧状态。许眠和陈雨始终身体朝黑板，其余学生保持座位，不围观、不起立。窗户始终在左、门在右、讲台在前。病历只出现一张，桌椅和学生不能漂移。禁止老师瞬移、穿过课桌、病历复制或漂浮、手指畸形、换脸换衣、学生回头、镜像空间、自动切近景、字幕、可读病历文字、水印、Logo和闪烁。
```

### 逐秒动作核对表

| 时间 | 画面动作 |
|---|---|
| 0—1.8 秒 | 老师从首帧已有位置再走一至两小步，病历始终只有一张。 |
| 1.8—2.8 秒 | 老师停在桌边，把病历平放并轻拍一次。 |
| 2.8—4.8 秒 | 老师保持站位，做克制询问表情和轻微口型。 |
| 4.8—8.8 秒 | 林昼轻侧头、抬一只手解释。 |
| 8.8—10 秒 | 所有动作收住，稳定落在正式尾帧。 |

### 第4步：生成1条草稿并逐项验收

- [ ] 老师只从首帧位置再走一至两步，没有重新从讲台出发、穿过课桌或瞬移。
- [ ] 病历全程只有一张，落桌前在右手，落桌后平放桌面。
- [ ] 林昼没有站起；其他学生没有集体回头或围观。
- [ ] 教室、服装、人物身份和日间光线没有改变。
- [ ] 结尾准确接近正式尾帧，能自然硬切到 Clip 03。

### 第5步：画面通过后再做音频

- 班主任：`夜盲？以前怎么没听你说过？`
- 林昼：`最近严重了，医生说天一黑就看不见。`
- 时序：`0.0—1.8秒`脚步与课堂底噪；`1.8—2.2秒`纸张落桌；`2.2—2.5秒`一次轻拍；`2.7—4.9秒`班主任对白；`4.9—5.2秒`短停顿；`5.2—9.2秒`林昼对白；`9.2—10.0秒`留课堂底噪。
- 先用独立配音覆盖。画面与人物稳定后，才考虑为林昼补极轻口型。
- 病历落桌加一声轻纸响；不能做成重拍、撞击或惊吓音效。

### 第6步：导出并命名

- 草稿：`clip02_v01_draft.mp4`

- [ ] 陈雨没有回头；林昼右侧近景位置没有跳动。
- [ ] 病历仍在林昼桌面；窗、门、课桌和日光没有漂移。

- 可做极轻口型；若口型导致换脸，立即取消口型，改为纯配音覆盖。
- 句尾留约 `0.5 秒` 静默，再硬切 Clip 04 夜景。

## 五、统一返工顺序

1. **换脸或发型漂移**：提高首尾帧约束，降低运动强度，沿用同一随机种子。
2. **教室或桌椅漂移**：关闭自动运镜、自动分镜和提示词扩写，把镜头明确设为固定。
3. **动作未走到尾帧**：提高尾帧权重或延长时长，不重写人物和场景描述。
4. **动作过大**：把运动强度再降一级，只保留该 Clip 的一个主动作。
5. **出现字幕、乱码或水印**：关闭模型内文字与字幕功能；所有文字统一后期添加。
6. **口型导致换脸**：放弃模型口型，保留画面，通过剪辑软件覆盖独立配音。
7. 每个 Clip 只有在人物、空间、动作和终态同时通过后，才进入配音与高清输出。

## 六、前三个 Clip 的剪辑连接

1. Clip 01 → Clip 02：同一日间教室；Clip01已经完整拍完老师下讲台并沿右侧过道走近，Clip01尾帧与Clip02首帧逐像素复用同一PNG，切点处不得跳位。
2. Clip 02 → Clip 03：病历留在林昼桌面；许眠听见解释后准备回头。
3. Clip 03 → Clip 04：许眠警告结束后停 `0.5 秒`，硬切至同一天 `23:00 后` 的校园夜景。
4. 不使用溶解、闪白或自动转场；白天到夜晚的时间跳跃靠硬切成立。
