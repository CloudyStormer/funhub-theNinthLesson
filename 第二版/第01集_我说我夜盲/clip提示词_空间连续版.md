# 第1集 Clip提示词：空间连续版

## 本集收紧剧情

林昼为了逃晚自习谎称夜盲。白天的教室是普通真实空间：讲台在前、黑板在讲台后、林昼在后排、许眠在前排。夜晚转到教学楼外，林昼从楼下看见三楼红光，再进入三楼走廊，从教室后门门缝看到第九节课。后半段所有空间都围绕“教室门口”展开：林昼在门外走廊，值夜老师从讲台走向门口，陈雨在教室队伍里回头，最后点名册仍在讲台。

## 全集空间轴

- 白天教室：镜头从教室后排面向讲台，黑板在画面前方/远处；林昼坐后排偏右，身体面向黑板和讲台，在画面里是侧背或后背；许眠坐前排偏左，身体也面向黑板和讲台，只有提醒林昼时才轻微回头；其他同学都面向前方，不看镜头。
- 夜晚教学楼外：林昼在楼下地面，三楼红光教室在画面上方，不要直接跳进室内。
- 三楼走廊：林昼站在教室后门外，门在画面右侧或正前方，红光从门缝漏出。
- 第九节课教室：讲台和黑板在教室前方，值夜老师在讲台，学生队伍在讲台前方或课桌间，门口在画面后侧。
- 点名册特写：点名册在讲台桌面，红光从教室应急灯来，手电光只照纸面。

## 统一负向限制

```text
不要突然换场景，不要新增主要角色，不要让林昼瞬移进教室，不要让门和讲台方向反转，不要让学生队伍突然变坐姿，不要让陈雨变成血腥怪物，不要让老师奔跑，不要让点名册变成手机或课本，不要出现乱码文字、字幕、水印
```

## Clip 01 白天宣布晚自习

```text
白天，高中教室真实空间，镜头从教室后排看向讲台，黑板在讲台后方，班主任站在讲台中央宣布晚自习纪律；林昼坐后排偏右，身体朝向黑板和讲台，趴在桌上听，在画面里是侧背或后背，不正对镜头；许眠坐前排偏左，只露侧背影。路人同学坐在各自座位上，身体都面向黑板和讲台，小声偷笑但不回头看镜头，不站起、不乱走。自然冷色课堂光，普通白天氛围。结尾镜头保持教室方向不变，方便下一段切到班主任把病历拍在桌上。
```

## Clip 02 病历撒谎

```text
承接同一间白天教室，讲台仍在前方，林昼仍坐后排偏右，身体朝向黑板和讲台，没有移动到讲台。班主任从讲台走到林昼课桌旁，把病历拍在林昼桌面；林昼坐着举起病历，身体仍在座位方向，只抬手和侧头解释自己夜盲，不整个人转向镜头。许眠仍在前排，只轻微回头看林昼。路人同学保持坐姿，身体面向前方，只小声偷笑，不围过来。病历纸不要出现可读乱码。结尾林昼仍在座位上，许眠回头的视线留给下一段。
```

## Clip 03 许眠提醒

```text
同一白天教室，同一座位关系，镜头从林昼肩后看向前排许眠；许眠坐前排偏左，身体仍朝向黑板和讲台，只把头和肩膀轻微回向林昼，低声警告“你最好别后悔”。林昼在后排偏右，身体面向前方，只侧头回应，手里病历放回桌面。其他同学在背景虚化，仍面向黑板，不突然站起、不看镜头。黑板和讲台仍在教室前方，不改变教室方向。结尾许眠转回前方，画面可自然切到夜晚校园外景。
```

## Clip 04 夜晚教学楼外看见红光

```text
深夜校园外景，林昼从教学楼一侧空地走过，背着单肩包，地面有冷色反光。镜头在楼下地面，向上看教学楼三楼，三楼某间教室透出暗红色应急灯；其他窗户熄灭。林昼停在楼下，不要直接出现在三楼走廊。校园路人为空，不新增学生。红光位置固定在三楼，为下一段“林昼上楼靠近门口”建立空间方向。
```

## Clip 05 三楼走廊靠近后门

```text
时间仍是深夜，地点切到教学楼三楼走廊。林昼从楼梯口方向走来，靠近高二三班后门；走廊空无一人，墙上有班牌和纪律标语，红光从教室门缝漏到走廊地面。镜头跟在林昼身后或侧后方，门始终在画面前方，不要让林昼瞬间进教室。结尾林昼停在门外，身体贴近门缝，给下一段门缝视角。
```

## Clip 06 门缝里的第九节课

```text
从林昼所在的门外走廊向门缝内窥视，门框两侧保持黑暗压迫。教室内空间固定：黑板和讲台在远处正前方，值夜老师站在讲台前低头看旧点名册，一排学生穿校服站在课桌间，背对门口，身体面向讲台和黑板，只露后背或侧背。林昼本人不进入教室，只在门外呼吸紧张。学生只允许轻微呼吸感，不转身、不走动。结尾值夜老师点到陈雨，镜头仍在门缝视角。
```

## Clip 07 陈雨慢慢回头

```text
承接门缝视角，红光教室空间不变，陈雨在学生队伍中间或偏前位置，身体面向讲台和黑板，背对门口站立。她听到门口动静后只把头非常缓慢地转向门缝方向，身体不转，脸部只有轻微错位感，不血腥、不怪物化。其他学生保持背对门口、面向讲台和黑板不动，值夜老师仍在讲台前，不突然出现在陈雨身后。结尾陈雨头部看向门口，全教室安静，为林昼后退撞门做衔接。
```

## Clip 08 林昼撞门被发现

```text
镜头回到三楼走廊门外，林昼站在教室后门旁，红光从门缝照到他侧脸和书包。林昼被陈雨回头吓到，后退半步，书包轻轻撞到门板或门框，动作真实克制，不摔倒。门内声音瞬间停止，走廊仍空无一人。林昼位置始终在门外，门在他身前，红光方向不变。结尾门内值夜老师开始朝门口走来。
```

## Clip 09 值夜老师走向门口

```text
从门缝或门内低角度看，值夜老师从讲台方向缓慢走向后门，手里拿旧点名册和手电，镜片强反光，看不清眼睛。学生队伍仍留在教室深处不动，黑板和讲台在老师身后，空间方向不要反转。老师不奔跑、不瞬移，步伐慢而压迫。结尾老师靠近门口，林昼在门外准备装瞎。
```

## Clip 10 林昼装瞎

```text
三楼走廊门口同一位置，值夜老师站在教室门内或门口阴影处，林昼在门外走廊。林昼立刻低头，伸手摸墙，故意轻轻撞上门框，假装自己看不见；他的眼神避开教室和老师方向，肩上的书包歪掉。老师停在门口看他，镜片反光。动作符合物理，不夸张喜剧，不摔倒。结尾老师确认“你夜盲”，转身回讲台。
```

## Clip 11 点名册写下待观察

```text
空间回到教室讲台，值夜老师已从门口回到讲台桌前，旧点名册放在桌面，红色应急灯照纸面，手电光只在点名册边缘。镜头做纸面特写，老师用黑色钢笔写下“林昼，待观察”。只允许出现这几个字，不要乱码、不加其他名字。手部、钢笔、纸张位置真实，结尾定格字迹，为第二集“待观察”承接。
```
