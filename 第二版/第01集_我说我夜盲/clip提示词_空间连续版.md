# 第1集 Clip提示词：空间连续生产版

生产前先读 `00_连续性锁定表.md`。本文件用于首尾帧通过质检后的图生视频；不得拿它替代独立首尾帧提示词。

## 全集统一限制

```text
成人向冷灰写实国漫分镜，细密稳定线稿，真实材质，低饱和冷蓝灰；夜景红色应急灯仅局部强调。横屏16:9。角色外观、衣服、发型、道具、门窗、讲台、过道和光源方向必须继承输入首尾帧。一个clip只推进一个主要动作。不要换脸、换衣、换手、镜像空间、瞬移、空桌、复制人、路人看镜头、字幕、水印、Logo、乱码、血腥怪物化、夸张运镜。
```

## Clip 01 白天宣布晚自习（静态单帧）

```text
风格：冷灰写实国漫白天课堂，克制、真实。
视角：固定后墙中央略高机位，面向黑板；只允许极慢2%推镜，不改焦段和构图。
主体：白天班主任站讲台中央宣布纪律；林昼在5排4列趴桌，许眠2排2列、陈雨3排2列朝前。
细节：老师只做轻微口型和一次自然抬手；学生仅呼吸、眨眼、轻微坐姿变化，所有人保持原座位。
背景：5排x5列，窗左、前门右、黑板和讲台在前；不得新增、删除或移动桌椅学生。
光彩：左窗冷白自然光稳定，不闪烁、不变时段。
质量词：静态稳定，脸和手不崩，背景不游动，首尾可无缝循环。
```

## Clip 02 病历拍桌解释夜盲

```text
风格：承接Clip01同一白天教室、同一画风。
视角：同一后墙机位，不切镜、不跨轴；老师沿画面右侧过道由远向近。
主体：白天班主任持病历走到5排4列林昼桌边；林昼始终坐着。
细节：老师右手把单张病历平放到林昼桌面并轻拍一次；林昼只侧头、抬一只手解释。许眠在2排2列轻微察觉，陈雨仍朝前。病历不弹起、不复制。
背景：25名学生一桌一人，身体朝黑板，不围观、不起立，门窗与讲台方向不变。
光彩：左侧冷白窗光连续；老师移动时阴影按同一方向变化。
质量词：动作克制、手纸接触真实、桌椅不漂、角色与座位连续。
```

## Clip 03 许眠低声提醒

```text
风格：同一白天教室，冷灰写实国漫。
视角：固定在林昼右后肩附近，朝左前方2排2列看；镜头不越过林昼与许眠视线轴。
主体：许眠低双马尾，身体、膝盖和脚尖始终朝黑板；林昼仅右后肩和侧脸在前景。
细节：许眠从完全朝前开始，只把头和肩向右后方转约30度，低声说“你最好别后悔”；尾帧保持这个警告姿态，不再转回。林昼轻微侧头听。
背景：陈雨在许眠正后一排3排2列，仍朝前；病历留在林昼桌面；其余学生不回头。
光彩：白天左窗光稳定，肤色自然。
质量词：只做一次小幅回头，双马尾不变单马尾，空间与人物关系稳定。
```

## Clip 04 夜晚教学楼红光

```text
风格：同日23:00后，冷蓝灰校园夜景，湿地反光，写实国漫悬疑。
视角：固定楼下道路广角，四层教学楼完整入画；仰角克制，不切镜。
主体：林昼左肩单肩背黑色双肩包，从画面左侧缓慢走向右侧。
细节：起始未注意教学楼；他走两步后减速、停下、抬头看从地面数第三排的固定窗口。红窗从头到尾同一位置，顶层全部黑。
背景：空校园，无其他学生车辆；树、窗格和楼层数量固定。
光彩：冷蓝夜色稳定，第三层一扇窗有压暗红光，不爆白、不扩散到别层。
质量词：走停抬头节奏清晰，楼层准确，脚步和地面接触真实。
```

## Clip 05 三楼走廊靠近后门

```text
风格：23:00后冷灰蓝三楼走廊，门缝暗红光，成人向悬疑国漫。
视角：摄影机固定在走廊左后侧，纵深看向右前方高二三班后门；跟随幅度很小，不换边。
主体：林昼从左后方楼梯平台进入，左肩单肩背包，向右前方后门走近。
细节：走三至四步后放慢，左侧贴近门框，左眼靠近约5厘米宽门缝；身体不进教室。尾帧背包在左后背、外缘邻近向走廊微开的门扇。
背景：楼梯在左后，后门在右前，门把手朝走廊；红光在地面形成固定窄梯形。走廊无人。
光彩：顶灯冷蓝微弱且稳定，红光只从门缝漏出。
质量词：路线连续，门扇开合不变，脚步真实，背包不换肩。
```

## Clip 06 门缝里的第九节课（静态单帧）

```text
风格：23:00后红光教室，冷灰写实国漫，安静压迫，不血腥。
视角：固定林昼左眼主观门缝机位；两侧门框黑边不移动，远端黑板和讲台居中。
主体：值夜老师在讲台前，左手旧点名册、右手手电；陈雨和其他学生背对后门、面向讲台站立。
细节：老师低头点名，只做一次翻页和轻微口型；学生只做呼吸，陈雨此时绝不回头。
背景：窗在左、右墙在右，学生和课桌位置不变；林昼不入画。
光彩：压暗红色应急灯稳定，手电弱光只扫点名册边缘。
质量词：静态低运动，角色不漂，学生不转身，门缝与讲台空间稳定。
```

## Clip 07 陈雨缓慢回头

```text
风格：完全承接Clip06同一红光教室与门缝视角。
视角：机位、焦段、门缝黑边和所有学生站位锁死，仅极慢推近陈雨。
主体：陈雨位于队伍左中区域，齐刘海、长直黑发、蓝白校服；身体背对门、面向讲台。
细节：被点名后她察觉窥视，只把头向右后方缓慢转60-70度，尾帧露三分之四侧脸；躯干、髋、膝盖和脚尖全程不动。五官只轻微错位，其他学生和老师不动。
背景：讲台、黑板、课桌、窗和门缝方向完全不变。
光彩：同一暗红主光，转头时脸侧逐渐进入红光，不新增光源。
质量词：单一转头动作，无180度鬼转头，无身体扭曲，无怪物化。
```

## Clip 08 林昼后退撞门

```text
风格：23:00后三楼冷灰走廊，门缝红光，克制惊吓。
视角：固定走廊左侧中近景，能同时看到林昼侧身、左后背书包、微开的门扇和门把手。
主体：林昼左眼贴近门缝，左肩单肩背包；门扇就在他的左侧/左后方，向走廊微开。
细节：他受惊后向右后方缩半步并把左肩向后扭，左后背包的外缘轻碰门扇，门扇震一下发出轻响；他不摔倒、不进入教室。门内声音在碰撞后停止。
背景：走廊无人，楼梯方向、后门位置、红光落地形状承接Clip05。
光彩：冷顶灯与门缝红光不变，包碰门时只出现轻微材质反光。
质量词：碰撞几何正确，背包不换肩，门不穿模，动作小而清楚。
```

## Clip 09 值夜老师走向门口

```text
风格：承接Clip06/07同一红光教室，压迫感逐步增加。
视角：固定在后门内侧低机位朝讲台方向拍，不使用门缝机位，不切反打；老师沿画面右侧过道由远向近。
主体：值夜老师从讲台前闻声抬头，左手旧点名册、右手手电，金属眼镜强反光。
细节：他先转身朝后门，再沿右侧过道缓慢走近，步伐完整可见；尾帧停在后门内侧一步处。点名册和手电全程不换手，手电光束随右手轻微摆动。
背景：学生队伍留在教室深处；除陈雨头部继续保持Clip07尾帧的60-70度回看门缝外，所有学生躯干、髋部、膝盖和脚尖仍面向讲台，陈雨头部不得复位。黑板和讲台始终在老师身后远处。
光彩：红光从同一侧照老师，手电弱光不扫镜头。
质量词：不奔跑、不瞬移、不换手，空间不反转，慢而真实。
```

## Clip 10 林昼撞门框装瞎

```text
风格：同一后门正反打，冷灰走廊与红光教室交界，紧张克制。
视角：固定走廊侧面中景；林昼在门外画面左侧，值夜老师在门内画面右侧，180度轴保持不变。
主体：林昼低头避开老师目光；值夜老师停在门内，左手点名册、右手手电。
细节：林昼低头，先抬人物左手贴住门框旁的灰墙，再向前试探半步，让左手掌缘故意轻碰门框，装作看不见；左肩背包留在身后且不再碰门。这是故意的第二次接触，不是Clip08的书包误撞。老师静看三秒后僵硬微笑，轻轻点头，尾帧开始向教室内转身。
背景：门、门框、门把手、走廊纵深和红光方向不变；学生仍在教室深处。
光彩：门内红光勾出老师轮廓，走廊冷光照林昼半边脸。
质量词：表演克制，肩触门框物理正确，两人不穿门，不夸张喜剧。
```

## Clip 11 点名册写下待观察

```text
风格：同一红光教室讲台，悬疑收束，纸张与手部写实。
视角：匹配剪辑到讲台桌面45度俯拍特写；明确已省略老师从门口走回讲台的数秒。
主体：旧点名册平放讲台，值夜老师右手持黑色钢笔；手电已放在讲台左侧，左手压住纸页。
细节：从笔尖尚未落纸开始，老师缓慢写完“林昼，待观察”，笔停在最后一个字右下方。点名册、手、钢笔和手电不漂移；若模型文字不准确，正式成片使用后期文字图层覆盖。
背景：只露讲台木纹、黑板下沿和虚化红光教室，不新增人物。
光彩：红色应急灯从右上方照纸面，讲台左侧手电形成固定小范围冷白光。
质量词：书写动作自然，手指正确，文字位置可控，结尾稳定停留三秒。
```
