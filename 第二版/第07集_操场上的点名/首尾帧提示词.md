# 第7集首尾帧提示词

## Clip 01 首帧

```text
深夜校园，全校停电，宿舍楼和教学楼出口刚有人走出，林昼混在人群里，许眠在不远处，学生安静低头
```

## Clip 01 尾帧

```text
同一深夜校园，人流已经走到操场入口，学生安静进入操场，不奔跑不说笑，冷蓝月光和红色应急灯远处闪烁
```

## Clip 02

```text
夜晚操场，学生按班级方阵站好，队伍正对主席台，所有学生身体和鞋尖朝向主席台，值夜老师站在主席台下中央，林昼在方阵中段，许眠在旁边或斜前方
```

## Clip 03 首帧

```text
同一操场方阵，一个学生刚被点到名，准备从队伍前侧走出，值夜老师仍在主席台下翻点名册
```

## Clip 03 尾帧

```text
同一操场，那个学生已经站到方阵和主席台之间，脸上像有薄膜裂开但不流血，其他学生保持队形
```

## Clip 04 首帧

```text
操场方阵中，林昼刚闭上眼，许眠仍在他旁边，周围学生队伍轮廓清楚
```

## Clip 04 尾帧

```text
同一方阵，林昼闭眼听见点名声贴近耳边，身体紧绷，操场空间仍保留，不切成幻觉黑场
```

## Clip 05 首帧

```text
林昼在方阵中猛地睁眼，远处值夜老师还低头看点名册，许眠站在旁边
```

## Clip 05 尾帧

```text
同一操场，远处值夜老师已经抬头看向林昼方向，林昼低头准备按许眠提醒看脚，双方距离仍很远
```

## Clip 06

```text
操场地面低角度，林昼只看周围学生鞋尖，鞋尖和小腿朝主席台，身体仍保持方阵方向；只有地面影子被红色应急灯拉偏，像压向林昼方向，不要让学生身体真的转向林昼，地面白线和粉笔痕清楚
```

## Clip 07

```text
主席台旁边边缘，真正夜盲的学生戴深色眼镜，手持折叠盲杖，安静站着不看任何人，值夜老师在主席台下点名，学生方阵在前景
```

## Clip 08 首帧

```text
值夜老师点名册特写，页面刚翻到林昼相关位置，红光和冷月光混合，背景是操场黑暗
```

## Clip 08 尾帧

```text
操场方阵，所有学生已经同时转头看林昼，但身体和鞋尖仍朝主席台，队形不乱，林昼僵住，许眠在旁边
```

## Clip 09 首帧

```text
操场地面，真正夜盲学生的盲杖刚点向地面，粉笔线还没完全出现，林昼低头看
```

## Clip 09 尾帧

```text
同一地面特写，粉笔线已经写出“跟我走”的感觉，林昼看到这行字，周围学生鞋尖静止，操场没有清空
```
