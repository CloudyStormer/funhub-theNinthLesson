# 第7集 Clip提示词：空间连续版

## 本集收紧剧情

全校从宿舍和教学楼走向操场，操场方阵、主席台和值夜老师位置要固定。点名时学生走出队伍，林昼和许眠始终在同一班级方阵里。

## 全集空间轴

- 操场入口：学生从宿舍楼和教学楼方向进入操场。
- 操场方阵：班级队伍正对主席台，所有学生身体和鞋尖默认朝向主席台，值夜老师在主席台下；学生转头时只能转头，身体和鞋尖仍保持队形方向。
- 林昼位置：在班级队伍中，许眠在旁边或斜前方。
- 真正夜盲学生：在主席台旁边边缘，不进入林昼队伍。

## 统一负向限制

```text
不要把操场画成礼堂，不要让学生乱跑，不要让方阵方向反转，不要让主席台位置跳动，不要血腥开裂，不要新增大批老师抢戏
```

## Clip 01 全校停电集合

```text
深夜校园，全校停电，宿舍楼和教学楼出口有学生安静走向操场。林昼混在人群里，许眠在不远处。路人学生只披校服低头走路，不说笑不奔跑。镜头从校园路面跟随人流到操场入口。结尾人群进入操场。
```

## Clip 02 操场方阵建立

```text
夜晚操场，学生按班级方阵站好，队伍正对主席台，所有学生身体和鞋尖朝向主席台。值夜老师站在主席台下方中央，手拿点名册。林昼在方阵中段，许眠在旁边或斜前方。操场边缘、主席台、班级队伍方向清楚。结尾老师开始点名。
```

## Clip 03 被点名学生出列

```text
同一操场方阵，一个学生被点名后从队伍前侧走出，站到方阵和主席台之间。脸上像有薄膜裂开但不流血。其他学生保持队形，不乱动。值夜老师仍在主席台下翻点名册。结尾林昼闭眼试图装看不见。
```

## Clip 04 闭眼听见点名

```text
林昼站在方阵中，闭上眼，周围操场声音压低，点名声像贴近耳边。画面仍保留操场队伍轮廓，不切成黑色幻觉空间。许眠仍在他旁边。结尾林昼猛地睁眼。
```

## Clip 05 睁眼被察觉

```text
林昼睁眼的瞬间，远处主席台下的值夜老师抬头看向林昼方向。队伍方向不变，老师与林昼之间隔着方阵距离，不要让老师瞬间贴近。许眠低声提醒他别看脸。结尾林昼低头看脚。
```

## Clip 06 只看鞋尖

```text
低角度操场地面视角，林昼只看周围学生鞋尖。学生鞋尖和小腿都朝主席台，身体仍保持方阵方向；只有地面影子被红色应急灯拉偏，像压向林昼方向，形成不协调感。不要让学生身体真的转向林昼。地面白线和粉笔痕清楚，人物不漂浮。结尾镜头抬一点，看到主席台旁真正夜盲学生。
```

## Clip 07 真正夜盲学生出现

```text
主席台旁边边缘，一个戴深色眼镜、拿折叠盲杖的学生安静站着，不看任何人。值夜老师在主席台下点名，学生方阵在前景。真正夜盲学生不进入方阵，不突然靠近林昼。结尾老师翻到林昼正式观察。
```

## Clip 08 林昼正式观察

```text
值夜老师点名册特写，页面写着“林昼，正式观察”的感觉。切回操场方阵，所有学生同时转头看林昼，但鞋尖仍保持朝主席台，动作诡异但物理上站位不变。结尾盲杖轻敲地面。
```

## Clip 09 粉笔线跟我走

```text
操场地面特写，真正夜盲学生的盲杖轻点地面，粉笔线像刚出现一样写出“跟我走”的感觉。林昼低头看见这行字，许眠在旁边还被队伍限制。方阵仍在周围，不要突然清空操场。结尾林昼准备跟着粉笔线离开。
```
