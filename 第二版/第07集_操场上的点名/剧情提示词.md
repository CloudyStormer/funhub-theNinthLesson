# 第7集剧情提示词：操场上的点名

## 统一正向风格

```text
国漫写实漫画质感，深夜校园操场恐怖，全校停电后的冷蓝月光和主席台红色应急灯对比，方阵、人群、点名册、粉笔线作为核心视觉符号，压迫感强，不血腥
```

## 统一负向限制

```text
不要血腥开裂，不要怪物化，不要大规模混乱奔跑，不要乱码文字，不要让操场像演唱会，不要让学生表情夸张，不要新增无关老师角色
```

## 关键剧情提示词

### 01 全校停电集合

```text
深夜高中宿舍和教学楼同时停电，学生披着校服从不同楼门安静走向操场，校园只有冷蓝月光和远处红色应急灯，林昼混在人群里，表情紧张，画面安静压迫
```

### 02 操场方阵

```text
夜晚学校操场，学生按班级方阵站好，所有学生身体和鞋尖朝向主席台，没有人说话，值夜老师站在主席台下，白衬衫，反光眼镜，手里拿旧点名册，主席台红色应急灯照亮半边身体，远景压迫构图
```

### 03 被点名学生脸膜裂开

```text
一个学生从方阵里走出，脸上像有一层薄薄的假膜轻轻裂开但不流血，不恐怖怪物化，其他学生静止不动，值夜老师低头点名，操场冷蓝光，画面克制
```

### 04 闭眼听见点名

```text
林昼在人群中闭上眼，周围操场声音消失，值夜老师的声音像贴到耳边；画面表现为林昼脸部近景、黑暗边缘压缩、冷汗细节，不出现夸张幻觉怪物
```

### 05 许眠提醒看脚

```text
操场方阵侧面，许眠站在林昼旁边，低马尾，校服整齐，压低声音提醒他不要看脸；林昼低头看脚，周围学生身体和鞋尖仍朝主席台，只有头部可能轻微偏向林昼，队形不乱，红色应急灯在远处闪
```

### 06 鞋尖和身体方向不一致

```text
操场地面低角度特写，一排学生鞋尖和小腿全部朝向主席台，身体仍保持方阵方向；只有地面影子被红色应急灯拉偏，像压向林昼方向，地面白线和粉笔痕清晰。不要让学生身体真的转向林昼，不要出现血或变形脚
```

### 07 真正夜盲的学生出现

```text
主席台旁边，一个17岁中国高中男生戴深色眼镜，手持折叠盲杖，校服干净，安静站着不看任何人，周围学生像没注意到他，红色应急灯远远闪烁，他表情平静神秘
```

### 08 林昼正式观察

```text
旧点名册特写，值夜老师翻到林昼一页，页面上出现“林昼，正式观察”的感觉，手电光照亮纸页，背景是操场黑暗和学生方阵，文字尽量清楚，不要生成其他乱码
```

### 09 粉笔线写着跟我走

```text
操场地面特写，折叠盲杖轻轻点地，地上粉笔线像刚出现一样写着“跟我走”的感觉，林昼低头看到这行字，周围学生鞋尖静止，画面低角度压迫
```
