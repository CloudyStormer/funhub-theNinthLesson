# 第10集剧情提示词：谁替我坐下

## 统一正向风格

```text
国漫写实漫画质感，红光第九节课教室心理恐怖，低饱和冷蓝灰绿色调，点名册、粉笔、备用林昼、黑板文字作为核心视觉符号，压迫感强，不血腥，人物五官稳定
```

## 统一负向限制

```text
不要血腥，不要怪物化，不要乱码文字，不要让备用林昼差异过大，不要夸张打斗，不要把老师画成怪物，不要让黑板出现大段不可读文字，不要新增无关角色
```

## 关键剧情提示词

### 01 备用林昼满教室

```text
红光第九节课教室，黑板上有“备用林昼”的感觉，后排多个和林昼相似的学生坐在不同座位上，身体默认朝向讲台和黑板，外貌相似但气质不同，林昼站在讲台旁，黑色布带和盲杖在手边，值夜老师站在旁边
```

### 02 好学生备用林昼

```text
教室后排第一个备用林昼坐在座位上慢慢坐直，身体朝向讲台和黑板，校服扣好，双肩包摆放端正，笑容标准，眼神没有温度，像完美好学生版本的林昼；其他备用林昼仍坐在旁边座位
```

### 03 真夜盲备用林昼

```text
教室后排第二个备用林昼坐在座位上，身体朝向讲台和黑板，眼神空洞，手里拿着一份病历；校服整齐但脸色发白，像已经承认自己是真的夜盲。病历不要乱码，不要血腥病变
```

### 04 镜中林昼诱惑

```text
镜中林昼坐在最后一排座位上，身体朝向讲台和黑板，外貌与林昼相似但校服更整齐，标准微笑，眼神没有温度；其他备用林昼在后排静止，镜中林昼不离开座位，不怪物化
```

### 05 许眠被挡在门外

```text
第九节课教室后门外，许眠被一排学生拦在门口边缘，无法进入教室深处；门内远处可见林昼站在讲台旁，值夜老师在讲台后方。许眠低马尾、校服整齐，焦急但克制，门口、讲台、后排方向保持一致
```

### 06 全班举手选择

```text
红光教室后排，所有备用林昼仍坐在座位上，身体朝向讲台和黑板，同时举手，动作整齐到不自然，林昼站在讲台旁握紧粉笔，没有看他们的脸，值夜老师微笑站在旁边，低角度压迫构图
```

### 07 林昼反向点名

```text
点名册特写，林昼把粉笔或钢笔按在空白处，写下“值夜老师，到”的感觉，手指用力，纸张泛黄，值夜老师的反光眼镜在后方虚化，文字尽量清楚，不要添加其他乱码
```

### 08 老师第一次失控

```text
讲台前，值夜老师的笑容停住，金属边眼镜不再反光，一瞬间露出空白的眼睛但不怪物化；林昼仍低头不看老师脸，后排备用林昼静止，黑板在老师身后，教室安静
```

### 09 真正夜盲学生门外发声

```text
教室后门外阴影里，真正夜盲学生戴深色眼镜，手持折叠盲杖，平静站着；许眠在旁边回头看他。两人都不进入讲台区，门内红光照出轮廓，林昼和值夜老师仍在远处讲台方向
```

### 10 黑板写下老师待观察

```text
红光教室黑板特写，黑板在讲台后方，原本的备用林昼字样被新字覆盖，出现“值夜老师，待观察”的感觉；林昼背影站在讲台前，值夜老师在旁边僵住，后排所有备用林昼坐在座位上放下手，结尾定格
```
