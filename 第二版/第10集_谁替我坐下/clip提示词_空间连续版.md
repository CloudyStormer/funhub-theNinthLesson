# 第10集 Clip提示词：空间连续版

## 本集收紧剧情

全集发生在第九节课教室和门口，不换大场景。林昼在讲台附近，备用林昼在后排座位，许眠位于门外或门口边缘；真正夜盲学生在 Clip 09 只以画外音存在，人物、脸、手、身体和盲杖均不入画。

## 全集空间轴

- 讲台区：林昼、值夜老师、点名册、粉笔。
- 后排座位：多个备用林昼坐在后排，身体默认朝向讲台和黑板；除举手外不要离开座位。
- 教室门口：许眠被学生拦住，真正夜盲学生后面出现。
- 黑板：在讲台后方，文字变化必须基于同一黑板。

## 统一负向限制

```text
不要让备用林昼瞬移到讲台，不要让许眠突然进入后排，不要让真正夜盲学生凭空出现在讲台，不要让黑板位置反转，不要乱码文字，不要打斗
```

## Clip 01 备用林昼满教室

```text
红光第九节课教室，林昼站在讲台旁，值夜老师在讲台后，后排座位上坐着多个备用林昼，身体默认朝向讲台和黑板。黑板在讲台后方，写着“备用林昼”的感觉。每个备用林昼都在座位上，不站到走廊。结尾老师要求林昼选择。
```

## Clip 02 好学生备用林昼

```text
教室后排第一个备用林昼坐直，校服整齐，笑容标准，像好学生版本。镜头只切到后排座位，不改变讲台和黑板方向。其他备用林昼仍坐在旁边座位。结尾切到第二个备用林昼。
```

## Clip 03 真夜盲备用林昼

```text
后排第二个备用林昼眼神空洞，手里拿病历，像承认自己是真的夜盲。病历不要乱码。座位位置仍在后排，红光照脸，不血腥。结尾镜头转向最后一排镜中林昼。
```

## Clip 04 镜中林昼诱惑

```text
最后一排，镜中林昼坐在座位上，校服更整齐，标准微笑，眼神没有温度。其他备用林昼在周围静止。镜中林昼不离开座位，不瞬移到讲台。结尾回到讲台前林昼。
```

## Clip 05 许眠被挡在门外

```text
教室后门门口，许眠被一排学生拦在门外或门口边缘，无法进入教室深处。林昼在讲台方向远处可见，值夜老师在讲台旁。门口、讲台、后排空间保持一致。许眠提醒林昼别选。结尾切回讲台点名册。
```

## Clip 06 全班举手

```text
值夜老师翻开点名册说不选就由全班举手。镜头转向后排，所有备用林昼坐在座位上同时举手，身体仍朝向讲台和黑板，动作整齐。林昼仍在讲台旁握粉笔，不看他们的脸。空间不变，不出现打斗。结尾林昼把粉笔按向点名册。
```

## Clip 07 林昼反向点名

```text
讲台点名册特写，林昼把粉笔或钢笔按在空白处，写下“值夜老师，到”的感觉。值夜老师站在讲台后方，镜片反光。文字只保留核心句，不要乱码。结尾教室安静，备用林昼放下手。
```

## Clip 08 老师失控

```text
讲台前，值夜老师笑容停住，镜片反光消失一瞬，露出空白眼睛但不怪物化。林昼仍低头不看老师脸。后排备用林昼静止，黑板在老师身后。结尾门外传来真正夜盲学生声音。
```

## Clip 09 门外声音

```text
固定教室右后门口，许眠独自在门外边缘，听见走廊深处真正夜盲学生的画外音后只转头看向非具象暗处。真正夜盲学生的人物、脸、手、身体和盲杖全部不入画；门口不得新增第二根盲杖。门内红光只勾出许眠和门框轮廓，林昼与老师仍在远处讲台方向；结尾只用教室前方的极轻擦写声引出黑板变化。
```

## Clip 10 黑板写下老师待观察

```text
同一教室黑板特写，原本“备用林昼”的字样被新字覆盖，出现“值夜老师，待观察”的感觉。林昼背影在讲台前，值夜老师站在旁边僵住，后排备用林昼安静放下手。黑板位置保持在讲台后方，最后定格。
```
