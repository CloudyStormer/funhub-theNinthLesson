# 第10集首尾帧提示词

## Clip 01

```text
红光第九节课教室，林昼站在讲台旁，值夜老师在讲台后，后排座位上坐着多个备用林昼，身体都朝向讲台和黑板，黑板在讲台后方写着“备用林昼”的感觉
```

## Clip 02 首帧

```text
教室后排第一个备用林昼刚坐直，校服整齐，笑容标准，其他备用林昼坐在旁边座位，讲台方向不变
```

## Clip 02 尾帧

```text
同一后排，第一个备用林昼已经抬头露出标准笑容，像好学生版本，仍坐在座位上不离开
```

## Clip 03 首帧

```text
后排第二个备用林昼坐在座位上，手里拿病历，眼神开始变空，红光照脸
```

## Clip 03 尾帧

```text
同一座位，第二个备用林昼已经抬起病历，像承认自己是真的夜盲，病历不要乱码，不血腥
```

## Clip 04 首帧

```text
最后一排，镜中林昼坐在座位上，低头微笑还没完全抬起，其他备用林昼静止
```

## Clip 04 尾帧

```text
同一最后一排，镜中林昼已经抬头标准微笑，眼神没有温度，仍坐在座位上，不离开后排
```

## Clip 05 首帧

```text
教室后门门口，许眠刚被一排学生拦在门外，林昼在远处讲台方向可见，红光从教室内照出
```

## Clip 05 尾帧

```text
同一门口，许眠已经被挡住无法进入，焦急提醒林昼别选，拦路学生站成一排，不进入讲台区
```

## Clip 06 首帧

```text
讲台前，值夜老师翻开点名册，林昼站在旁边握粉笔，后排备用林昼还没举手
```

## Clip 06 尾帧

```text
同一教室后排，所有备用林昼仍坐在座位上，身体朝向讲台和黑板，已经同时举手，林昼仍在讲台旁不看他们的脸，空间不变
```

## Clip 07 首帧

```text
讲台点名册特写，林昼刚把粉笔或钢笔按到空白处，值夜老师站在讲台后方，镜片反光
```

## Clip 07 尾帧

```text
同一点名册特写，空白处已经写下“值夜老师，到”的感觉，只保留核心句，不要乱码，教室安静
```

## Clip 08 首帧

```text
讲台前，值夜老师笑容刚停住，镜片仍反光，林昼低头不看老师脸，后排备用林昼静止
```

## Clip 08 尾帧

```text
同一讲台前，值夜老师镜片反光消失一瞬，露出空白眼睛但不怪物化，林昼仍低头，黑板在老师身后
```

## Clip 09

```text
固定第九节课教室右后门口，许眠独自在门外边缘，正回头看向走廊深处一片非具象暗处；真正夜盲学生只作为画外音存在，人物、脸、手、身体和盲杖全部不入画，门口不得出现第二根盲杖；门内暗红光只勾出许眠与门框轮廓，讲台保持在教室远处，不生成任何可读文字
```

## Clip 10 首帧

```text
第九节课教室黑板特写，原本“备用林昼”的字样还在，林昼背影站在讲台前，值夜老师在旁边僵住
```

## Clip 10 尾帧

```text
同一黑板，原本字样被新字覆盖，出现“值夜老师，待观察”的感觉，后排备用林昼放下手，最后定格
```
