# 第二版交付目录

这个目录是干净交付版，用来把总剧本、剧情提示词和人物抽卡提示词集中放在一起。

## 目录结构

- `抽卡/`：每个人物一个文件夹，包含 `人物提示词.md`、`多视角提示词/` 和 `图片占位/`。
- `封面图/`：项目正式封面；当前正式版本为 `第九节课_项目封面_v2.png`。
- `../第二版_平台建集参考图/`：平台“新建集·开场设置”专用的共享锚点、十集整组上传包、填表说明与质检记录；与本目录同级，旧素材不受影响。
- `第九节课_第二版_10集分集剧本_导入稿.txt`：供漫剧平台选择“从小说/剧本导入”与“分集剧本”时直接上传。
- `第01集_我说我夜盲/` 到 `第10集_谁替我坐下/`：每集一个文件夹。
- 每集文件夹内固定包含：
  - `总剧本.md`：本集完整分镜剧本、台词和结尾钩子。
  - `剧情提示词.md`：本集关键剧情画面提示词，方便生成关键帧或图生视频。
  - `clip提示词_空间连续版.md`：更细的生产版 clip 提示词，包含时间轴、空间轴、角色站位、路人状态和前后帧衔接。
  - `首尾帧提示词.md`：用于生成每个 clip 的首帧/尾帧；静态镜头只给一个提示词，动作镜头拆成首帧和尾帧。

## 统一项目名

项目名：《夜盲同学》  
类型：校园恐怖 / 悬疑反转 / 规则怪谈  
核心钩子：林昼为了逃晚自习谎称夜盲，却因为这个谎暂时保命；夜里的东西不断试探他到底能不能看见。

## 生产提醒

角色出图先用 `抽卡/` 内的角色锚点锁脸。每集剧情图或图生视频生成时，先用 `首尾帧提示词.md` 生成关键首尾帧，再用 `clip提示词_空间连续版.md` 生成视频；`剧情提示词.md` 作为简版参考。
