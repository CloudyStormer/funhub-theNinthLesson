# 《夜盲同学》项目总览

## 当前目标

先做一部校园恐怖 AI 漫剧样片，不追求一口气做完整季。

第一阶段交付：

- 第1集成片样片，60-90秒
- 前3集完整剧本和分镜
- 4个固定角色设定
- 3个固定场景设定
- 1套可复用画风提示词
- 后续投平台用的项目资料包

## 你现在要理解的事

这个项目分成三层：

1. **故事层**：剧名、世界观、人物、规则、分集。
2. **制作层**：角色图、场景图、关键画面、配音、字幕、剪辑。
3. **发行层**：封面、标题、简介、AI标识、投稿资料、数据复盘。

先不要被全部流程吓到。今天只做第1集样片资产。

## 第一集成片结构

```text
0-10秒：男主为了不上晚自习，谎称夜盲
10-25秒：晚上路过教学楼，发现三楼亮红光
25-45秒：门缝里看见第九节课和换脸仪式
45-65秒：老师发现他，男主装作看不见
65-75秒：点名册写下“林昼，待观察”
```

## 核心钩子

男主的谎言不是“他夜盲”，而是“他明明看得见，却必须一直播瞎”。

## 今天最小任务

1. 生成角色设定图。
2. 生成第1集6张关键画面。
3. 用配音稿和字幕稿做出粗剪。
4. 看完粗剪后，只改节奏，不改世界观。
