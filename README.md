# 项目文件结构

- `人物抽卡/`：角色卡、人物设定图。
- `剧集/第一集/`：第 1 集片段、首尾帧、prompt、视频、剪辑版本。
- `剧集/第二集/`：第 2 集片段 prompt 和后续待生成素材位置。
- `项目资料/`：世界观、剧本、流程、资产清单、版本记录等项目文档。

每个片段文件夹一级固定放：`首帧.png`、`尾帧.png`、`视频.mp4`、`prompt.md`。

第 1 集的首尾帧专业建议见：`剧集/第一集/首尾帧连续性制作建议.md`。

## 当前使用：第二版

- `第二版/`：当前十集剧本、角色抽卡、第一集正式 clip 首尾帧与空间连续提示词。
- `第二版_平台建集参考图/`：漫剧平台“新建集·开场设置”专用；包含共享角色/空景锚点和十个可整组拖入的每集上传包。
- 平台创建每集时选择“剧本起”，粘贴该集 `总剧本.md`；参考图 1–9 张是上限，不必凑满。
