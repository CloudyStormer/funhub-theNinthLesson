# 平台“新建集”参考图包

本目录专门对应漫剧平台的“新建集·开场设置”。原有 `第二版/`、抽卡图和各 clip 首尾帧均保留不动。

## 先说结论

- “参考图 1–9 张”表示**最多 9 张**，不是必须每集凑满 9 张。
- 十集统一选择 **“剧本起”**。现有 `总剧本.md` 已是完整分集剧本，不能再选“全流程”或“小说起”让平台二次改写。
- 每集只上传 4–6 张在整集内都成立的角色/空间锚点。
- 进入工作台后，每个 clip 的首尾帧再到对应 clip 单独上传；不要把整集首尾帧一次性塞进开场参考图。

## 最省事的上传方法

进入 `01_每集整组上传包/`，打开当前集文件夹，按文件名顺序全选其中的 PNG，整组拖进“参考图”。文件名开头的 `01_`、`02_` 同时代表推荐优先级。

每集包都已控制在平台上限以内，只包含该集需要的角色和一张主空间锚点。不要额外混入旧版图片、总角色拼图、封面或其他集的角色。

## 表单逐项填写

1. **创作起点**：选“剧本起”。
2. **对应集数**：选择正在创建的那一集。
3. **集号**：依次填 1–10，不重复。
4. **集标题**：按下表填写。
5. **剧本**：粘贴对应 `第二版/第XX集.../总剧本.md` 的全文；不要粘贴十集导入稿、`剧情提示词.md` 或 `clip提示词_空间连续版.md`。
6. **参考图**：整组拖入本目录中对应集的上传包。
7. **风格预设**：保持“继承项目风格”勾选，不再叠加其他风格。
8. **制作规格**：保持项目锁定项；如可选画幅/清晰度，使用 16:9、1920×1080。
9. **视频模型**：先保留界面当前紫色选中的 Seedance 2.0；进工作台后先试 1–2 个 clip，再按效果和成本决定是否切换。
10. 点击“创建并进入工作台”。

## 三类图片不要混用

- `00_共享锚点/角色/`：锁定整季角色身份；同一角色跨集复用同一张。
- `00_共享锚点/场景/`：锁定固定空间的方向和布置，均为空景，避免平台把参考图中的人强塞进镜头。
- `第二版/第01集.../clip首尾帧图/`：只在进入工作台后给对应 clip 使用，负责动作起点、动作终点、空间轴和时间轴衔接。

剧情中的旧照片、病历、点名册、盲杖、布带和“备用林昼”等是局部道具或状态变化，不作为整集开场身份参考；需要时放到对应 clip 再生成或绑定。

## 质量边界

- 新上传包不使用带“可灵AI”字样的旧图，不使用 `00_角色总参考图.png`，也不混入第一版/旧版画风。
- 白天班主任和值夜老师是两个人，不能互相代替。
- 镜中林昼必须与林昼保持同脸同发型同校服，只允许气质和细微表情异常。
- 第 8 集值夜老师只出声，不放进开场参考图；第 9、10 集真正夜盲学生只通过道具或声音延续，也不放进开场参考图。
