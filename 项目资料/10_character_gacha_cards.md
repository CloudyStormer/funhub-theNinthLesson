# 角色抽卡池

## 使用方式

先不要急着出最终图。你先从每个角色里选一个方向：

- 林昼：选1张
- 许眠：选1张
- 值夜老师：选1张
- 陈雨：选1张
- 隐藏角色：选1-2张

选完后，我再把每张卡扩成“正面、侧面、表情、动作、出图Prompt、图生视频Prompt”。

# 男主：林昼

## A卡：懒散聪明型

定位：最适合短剧主角，观众容易代入。  
形象：黑色短发，校服外套敞开，里面白T，背包单肩背，眼神机灵，表情有点欠。  
气质：嘴硬、聪明、怕麻烦，但关键时刻反应快。  
动画优势：表情变化大，适合惊恐、装傻、嘴硬。  
推荐指数：★★★★★

出图关键词：

```text
17岁中国高中男生，黑色短发，校服外套敞开，白T恤，单肩背包，眼神机灵，表情懒散又聪明，普通学生感，国漫写实风，校园悬疑氛围
```

## B卡：阴郁敏感型

定位：更恐怖，更像规则怪谈主角。  
形象：偏瘦，刘海稍长，校服整齐但有点旧，眼神敏感，黑眼圈轻微。  
气质：不爱说话，像长期被同学忽视的人。  
动画优势：压迫感强，适合心理恐怖。  
推荐指数：★★★★

出图关键词：

```text
17岁中国高中男生，偏瘦，稍长刘海，校服整齐但陈旧，眼神敏感，轻微黑眼圈，沉默紧张，国漫写实恐怖风，冷蓝校园灯光
```

## C卡：阳光反差型

定位：白天越阳光，晚上越恐怖。  
形象：短发清爽，笑起来很普通，校服干净，像班里人缘不错的男生。  
气质：白天嬉皮笑脸，晚上崩溃装瞎。  
动画优势：反差强，适合前10秒吸引人。  
推荐指数：★★★★

出图关键词：

```text
17岁中国高中男生，清爽短发，干净校服，阳光笑容，眼神灵活，白天普通开朗，夜晚惊恐反差，国漫写实风，校园悬疑
```

# 女主/班长：许眠

## A卡：冷静班长型

定位：最稳，像知道规则的人。  
形象：黑色长发低马尾，校服扣子扣好，干净、克制，眼神冷。  
气质：不多话，但一句话能救命。  
动画优势：神秘感强，适合递纸条、黑暗提醒。  
推荐指数：★★★★★

出图关键词：

```text
17岁中国高中女生，黑色长发低马尾，干净校服，冷静眼神，表情克制，手里拿纸条，像知道秘密的班长，国漫写实风，校园悬疑氛围
```

## B卡：温柔危险型

定位：更有暧昧和反转感。  
形象：长发披肩，眼神温柔，笑容很轻，校服外套整齐。  
气质：看起来在帮男主，但总让人怀疑她藏着东西。  
动画优势：适合“她到底是不是人”的悬念。  
推荐指数：★★★★

出图关键词：

```text
17岁中国高中女生，长发披肩，温柔浅笑，校服整齐，眼神安静但有隐藏秘密，手持旧照片，国漫写实风，轻微恐怖氛围
```

## C卡：病弱学霸型

定位：更怪，更适合后期揭露她被换过脸。  
形象：皮肤偏白，细框眼镜，校服宽大，手腕细，脸色略疲惫。  
气质：像随时会消失的人。  
动画优势：适合镜子、医务室、旧照片剧情。  
推荐指数：★★★★

出图关键词：

```text
17岁中国高中女生，皮肤偏白，细框眼镜，宽大校服，脸色疲惫，安静学霸气质，眼神藏着恐惧，国漫写实校园恐怖
```

# 反派压迫源：值夜老师

## A卡：眼镜教导主任型

定位：最适合第一季主反派。  
形象：瘦高，中年男教师，白衬衫，深色裤子，金属眼镜强反光，拿点名册。  
气质：说话客气，但每句话都像审判。  
动画优势：好识别，压迫感强。  
推荐指数：★★★★★

出图关键词：

```text
中年中国男教师，瘦高身材，白衬衫，深色裤子，金属边眼镜强反光，看不清眼睛，手持旧点名册和手电，僵硬微笑，红色应急灯，国漫恐怖风
```

## B卡：温和班主任型

定位：更阴险，白天亲切，晚上变味。  
形象：圆脸，微胖，穿灰色夹克，笑容亲切，手里保温杯。  
气质：越像好老师，夜里越吓人。  
动画优势：反差强，适合“你不是夜盲吗？”这句台词。  
推荐指数：★★★★

出图关键词：

```text
中年中国男教师，微胖圆脸，灰色夹克，亲切笑容，手拿保温杯，夜晚笑容僵硬，眼神空洞，校园恐怖国漫风，红色走廊灯光
```

## C卡：无声巡楼人型

定位：更像怪谈，不像普通人。  
形象：高瘦，黑色旧西装，袖口过长，脸被阴影遮住，只有眼镜反光。  
气质：几乎不说话，只翻点名册。  
动画优势：画面恐怖强，但人味少。  
推荐指数：★★★★

出图关键词：

```text
高瘦中年男教师，黑色旧西装，袖口过长，脸部被阴影遮住，眼镜反光，手持点名册，沉默巡楼，国漫规则怪谈风，压迫感强
```

# 第一异常学生：陈雨

## A卡：普通女同学型

定位：最吓人，因为太正常。  
形象：短发女生，普通校服，白天笑容自然，夜晚眼神空洞。  
气质：第二天像没事人一样回来。  
动画优势：前后对比强。  
推荐指数：★★★★★

出图关键词：

```text
17岁中国高中女生，短发，普通校服，白天自然微笑，夜晚眼神空洞，同一张脸有轻微不协调感，国漫写实校园恐怖
```

## B卡：乖乖女型

定位：被换脸后更诡异。  
形象：齐刘海，圆眼睛，校服整齐，笑容甜。夜里脸部像贴偏一点。  
气质：乖到不真实。  
动画优势：适合“新的脸不太合适”。  
推荐指数：★★★★

出图关键词：

```text
17岁中国高中女生，齐刘海，圆眼睛，校服整齐，甜美笑容，夜晚五官轻微错位但不血腥，诡异乖巧，国漫写实风
```

## C卡：冷淡转学生型

定位：后面能扩成支线。  
形象：长直发，表情冷，校服外套偏大，手腕戴黑色发圈。  
气质：像知道自己已经不是本人。  
动画优势：可以做二次反转。  
推荐指数：★★★

出图关键词：

```text
17岁中国高中女生，长直发，冷淡表情，宽大校服外套，手腕黑色发圈，眼神空洞，像隐藏秘密的转学生，校园悬疑国漫风
```

# 隐藏角色1：真正夜盲的学生

## A卡：盲杖少年

定位：中后期关键解释者。  
形象：男生，短发，戴深色眼镜，拿折叠盲杖，校服干净。  
气质：平静，像早已习惯学校夜晚。  
作用：告诉男主“夜盲不是看不见黑，而是看不见伪装成人的东西”。  
推荐指数：★★★★★

出图关键词：

```text
17岁中国高中男生，戴深色眼镜，手持折叠盲杖，校服干净，表情平静，站在夜晚走廊，神秘解释者，国漫写实悬疑风
```

## B卡：医务室女生

定位：更柔，更恐怖。  
形象：女生，眼睛被纱布遮住，穿校服，坐在医务室床边。  
气质：很安静，说出最可怕的真相。  
作用：连接医务室和病历秘密。  
推荐指数：★★★★

出图关键词：

```text
17岁中国高中女生，眼睛缠白色纱布，穿校服，坐在医务室床边，表情安静，冷白灯光，校园恐怖国漫风
```

# 隐藏角色2：镜中林昼

## A卡：完美替身型

定位：男主的恐怖倒影。  
形象：和林昼几乎一样，但校服更整齐，笑容更标准，眼神没有温度。  
气质：像一个被学校训练好的“好学生版本林昼”。  
作用：诱惑男主让它替自己活。  
推荐指数：★★★★★

出图关键词：

```text
17岁中国高中男生，外貌与男主相似但更整齐，校服扣好，标准微笑，眼神没有温度，站在镜子里的红光教室中，国漫心理恐怖风
```

## B卡：影子替身型

定位：更怪谈，更抽象。  
形象：只有轮廓像林昼，脸部模糊，校服像被水浸湿，站在镜子深处。  
气质：不像人，更像被复制失败。  
作用：适合第三集结尾强恐怖画面。  
推荐指数：★★★★

出图关键词：

```text
高中男生影子替身，轮廓像主角，脸部模糊，校服湿冷，站在镜子深处的红光教室，心理恐怖，国漫写实风，不血腥
```

# 我建议的首选阵容

最稳、最适合第一条样片：

```text
林昼：A卡 懒散聪明型
许眠：A卡 冷静班长型
值夜老师：A卡 眼镜教导主任型
陈雨：B卡 乖乖女型
真正夜盲学生：A卡 盲杖少年
镜中林昼：A卡 完美替身型
```

这套的好处：

- 主角好代入。
- 班长有神秘感。
- 老师一眼就是压迫源。
- 陈雨第一集够吓人。
- 后面能自然引出镜子、医务室、真正夜盲学生。
