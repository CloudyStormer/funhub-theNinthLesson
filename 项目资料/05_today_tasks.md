# 今天生产任务

## 目标

先做第1集样片，不做长篇空想。

## 必做

1. 生成四个固定角色设定图：林昼、许眠、值夜老师、陈雨。
2. 生成三个固定场景：夜晚走廊、第九节课教室、宿舍。
3. 生成第1集6张关键画面。
4. 做第1集配音稿。
5. 做第1集字幕稿。

## 第1集配音稿

我只是想逃一次晚自习。

老师问我，夜盲以前怎么没听说过。

我说最近严重了，医生说天一黑就看不见。

那天晚上，我路过教学楼。

明明全校停电，三楼却亮着红光。

我从门缝里看见一排学生背对黑板。

老师站在讲台上点名。

他说，陈雨，换完了，下一个。

那个女生慢慢回头。

她的脸，像贴错了一样。

我撞到了门。

整个教室都安静了。

老师走到我面前，问：林昼，你不是夜盲吗？

我低下头，故意撞上门框。

我说，老师，我看不见。

他笑了。

他说，那就好。

可我看见他在点名册上写下了我的名字。

林昼，待观察。
