# 正式角色提示词包

## 使用目标

这份文件用于正式生产《夜盲同学》的角色设定图、表情图、动作图和后续图生视频。  
`10_character_gacha_cards.md` 是选角池，本文件是已经锁定的第一版生产标准。

## 锁定阵容

| 角色 | 采用方向 | 用途 |
| --- | --- | --- |
| 林昼 | A卡：懒散聪明型 | 男主，负责代入感和惊恐反应 |
| 许眠 | A卡：冷静班长型 | 女主/规则知情者，负责悬疑感 |
| 值夜老师 | A卡：眼镜教导主任型 | 第一季压迫源，负责审判感 |
| 陈雨 | B卡：乖乖女型 | 第一集异常学生，负责第一波恐怖画面 |
| 真正夜盲的学生 | A卡：盲杖少年 | 中后期解释者 |
| 镜中林昼 | A卡：完美替身型 | 第3集心理恐怖核心 |

## 统一正向风格

```text
国漫风格，写实漫画质感，校园恐怖氛围，低饱和色彩，冷蓝和灰绿色阴影，红色应急灯点缀，电影感构图，细腻面部表情，干净线条，轻微颗粒质感，阴影压迫感强，不血腥，不低俗，不夸张怪物化，人物五官稳定，角色一致性强
```

## 统一负向限制

```text
不要欧美脸，不要日式夸张大眼，不要Q版，不要二次元萌系，不要血腥伤口，不要肢体畸形，不要怪物化，不要过度暴露，不要赛博朋克，不要奇幻服装，不要古风，不要夸张表情包，不要多余手指，不要文字水印，不要画面中出现乱码文字
```

## 一致性检查

每次出图后只看四件事：

1. 林昼是否还是黑短发、松垮校服、懒散机灵。
2. 许眠是否还是低马尾、整齐校服、冷静克制。
3. 值夜老师是否还是瘦高、白衬衫、反光眼镜、旧点名册。
4. 陈雨是否还是齐刘海、圆眼睛、乖巧但空洞。

如果这四项偏了，先重抽角色图，不进入关键帧。
